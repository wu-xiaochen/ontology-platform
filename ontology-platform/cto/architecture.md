### 1.2 技术栈

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| 后端 | Python 3.10+ | 主开发语言 |

| 本体存储 | YAML/JSON | 结构化知识库 |
| 缓存 | Redis | 会话/推理缓存 |
| 数据库 | SQLite/PostgreSQL | 结构化数据 |
| 前端 | React + TypeScript | Web界面 |

## 三、部署方案

### 3.1 容器化部署

```yaml
# docker-compose.yml
version: '3.8'
services:
  api:
    build: .
    ports:
      - "8000:8000"
    environment:
      - REDIS_URL=redis://redis:6379
      - DB_URL=sqlite:///ontology.db
    
  redis:
    image: redis:7-alpine
    
  nginx:
    image: nginx:alpine
    ports:
      - "80:80"
```

### 3.2 Kubernetes部署

- 使用Helm Chart管理
- 支持Horizontal Pod Autoscaling
- 配置健康检查和探针
