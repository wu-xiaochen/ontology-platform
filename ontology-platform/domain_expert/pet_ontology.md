# 宠物领域本体设计

本文档定义宠物领域的核心本体模型，用于指导宠物行业知识库构建和智能化应用。

---

## 一、概念定义 (Concepts)

## 二、规律 (Laws)

### 2.1 宠物生命周期规律

**定义**: 宠物从幼年到老年各阶段的生理和行为特征变化规律。

**阶段划分**:

```
幼年期 (0-1岁)
├── 身体发育关键期
├── 性格形成期
├── 社会化训练期
└── 疫苗接种期

成年期 (1-7岁)
├── 体能稳定期
├── 繁殖适龄期
├── 行为稳定期
└── 常规保健期

老年期 (7岁以上)
├── 身体机能衰退
├── 慢性病高发期
├── 行为变化期
└── 特殊护理期
```

**养护原则**:
- 幼年期：重点关注营养、免疫、训练
- 成年期：重点关注健康体检、适度运动、心理健康
- 老年期：重点关注定期体检、特殊护理、舒适度

### 2.2 宠物体重与健康规律

**定义**: 宠物体重与健康状况的关联规律。

**体重评估标准**:

| 体型 | 视觉特征 | 触摸特征 | 健康风险 |
|------|----------|----------|----------|
| 偏瘦 | 肋骨可见，腰部明显 | 肋骨易触摸，无脂肪 | 营养不良，免疫力低 |
| 标准 | 肋骨不明显，腰部曲线 | 肋骨有薄脂肪层 | 健康 |
| 偏胖 | 无腰身，腹部圆润 | 肋骨难触摸，厚脂肪层 | 糖尿病、关节问题 |
| 肥胖 | 腹部下垂，行动笨拙 | 完全触摸不到肋骨 | 心脏病、胰腺炎 |

**计算公式**:
```
理想体重 = 品种标准体重 × 体型系数
体型系数：偏瘦0.9，标准1.0，偏胖1.1，肥胖1.2+
```

### 2.3 疫苗接种规律

**定义**: 宠物疫苗接种的时间、类型和程序规律。

**犬类疫苗程序**:
```
首次免疫 (幼犬)
├── 6-8周：犬二联/犬四联 (第1针)
├── 10-12周：犬四联/犬六联 (第2针)
├── 14-16周：犬四联/犬六联 + 狂犬疫苗 (第3针)
└── 12-16个月：加强免疫

年度免疫
├── 犬四联/犬六联：每年加强
└── 狂犬疫苗：每年/每三年加强
```

**猫类疫苗程序**:
```
首次免疫 (幼猫)
├── 8周：猫三联 (第1针)
├── 12周：猫三联 (第2针)
├── 16周：猫三联 + 狂犬疫苗 (第3针)
└── 12-16个月：加强免疫

年度免疫
├── 猫三联：每年加强
└── 狂犬疫苗：每年加强
```

### 2.4 驱虫规律

**定义**: 宠物体内外寄生虫预防的时间规律。

**体内驱虫**:
```
幼年宠物：每2周一次至3月龄
          每月一次至6月龄
          之后每3个月一次
成 年 宠 物：每3个月一次
繁殖宠物：配种前驱虫
母犬/母猫：产后驱虫
```

**体外驱虫**:
```
季节性驱虫：春夏季每月一次
           冬季每2-3个月一次
高风险环境：每月一次
常用药品：滴剂、喷剂、口服药
```

### 2.5 宠物发情繁殖规律

**定义**: 宠物性成熟和繁殖的生理规律。

**犬类发情周期**:
```
性成熟：小型犬6-9月龄，大型犬9-15月龄
发情周期：平均180天 (120-240天)
发情期：持续7-14天
最佳配种期：发情后第9-13天
妊娠期：58-68天，平均63天
哺乳期：4-6周
繁殖适龄：1.5-6岁
```

**猫类发情周期**:
```
性成熟：平均6月龄 (4-12月龄)
发情周期：季节性发情 (春秋高发)
发情期：持续4-7天
最佳配种期：发情后24-48小时
妊娠期：58-71天，平均65天
哺乳期：4-6周
繁殖适龄：1-7岁
```

## 三、最佳实践 (Best Practices)

### 3.1 新手养宠 Checklist

**养宠前准备**:
- [ ] 确认家庭成员对宠物无过敏
- [ ] 评估居住环境是否适合养宠
- [ ] 了解当地养宠法规和登记要求
- [ ] 预算养宠费用 (首年5000-15000元)
- [ ] 选择适合自己生活方式的宠物品种

**接宠准备**:
- [ ] 准备好宠物窝/笼子
- [ ] 准备好食碗、水碗
- [ ] 准备好宠物粮 (建议延续原饲养)
- [ ] 准备好尿垫/猫砂
- [ ] 准备好牵引绳/宠物包

**健康护理**:
- [ ] 1周内完成体检
- [ ] 按时完成疫苗接种
- [ ] 按时完成驱虫
- [ ] 办理宠物登记/芯片植入
- [ ] 购买宠物保险 (推荐)

### 3.2 宠物喂养指南

**犬类喂养原则**:
```
幼犬 (2-6月龄)
├── 每日3-4餐
├── 选用幼犬专用粮
├── 控制食量，少食多餐
├── 保证新鲜饮水
└── 可补充羊奶粉

成犬 (6月龄以上)
├── 每日2餐
├── 选用成犬粮
├── 根据体重和运动量调整食量
├── 禁喂人类食物
└── 注意体重控制
```

**猫类喂养原则**:
```
幼猫 (2-12月龄)
├── 每日3-4餐
├── 选用幼猫粮
├── 可喂湿粮补充水分
└── 保证新鲜饮水

成猫 (12月龄以上)
├── 每日2-3餐
├── 可自由采食或定时喂食
├── 选用成猫粮
├── 多喂湿粮预防泌尿问题
└── 保证新鲜饮水
```

**常见禁喂食物**:
| 食物 | 对宠物的危害 |
|------|--------------|
| 巧克力 | 可可碱中毒，可能致命 |
| 洋葱/大蒜 | 破坏红细胞，溶血性贫血 |
| 葡萄/葡萄干 | 急性肾衰竭 |
| 木糖醇 | 胰岛素大量分泌，低血糖 |
| 酒精 | 呕吐、腹泻、昏迷、死亡 |
| 咖啡因 | 心跳加速、神经系统异常 |
| 骨头 | 可能刺穿消化道 |
| 牛奶 | 乳糖不耐受，腹泻 |
| 生鸡蛋 | 沙门氏菌风险 |

### 3.3 宠物美容护理周期

**犬类美容**:
| 项目 | 频率 | 备注 |
|------|------|------|
| 洗澡 | 夏季1-2周，冬季2-4周 | 根据品种和活动量调整 |
| 梳毛 | 每天/每周 | 长毛犬每天，短毛犬每周 |
| 剪指甲 | 每2-4周 | 室内犬需更频繁 |
| 清洁耳朵 | 每1-2周 | 垂耳犬需更频繁 |
| 刷牙 | 每天/每周 | 预防牙结石 |
| 剃毛/修剪 | 根据品种 | 炎热季节或美观需要 |

**猫类美容**:
| 项目 | 频率 | 备注 |
|------|------|------|
| 梳毛 | 每天/每周 | 长毛猫每天，短毛猫每周 |
| 洗澡 | 必要时 | 猫自洁能力强，一般不需洗澡 |
| 剪指甲 | 每2-3周 | 使用专用指甲剪 |
| 清洁耳朵 | 每1-2周 | 注意观察分泌物 |
| 刷牙 | 每天/每周 | 预防牙周疾病 |
| 猫砂更换 | 每天铲屎，每周换砂 | 保持清洁预防疾病 |

### 3.4 宠物就医指南

**常见症状与对应科室**:
| 症状 | 可能疾病 | 建议科室 |
|------|----------|----------|
| 呕吐/腹泻 | 肠胃炎、传染病 | 内科 |
| 咳嗽/打喷嚏 | 呼吸道感染 | 呼吸科 |
| 皮肤瘙痒/脱毛 | 皮肤病、寄生虫 | 皮肤科 |
| 跛行/不愿动 | 关节炎、骨折 | 骨科 |
| 眼睛异常 | 结膜炎、青光眼 | 眼科 |
| 尿频/血尿 | 泌尿系统疾病 | 泌尿科 |
| 精神萎靡 | 多种疾病可能 | 全科/急诊 |

**选择宠物医院要点**:
- [ ] 正规资质：动物诊疗许可证
- [ ] 执业兽医：持证上岗
- [ ] 设备条件：X光、B超、血液检测设备
- [ ] 透明价格：明码标价
- [ ] 口碑评价：其他宠主评价
- [ ] 距离便利：距离近，方便急诊

## 四、常见场景 (Common Scenarios)

### 4.1 宠物领养场景

**领养流程**:
```
1. 选择领养渠道
   ├── 动物救助中心
   ├── 宠物医院合作领养
   ├── 宠物店领养
   └── 个人送养

2. 了解宠物背景
   ├── 年龄、品种
   ├── 健康状况
   ├── 性格特点
   ├── 疫苗接种情况
   └── 是否有行为问题

3. 见面互动
   ├── 观察宠物状态
   ├── 与宠物互动
   ├── 了解宠物习惯

4. 签署领养协议
   ├── 提供身份证明
   ├── 承诺科学喂养
   └── 了解领养条件

5. 带回宠物
   ├── 准备运输工具
   ├── 预约体检
   └── 准备生活用品
```

### 4.2 宠物托运场景

**国内托运**:
```
方式选择
├── 航空托运
│   ├── 优点：速度快，适合长距离
│   ├── 缺点：需符合航空公司规定
│   └── 注意事项：提前申请氧舱、准备检疫证明
│
├── 火车托运
│   ├── 优点：价格较低，平稳
│   ├── 缺点：时间较长，班次有限
│   └── 注意事项：需提前咨询可托运车次
│
└── 汽车托运
    ├── 优点：灵活便捷
    ├── 缺点：不适合长距离
    └── 注意事项：保证通风、安全
```

**所需证件**:
- 宠物健康检疫证明 (有效期3-7天)
- 狂犬疫苗接种证明
- 主人身份证

### 4.3 宠物寄养场景

**寄养方式**:
```
家庭式寄养
├── 优点：环境温馨，一对一照顾
├── 缺点：专业性参差不齐
└── 适合：胆小型猫咪、需要陪伴的宠物

宠物店/寄养中心
├── 优点：专业设备，每日消毒
├── 缺点：环境陌生，可能应激
└── 适合：适应能力强的宠物

宠物医院寄养
├── 优点：医疗保障，专业护理
├── 缺点：价格较高
└── 适合：老年宠物、有健康问题的宠物
```

**寄养注意事项**:
- [ ] 提前预约，留意节假日高峰
- [ ] 准备惯用的食物和用品
- [ ] 提供宠物习性说明
- [ ] 留下紧急联系方式
- [ ] 确认寄养环境和护理标准
- [ ] 了解费用组成

### 4.4 宠物保险场景

**主要险种**:
| 险种 | 保障范围 | 适用对象 | 年保费 |
|------|----------|----------|--------|
| 医疗险 | 疾病医疗、意外医疗 | 所有宠物 | 500-3000元 |
| 意外险 | 意外伤害、死亡 | 所有宠物 | 200-1000元 |
| 综合险 | 医疗+意外+责任 | 所有宠物 | 800-5000元 |
| 责任险 | 宠物伤人赔偿 | 犬类主人 | 100-500元 |

**保险选择要点**:
- [ ] 关注等待期 (常见30天)
- [ ] 了解免赔额和赔付比例
- [ ] 查看指定医院范围
- [ ] 注意既往症不赔
- [ ] 考虑续保稳定性
