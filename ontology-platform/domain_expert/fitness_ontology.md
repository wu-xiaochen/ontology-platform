# 健身领域本体 (Fitness Ontology)

## 概述

本文档定义健身领域的核心本体模型，用于指导健身知识库构建、智能训练规划和健康管理应用。

---

## 一、概念定义 (Concepts)

## 二、规律 (Laws)

### 2.1 渐进超负荷法则

**定义 1**: 肌肉生长需要持续增加训练刺激，渐进超负荷是力量训练的核心原则。

**公式**:
```
新训练重量 = 当前重量 × (1 + 渐进系数)
```
渐进系数范围：1.5%-5%，根据训练水平和动作难度调整

**应用规则**:
- 每次训练重量增加 2.5-5kg（上肢）/ 5-10kg（下肢）
- 或每两周增加一次重量
- 出现技术变形时停止加重

### 2.2 肌肉生长规律

**定义 2**: 肌肉肥大遵循机械张力、代谢应激和肌肉损伤三大机制。

**公式**:
```
肌肉增长率 = (机械张力 × 0.4) + (代谢应激 × 0.3) + (肌肉损伤 × 0.3)
```
- 机械张力：6-12次力竭范围最佳
- 代谢应激：泵感训练、递减组
- 肌肉损伤：离心收缩训练、新动作

### 2.3 恢复周期规律

**定义 3**: 不同肌群和训练强度需要不同的恢复时间。

**公式**:
```
恢复时间 = 基础恢复时间 × 强度系数 × 年龄系数
```
- 大肌群（胸、背、腿）：72小时
- 小肌群（肩、手臂、腹）：48小时
- 高强度训练：+24小时恢复
- 年龄>35岁：×1.2系数

### 2.4 热量平衡规律

**定义 4**: 体重变化取决于热量摄入与消耗的差值。

**公式**:
```
体重变化 = (摄入热量 - 消耗热量) / 7700
```
- 7700 kcal ≈ 1kg 体重

**消耗热量分解**:
```
总消耗 = 基础代谢 × 活动系数 + 运动消耗 + 食物热效应
```
- 基础代谢：静息状态下维持生命所需热量
- 活动系数：1.2（久坐）至 1.9（运动员）
- 食物热效应：约占总消耗的 10%

### 2.5 训练频率法则

**定义 5**: 每块肌肉每周需要足够的训练频率来最大化生长。

**公式**:
```
最佳训练频率 = 每周总训练量 / 单次训练量
```
- 每个肌群每周至少训练10-20组
- 高频方案：每周2-3次/肌群
- 低频方案：每周1次/肌群
- 根据恢复能力和训练经验调整

### 2.6 蛋白质需求规律

**定义 6**: 蛋白质摄入量需要根据体重和训练目标确定。

**公式**:
```
蛋白质需求 = 体重(kg) × 蛋白质系数
```
- 维持：0.8-1.0g/kg
- 增肌：1.6-2.2g/kg
- 减脂保肌：2.0-2.6g/kg
- 每餐建议：20-40g 蛋白质

### 2.7 心率训练区间

**定义 7**: 不同训练目标对应不同的心率区间。

**公式**:
```
目标心率 = (最大心率 - 静息心率) × 目标区间% + 静息心率
```
- 最大心率 = 220 - 年龄
- 区间划分：
  - Z1（热身）：50-60%
  - Z2（耐力）：60-70%
  - Z3（有氧）：70-80%
  - Z4（阈值）：80-90%
  - Z5（最大）：90-100%

## 三、业务规则 (Rules)

### 3.1 训练计划制定规则

| 步骤 | 内容 | 关键要点 |
|------|------|----------|
| 1. 目标确定 | 明确训练目标 | 增肌/减脂/力量/耐力/健康 |
| 2. 评估现状 | 测试当前水平 | 1RM测试、体脂测量、体能评估 |
| 3. 计划设计 | 制定训练方案 | 动作选择、组数次数、休息时间 |
| 4. 进度安排 | 规划周期 | 4-8周周期、渐进负荷、测试周 |
| 5. 营养配合 | 制定饮食方案 | 热量、蛋白质、补剂 |
| 6. 恢复安排 | 规划休息 | 睡眠、主动恢复、拉伸 |

**训练计划模板**:
```
需求确认 → 体能评估 → 目标设定 → 计划设计 → 动作编排 
→ 负荷确定 → 周期规划 → 营养配合 → 执行跟踪 → 效果评估 → 计划调整
```

### 3.2 动作选择规则

| 目标 | 动作类型 | 推荐动作 |
|------|----------|----------|
| 胸肌 | 推类动作 | 卧推、哑铃卧推、俯卧撑 |
| 背阔肌 | 拉类动作 | 引体向上、划船、高位下拉 |
| 股四头肌 | 蹲类动作 | 深蹲、腿举、弓步 |
| 腘绳肌 | 弯举类 | 硬拉、腿弯举 |
| 肩部 | 推/侧平举 | 肩上推举、侧平举、面拉 |
| 手臂 | 弯举/臂屈伸 | 哑铃弯举、绳索下压 |
| 核心 | 稳定/抗旋 | 平板支撑、俄罗斯转体 |

### 3.3 训练强度规则

| 强度等级 | RPE | 描述 | 适用场景 |
|----------|-----|------|----------|
| 1-3 | 非常轻 | 热身、康复 | 热身、受伤恢复 |
| 4-5 | 轻 | 技术练习 | 新动作学习 |
| 6-7 | 中等 | 耐力训练 | 肌耐力、减脂 |
| 8-9 | 较重 | 增肌训练 | 肌肉肥大 |
| 10 | 力竭 | 极限挑战 | 冲刺测试 |

**组数与次数推荐**:
| 目标 | 组数 | 次数 | 休息时间 |
|------|------|------|----------|
| 力量 | 3-5 | 1-5 | 3-5分钟 |
| 肌肉肥大 | 3-4 | 6-12 | 1-2分钟 |
| 肌耐力 | 2-3 | 15-20 | 30-60秒 |
| 力量耐力 | 3-4 | 10-15 | 1分钟 |

### 3.4 营养计划规则

**宏量营养素分配**:
| 目标 | 碳水 | 蛋白质 | 脂肪 |
|------|------|--------|------|
| 增肌 | 50-60% | 25-30% | 15-20% |
| 减脂 | 30-40% | 30-40% | 20-30% |
| 维持 | 40-50% | 25-30% | 25-30% |

**餐次安排**:
- 训练前1-2小时：中等碳水、低蛋白、低脂
- 训练后30分钟内：高碳水、高蛋白
- 正餐：均衡搭配
- 加餐：根据热量需求安排

### 3.5 补剂使用规则

| 补剂 | 功效 | 剂量 | 使用时机 |
|------|------|------|----------|
| 乳清蛋白 | 快速补充蛋白质 | 20-40g | 训练后、早餐 |
| 肌酸 | 提升力量和恢复 | 3-5g/天 | 任何时候 |
| 咖啡因 | 提升训练表现 | 3-6mg/kg | 训练前30分钟 |
| 支链氨基酸(BCAA) | 减少肌肉分解 | 5-10g | 训练中/后 |
| 谷氨酰胺 | 免疫支持和恢复 | 5g/天 | 睡前或训练后 |
| 鱼油 | 抗炎和关节健康 | 2-3g EPA+DHA | 随餐 |

### 3.6 训练状态评估规则

**过度训练诊断标准**:
| 指标 | 正常 | 轻度过度 | 严重过度 |
|------|------|----------|----------|
| 静息心率 | 60-70 | +10% | +20% |
| 睡眠质量 | 良好 | 较差 | 差 |
| 训练表现 | 稳定 | 下降5-10% | 下降>15% |
| 情绪状态 | 正常 | 烦躁 | 抑郁 |
| 恢复感知 | 恢复快 | 恢复慢 | 无法恢复 |

### 3.7 安全规则

**动作规范**:
- 始终保持腰椎中立
- 重量选择以技术不变形为前提
- 孤立动作优先于复合动作（初学者）
- 关节活动范围内完成动作

**禁忌症**:
- 急性伤病期间禁止训练
- 力量训练前确认无心血管问题
- 关节疼痛时停止相关动作
- 怀孕期间避免高强度训练

## 五、应用场景

### 5.1 智能训练规划

- 根据健身者的目标、体型、运动经验生成个性化训练计划
- 自动编排动作顺序和负荷进度
- 根据训练状态动态调整计划

### 5.2 动作指导与纠正

- 识别动作中的常见错误并提供纠正建议
- 提供动作技巧的文字/视频指导
- 实时监测训练姿态（结合可穿戴设备）

### 5.3 营养管理

- 根据训练目标和体重计算热量和宏量营养素需求
- 推荐合适的补剂组合
- 追踪饮食日志并提供改进建议

### 5.4 进度追踪与分析

- 记录训练数据（重量、次数、组数）
- 可视化身体指标变化趋势
- 分析训练效果并提供调整建议

### 5.5 伤病预防

- 根据动作模式识别潜在的伤病风险
- 推荐康复训练动作
- 监控恢复状态并调整训练强度

### 5.6 知识问答

- 回答关于训练动作、肌肉群、营养的问题
- 解释健身术语和概念
- 提供健身科普和最新研究解读

### 5.7 教练匹配

- 根据会员目标匹配合适的健身教练
- 根据教练专长推荐针对性的训练指导
