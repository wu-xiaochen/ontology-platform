# 时尚领域本体设计

## 概述

本文档定义时尚领域的核心本体模型，用于指导时尚行业知识管理、趋势分析、电商运营、品牌营销等智能化应用。涵盖服装、配饰、美妆、设计师、品牌、时装周、时尚媒体等细分领域。

---

## 一、概念定义 (Concepts)

### 1.1 核心实体

#### 1.1.1 服装与配饰

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 服装 | Apparel | 上衣、裤子、裙子、外套等衣着产品 | 产品编号、名称、品类、款式、面料、尺码、价格 |
| 奢侈品 | Luxury | 高端品牌的服装、配饰产品 | 品牌编号、产品线、定价区间、稀缺性、工艺 |
| 设计师品牌 | Designer Brand | 由知名设计师创立的高端品牌 | 品牌编号、创始人、创立年份、设计风格、代表作品 |
| 成衣 | Ready-to-Wear | 工业化生产的标准尺码服装 | 产品编号、系列名称、季别、尺码范围、价格带 |
| 高定 | Haute Couture | 高级定制服装 | 客户编号、设计师、手工工艺、工时、价格 |
| 配饰 | Accessories | 包包、鞋子、珠宝、手表、眼镜等 | 产品编号、品类、材质、品牌、价格带 |
| 美妆产品 | Beauty Product | 化妆品、护肤品、香水等 | 产品编号、品牌、品类、功能、适用肤质、价格 |

#### 1.1.6 运营指标

| 概念 | 英文 | 定义 | 计算方式/说明 |
|------|------|------|---------------|
| 品牌热度 | Brand Popularity | 品牌在社交媒体的讨论热度 | 社交声量加权计算 |
| 时尚指数 | Fashion Index | 品牌/设计师的时尚影响力 | 媒体曝光+社交讨论+专家评分 |
| 带货能力 | Sales Drive | KOL/明星的产品销售转化能力 | GMV/粉丝数 或 转化率 |
| 坪效 | Sales per Area | 店铺单位面积销售额 | 销售额/店铺面积 |
| 季末清仓率 | Markdown Rate | 季末打折销售比例 | 折扣商品数/总库存数 |
| 新品售罄率 | Sell-through Rate | 新品上市后售完比例 | 售出数量/进货数量 |
| 复购率 | Repurchase Rate | 消费者重复购买比例 | 重复购买客户数/总客户数 |

## 二、规律 (Laws)

### 2.1 时尚流行周期

**定义**: 时尚趋势从兴起到衰退的时间规律。

**周期阶段**:
```
引入期 (Intro) → 增长期 (Growth) → 成熟期 (Peak) → 衰退期 (Decline)
     ↓              ↓               ↓              ↓
  小众尝试      追随者增加     主流普及      过时被淘汰
```

**周期时长**:
- 经典款: 5-10年甚至更长 (如小黑裙、牛仔裤)
- 流行趋势: 1-3年 (如特定颜色、面料)
- 瞬时潮流: 1-6个月 (如社交媒体爆款)

**规律**: 时尚是个轮回，衰退的趋势可能在2-5年后以"复古"名义回归。

### 2.2 时尚幂律分布

**定义**: 时尚行业资源、影响力呈现极端的不均匀分布。

**规律**:
```
头部20%品牌/设计师 → 占据80%市场份额/媒体曝光
头部1%超模/KOL → 占据90%顶级走秀/代言机会
```

**应用**:
- 品牌资源集中于头部集团
- 设计师出头难度极高
- KOL头部效应明显

### 2.3 奢侈品价格弹性

**定义**: 奢侈品价格与需求的关系规律。

**规律**:
- **凡勃伦效应**: 价格越高，需求越大 (炫耀性消费)
- **吉芬效应**: 经济下行时，低端奢侈品需求上升
- **价格锚定**: 入门款价格设定影响品牌形象

**定价公式**:
```
奢侈品价格 = 成本 × (20-50) 倍数
高端奢侈品价格 = 成本 × (50-100) 倍数
```

### 2.4 时尚趋势传播规律

**定义**: 流行趋势从源头到大众的传播路径。

**传播链路**:
```
时装周/设计师 → 时尚媒体 → 时尚icon/KOL → 大众消费者
   (T台)       (杂志/网站)   (街拍/带货)    (电商/门店)
      ↓            ↓              ↓            ↓
   6-12个月     3-6个月        1-3个月      即时可得
```

**加速趋势**: 社交媒体使传播周期缩短50%以上，爆款从T台到电商可短至2周。

### 2.5 库存周转规律

**定义**: 时尚产品季节性库存管理规律。

**核心指标**:
```
库存周转率 = 期间销售成本 / 平均库存
时尚行业标准:
- 快时尚 (Zara, H&M): 4-6次/年
- 设计师品牌: 1-2次/年
- 奢侈品: 0.5-1次/年
```

**季末折扣规律**:
- 第一轮折扣 (季末): 7折
- 第二轮折扣 (季末+2周): 5折
- 第三轮折扣 (季末+4周): 3折
- 奥特莱斯: 过季产品2-5折

## 三、业务规则 (Rules)

### 3.1 设计师品牌准入规则

| 条件 | 要求 |
|------|------|
| 资质要求 | 具备独立设计师品牌资质，2年以上设计经验 |
| 作品集 | 至少10套完整系列作品，参加过一次国际时装周 |
| 生产体系 | 拥有稳定的面料供应商和加工厂 |
| 合规要求 | 符合所在国家/地区的商标、税务、劳动法规 |
| 可持续发展 | 符合环保标准，面料可追溯 |

### 3.2 四大时装周参与规则

#### 3.2.1 巴黎/米兰时装周

| 要求 | 说明 |
|------|------|
| 邀请制 | 必须受主办方邀请或品牌符合资质的自主申请 |
| 品牌门槛 | 需有完整系列设计能力，通常需提前一季申请 |
| 场地要求 | 必须在指定秀场或符合资质的场地举办 |
| 日程安排 | 需提前3个月提交申请，确认后不得随意变更 |

#### 3.2.2 纽约/伦敦时装周

| 要求 | 说明 |
|------|------|
| 申请制 | 品牌可自主申请参加官方日程 |
| 商业考量 | 更注重商业化能力，鼓励即买即穿系列 |
| 场地灵活 | 秀场形式更多样化，接受非传统场地 |

### 3.3 品牌代言人选择规则

| 考量维度 | 权重 | 评估要素 |
|----------|------|----------|
| 形象契合度 | 30% | 品牌调性、目标客群、风格匹配 |
| 流量影响力 | 25% | 粉丝数量、互动率、话题热度 |
| 带货能力 | 20% | 历史代言销量、转化率、品类相关 |
| 公众形象 | 15% | 口碑、舆情、潜在风险 |
| 性价比 | 10% | 代言费 vs 预期ROI |

**禁选条件**:
- 近期有重大负面舆情
- 与竞品有代言关系
- 公众形象与品牌调性冲突

### 3.4 电商平台新品上架规则

| 平台 | 上架要求 |
|------|----------|
| 天猫/淘宝 | 资质审核、品牌授权、质检报告 |
| 京东 | 品牌授权、京东入驻费、仓储物流 |
| 抖音电商 | 直播资质、内容带货能力考核 |
| 小红书 | 品牌号认证、内容种草能力 |

**新品冷启动规则**:
1. 预热期: 提前2周进行内容种草
2. 首销期: 限时优惠刺激首批购买
3. 爆发期: KOL矩阵持续种草
4. 稳定期: 好评沉淀，搜索流量承接

### 3.5 时尚内容创作规则

#### 3.5.1 穿搭内容

| 类型 | 时长/篇幅 | 核心要素 |
|------|-----------|----------|
| 穿搭教程 | 3-5分钟/800字 | 搭配技巧、适用场景、单品链接 |
| 趋势解读 | 2-3分钟/500字 | 趋势分析、实穿建议、避坑指南 |
| 开箱测评 | 5-10分钟/1000字 | 产品展示、真实体验、优缺点 |

#### 3.5.2 时尚摄影

| 类型 | 用途 | 拍摄要求 |
|------|------|----------|
| 电商主图 | 销售转化 | 白底/场景图、高清、细节清晰 |
| 时尚大片 | 品牌传播 | 创意突出、视觉冲击、故事性 |
| 街拍 | 社交媒体 | 真实自然、光线优秀、穿搭亮点 |

### 3.6 奢侈品定价规则

| 定价因素 | 说明 | 影响权重 |
|----------|------|----------|
| 品牌溢价 | 品牌历史、声望、定位 | 40-60% |
| 物料成本 | 面料、五金、人工 | 15-25% |
| 研发设计 | 设计、工版、创意 | 10-20% |
| 营销费用 | 广告、代言、活动 | 10-15% |
| 渠道利润 | 代理、百货、电商抽成 | 15-30% |

**新品定价公式**:
```
零售价 = (物料成本 + 研发费) × 品牌倍数 × 渠道系数
品牌倍数: 顶级奢华 80-150x, 高端 30-80x, 设计 15-30x
渠道系数: 直营 1.0, 百货 1.3-1.5, 电商 1.1-1.2
```
