# 养老领域本体设计

## 概述

本文档定义养老领域的核心本体模型，用于指导养老知识库构建和智能化应用。涵盖老年人健康管理、养老服务、护理照料、康复理疗、养老机构、养老产品、养老金融等领域。

---

## 一、概念定义 (Concepts)

### 1.1 核心实体

#### 1.1.1 老年人主体

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 老年人 | Elderly Person | 年龄达到一定标准的自然人 | 老人编号、姓名、性别、年龄、身份证号、联系方式、居住地址、退休前职业 |
| 活力老人 | Active Elderly | 生活能自理的健康老人 | 老人编号、自理能力评估、健康状况、兴趣爱好、活动能力 |
| 失能老人 | Disabled Elderly | 部分或完全丧失生活自理能力的老人 | 老人编号、失能等级、护理需求、康复潜力、照护方案 |
| 半失能老人 | Partially Disabled Elderly | 部分丧失生活自理能力的老人 | 老人编号、自理项目清单、需协助项目、护理等级 |
| 失智老人 | Dementia Elderly | 患有认知功能障碍的老人 | 老人编号、痴呆类型、认知障碍程度、精神行为症状、安全风险评估 |
| 临终老人 | Terminal Elderly | 处于生命末期的老人 | 老人编号、预期寿命、临终关怀需求、舒缓治疗方案、家属陪伴需求 |
| 独居老人 | Living Alone Elderly | 独自居住的老人 | 老人编号、独居原因、紧急联系人、社区探访记录、安全检查频率 |

#### 1.1.2 养老服务人员

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 养老护理员 | Caregiver | 从事老年人生活照料的专业人员 | 护理员编号、姓名、资质等级、擅长领域、服务经验、健康证明 |
| 护士 | Nurse | 具备护理资质的专业人员 | 护士编号、姓名、职称、执业资格、护理专长、工作机构 |
| 康复治疗师 | Rehabilitation Therapist | 从事康复治疗的专业人员 | 治疗师编号、姓名、康复专业、资质等级、擅长领域 |
| 社工 | Social Worker | 从事老年人社会工作的专业人员 | 社工编号、姓名、专业资质、服务领域、个案经验 |
| 营养师 | Nutritionist | 从事老年营养指导的专业人员 | 营养师编号、姓名、资质等级、擅长领域、营养评估能力 |
| 心理咨询师 | Psychologist | 从事老年人心理服务的专业人员 | 心理咨询师编号、姓名、资质、擅长领域、咨询经验 |
| 养老机构管理人员 | Facility Manager | 养老机构的管理人员 | 编号、姓名、职务、管理资质、从业年限 |
| 家庭照护者 | Family Caregiver | 在家庭中照护老年人的家属 | 与老人关系、照护经验、照护培训、喘息服务需求 |

#### 1.1.3 养老机构

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 养老机构 | Elderly Care Facility | 提供养老服务的专业机构 | 机构编号、机构名称、机构类型、床位数、地址、联系方式 |
| 养老院 | Nursing Home | 为老年人提供集中居住和照料服务的机构 | 机构编号、床位类型、护理等级、服务内容、收费标准 |
| 护理院 | Care Home | 提供专业护理服务的医疗机构 | 机构编号、护理等级、医疗资质、康复设施、医保定点 |
| 社区养老中心 | Community Care Center | 社区层面的养老服务平台 | 中心编号、服务范围、日间照料床位、助餐能力、辐射社区 |
| 居家养老服务站 | Home Care Station | 提供居家养老服务的站点 | 站点编号、服务区域、服务项目、工作人员数量 |
| 老年公寓 | Senior Apartment | 提供独立居住的老年住宅社区 | 公寓编号、户型、配套设施、服务等级、收费模式 |
| 认知症照护专区 | Dementia Care Unit | 专门照护失智老人的专区 | 专区编号、收住能力、封闭管理措施、认知训练设施 |
| 临终关怀机构 | Hospice | 提供临终关怀服务的机构 | 机构编号、服务内容、舒缓治疗、哀伤辅导、家属支持 |

| 适老化改造服务 | Home Modification Service | 居家环境适老化改造服务 | 改造项目、改造标准、验收要求、补贴政策 |

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|

| 认知功能评估 | Cognitive Assessment | 老年人认知功能评估 | 评估量表、认知障碍程度、诊断建议、干预方案 |
| 心理健康评估 | Mental Health Assessment | 老年人心理健康状态评估 | 评估量表、心理问题筛查、干预建议、转介指征 |
| 老年综合征 | Geriatric Syndrome | 老年人特有的健康问题 | 综合征类型、临床表现、影响因素、干预措施 |

## 二、属性定义 (Properties)

### 2.1 老年人属性

| 属性 | 英文 | 数据类型 | 约束 | 说明 |
|------|------|----------|------|------|
| 老人编号 | ElderID | String | 必填 | 唯一标识 |
| 姓名 | Name | String | 必填 | 真实姓名 |
| 性别 | Gender | Enum | 必填 | 男/女 |
| 出生日期 | BirthDate | Date | 必填 | 格式：YYYY-MM-DD |
| 身份证号 | IDNumber | String | 必填 | 18位身份证号 |
| 联系电话 | Phone | String | 必填 | 11位手机号 |
| 居住地址 | Address | String | 必填 | 详细居住地址 |
| 自理能力 | SelfCareAbility | Enum | 必填 | 完全自理/部分自理/完全不能自理 |
| 失能等级 | DisabilityLevel | Enum | 可选 | 0-6级 |
| 认知状态 | CognitiveStatus | Enum | 可选 | 正常/轻度认知障碍/中度认知障碍/重度认知障碍 |
| 经济来源 | IncomeSource | Array | 可选 | 退休金/子女赡养/政府补贴等 |
| 医保类型 | InsuranceType | Enum | 可选 | 城镇职工/城镇居民/新农合/无 |

## 四、应用场景 (Use Cases)

### 4.1 智能养老服务匹配

根据老年人的健康状况、自理能力、经济条件、居住偏好等因素，智能推荐最适合的养老服务和机构。

**输入**：老人基本信息、健康档案、服务需求  
**输出**：养老服务推荐列表、机构推荐排名

### 4.2 养老机构评估选址

综合分析养老机构的地理位置、服务质量、收费标准、用户评价等因素，为老人选择最优养老机构。

**输入**：老人需求偏好、机构信息库  
**输出**：机构评估报告、推荐方案

### 4.3 老年人健康监测预警

通过可穿戴设备和智能家居设备采集老年人健康数据，实现健康风险预警和紧急救助。

**输入**：实时健康数据、历史健康档案  
**输出**：健康报告、异常预警、干预建议

### 4.4 养老服务资源调度

优化养老服务人员、服务机构、服务设备等资源的调度和分配，提高服务效率。

**输入**：服务需求、服务资源、地理位置  
**输出**：最优调度方案、服务人员安排

### 4.5 养老政策智能问答

基于养老领域本体知识库，构建智能问答系统，回答老年人及家属关心的政策问题。

**输入**：用户问题  
**输出**：精准答案、政策解读、办事指引
