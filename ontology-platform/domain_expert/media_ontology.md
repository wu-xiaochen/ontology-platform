# 媒体领域本体 (Media Ontology)

## 概述

本文档定义媒体领域的核心本体模型，涵盖传统媒体、数字媒体、社交媒体等多种形态，用于指导媒体内容管理、传播分析、智能推荐等应用场景。

---

## 一、概念定义 (Concepts)

### 1.1 媒体主体

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 媒体机构 | Media Organization | 从事内容生产、传播的组织 | 机构编码、名称、类型、资质、影响力 |
| 内容创作者 | Content Creator | 创作媒体内容的个人或团队 | ID、名称、类型、粉丝数、擅长领域 |
| 媒体记者 | Journalist | 从事新闻采编的专业人员 | 记者证、专业领域、工作年限 |
| 自媒体 | Self Media | 个人或团队运营的独立媒体 | 账号ID、平台、类型、粉丝量级 |
| 广告主 | Advertiser | 投放广告的企业或个人 | 企业ID、广告预算、投放目标 |

### 1.2 媒体内容

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 新闻 | News | 及时报道的客观信息 | 标题、来源、发布时间、类别 |
| 文章 | Article | 文字为主的深度内容 | 标题、作者、正文、字数、标签 |
| 视频 | Video | 视听结合的内容形式 | 标题、时长、分辨率、格式、标签 |
| 音频 | Audio | 声音为主的内容 | 标题、时长、格式、类型 |
| 图片 | Image | 视觉图像内容 | 格式、分辨率、版权、主题 |
| 直播 | Live Stream | 实时传输的内容 | 标题、主播、观看人数、时长 |
| 短视频 | Short Video | 时长较短的视频内容 | 标题、时长、平台、点赞数 |
| 专题报道 | Feature Story | 深度专题内容 | 主题、篇幅、系列化程度 |

### 1.3 媒体渠道

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 传统媒体 | Traditional Media | 报纸、杂志、电视、广播 | 发行量/收视率、覆盖区域、受众群体 |
| 门户网站 | Portal Website | 综合类网站 | 流量、Alexa排名、内容分类 |
| 社交媒体 | Social Media | 社交网络平台 | 用户规模、月活、功能特性 |
| 短视频平台 | Short Video Platform | 抖音、快手等 | 日活、用户画像、内容生态 |
| 资讯客户端 | News App | 移动端资讯应用 | 下载量、日活、算法推荐 |
| 垂直平台 | Vertical Platform | 特定领域的专业平台 | 领域、用户规模、专业度 |
| 音频平台 | Audio Platform | 喜马拉雅等音频平台 | 用户规模、内容库、付费率 |

### 1.4 受众群体

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 读者 | Reader | 文字内容消费者 | 画像、阅读偏好、活跃度 |
| 观众 | Viewer | 视频/直播内容消费者 | 画像、观看偏好、互动行为 |
| 听众 | Listener | 音频内容消费者 | 画像、收听场景、付费意愿 |
| 用户 | User | 平台使用者 | 画像、行为数据、价值分层 |
| 粉丝 | Fan | 忠实追随者 | 活跃度、互动率、消费能力 |
| 网民 | Netizen | 互联网用户 | 规模、分布、触网习惯 |

### 1.6 关系定义

```
媒体机构 ⊚ 雇佣 [媒体记者] (一对多)
媒体机构 ⊚ 产出 [媒体内容] (一对多)
内容创作者 ⊚ 创作 [媒体内容] (一对多)
媒体内容 ⊚ 发布于 [媒体渠道] (多对多)
媒体内容 ⊚ 针对 [受众群体] (多对多)
受众群体 ⊚ 关注 [内容创作者] (多对多)
受众群体 ⊚ 互动 [媒体内容] (多对多)
广告主 ⊚ 投放 [广告内容] (一对多)
广告内容 ⊚ 植入于 [媒体内容] (多对一)
媒体渠道 ⊚ 归属于 [平台方] (多对一)
```

## 二、规律 (Laws)

### 2.1 新闻价值规律

**定义**: 新闻事实本身具有的、决定其能否成为新闻的内在属性。

**核心要素**:

| 要素 | 说明 | 权重 |
|------|------|------|
| 时效性 | 事件的新近程度 | 高 |
| 重要性 | 对受众的影响范围 | 高 |
| 接近性 | 与受众的地理/心理距离 | 中 |
| 显著性 | 人物/地点的知名度 | 中 |
| 冲突性 | 矛盾、争议、对比 | 中 |
| 人情味 | 情感共鸣程度 | 低 |

**公式**:
```
新闻价值 = Σ(要素权重 × 要素得分) / 要素数量
```

### 2.2 传播扩散规律

**定义**: 信息在媒体渠道中传播扩散的内在机制。

**传播层级模型**:

```
第一层级: 核心媒体首发 (权威背书)
    ↓
第二层级: 主流媒体跟进 (扩大覆盖)
    ↓
第三层级: 社交媒体扩散 (病毒传播)
    ↓
第四层级: 长尾平台覆盖 (持续影响)
```

**传播速度公式**:
```
传播速度 = ln(转发次数 + 1) / 时间(小时)
```

**爆发阈值**:
- 临界点: 转发≥1000次后进入爆发期
- 衰减周期: 48小时后热度下降50%

### 2.3 受众注意力规律

**定义**: 受众在媒体消费过程中的注意力分配机制。

**注意力曲线**:
```
黄金15秒: 视频开头15秒决定是否继续观看
内容3秒法则: 标题/封面3秒内吸引注意
注意力周期: 单次阅读时长≤7分钟为最佳
```

**影响因素**:
- 标题吸引力
- 封面视觉
- 内容节奏
- 互动设计

### 2.4 内容生命周期规律

**定义**: 内容从发布到衰退的完整周期。

**生命周期阶段**:

| 阶段 | 时长 | 特征 |
|------|------|------|
| 首发期 | 0-24h | 流量快速增长 |
| 爆发期 | 24-72h | 达到流量峰值 |
| 稳定期 | 3-7天 | 流量平稳 |
| 衰退期 | 7-30天 | 流量下降 |
| 长尾期 | 30天+ | 搜索/推荐流量 |

**内容类型差异**:
- 新闻类: 生命周期短(24-48h)
- 知识类: 生命周期中(7-30天)
- 娱乐类: 生命周期差异大
- 经典内容: 可持续数年

### 2.5 平台算法推荐规律

**定义**: 平台推荐算法对内容分发的影响机制。

**推荐权重模型**:

| 因素 | 权重 | 说明 |
|------|------|------|
| 完播率/阅读完成率 | 30% | 内容质量核心指标 |
| 互动率 | 25% | 点赞、评论、收藏、转发 |
| 粉丝观看比例 | 20% | 粉丝忠诚度 |
| 收藏率 | 15% | 内容价值认可 |
| 发布频率 | 10% | 账号活跃度 |

**流量分发比例**:
- 私域流量(粉丝): 约10-30%
- 推荐流量(算法): 约50-80%
- 搜索流量: 约5-10%

### 2.6 受众画像规律

**定义**: 基于行为数据构建用户特征的方法。

**核心维度**:

| 维度 | 数据来源 | 分析方法 |
|------|----------|----------|
| 基本属性 | 注册信息、设备数据 | 统计分析 |
| 兴趣偏好 | 浏览、搜索、收藏 | 协同过滤 |
| 行为特征 | 访问时间、时长、路径 | 序列分析 |
| 消费能力 | 付费、购买行为 | RFM模型 |
| 社交关系 | 转发、评论、关注 | 社交网络分析 |

## 三、业务规则 (Rules)

### 3.1 内容创作规则

#### 3.1.1 标题创作规则

| 规则 | 说明 | 示例 |
|------|------|------|
| 数字法则 | 标题中使用具体数字 | "5个技巧"、"3个方法" |
| 悬念法则 | 引发好奇心 | "竟然是这样"、"没想到" |
| 热点法则 | 结合当下热点 | 蹭热点关键词 |
| 情绪法则 | 调动情绪 | 愤怒、感动、惊喜 |
| 利益法则 | 明确收益 | "必看"、"必备"、"必知" |

**违规词库**:
- 禁用: 虚假宣传、夸大其词、恶意引流
- 慎用: 绝对化用语、敏感话题

#### 3.1.2 内容质量标准

| 指标 | 达标要求 |
|------|----------|
| 原创度 | ≥70% |
| 信息价值 | 有明确知识点或信息增量 |
| 结构清晰 | 层次分明、逻辑通顺 |
| 格式规范 | 段落分明、配图适当 |
| 错别字 | ≤3处/千字 |

#### 3.1.3 视频内容规范

| 时长 | 平台 | 适用场景 |
|------|------|----------|
| 15-60秒 | 抖音 | 快速娱乐、资讯 |
| 1-3分钟 | B站 | 知识科普、日常 |
| 3-10分钟 | 视频平台 | 深度内容 |
| 10分钟+ | 视频平台 | 纪录片、访谈 |

**内容节奏**:
- 每5-8秒需要有一个吸引点
- 背景音乐需要符合内容情绪
- 字幕需要清晰可读

### 3.2 内容发布规则

#### 3.2.1 发布时间规则

| 受众类型 | 最佳发布时间 |
|----------|--------------|
| 职场人士 | 7:00-8:00, 12:00-13:00, 18:00-20:00 |
| 学生群体 | 12:00-13:00, 16:00-18:00, 21:00-23:00 |
| 老年人 | 6:00-7:00, 11:00-12:00, 18:00-19:00 |
| 周末 | 全天活跃，集中于10:00-12:00, 15:00-17:00 |

**行业差异**:
- 科技类: 周二至周四
- 娱乐类: 周五至周日
- 财经类: 周一至周三
- 健康类: 早间时段

#### 3.2.2 发布频率规则

| 账号类型 | 日更 | 周更 | 月更 |
|----------|------|------|------|
| 头部账号 | 1-3条 | - | - |
| 腰部账号 | 1条 | - | - |
| 尾部账号 | - | 3-5条 | - |
| 企业号 | - | 3-5条 | - |

#### 3.2.3 多平台分发规则

```
内容创作 → 平台适配 → 差异化发布 → 数据监控 → 优化迭代

平台适配要点:
- 标题: 微信公众号(深度) vs 抖音(简短)
- 封面: 小红书(竖版) vs B站(横版)
- 内容: 知乎(专业) vs 微博(简短)
- 标签: 各平台标签体系不同
```

### 3.3 审核合规规则

#### 3.3.1 内容红线

| 类别 | 禁止内容 | 处罚 |
|------|----------|------|
| 政治敏感 | 违反国家政策、损害国家形象 | 封号 |
| 违法违规 | 违法犯罪、欺诈行为 | 封号+法律追责 |
| 低俗色情 | 色情内容、低俗表演 | 限流+封禁 |
| 虚假信息 | 谣言、虚假广告 | 删除+限流 |
| 侵权内容 | 盗版、抄袭 | 下架+追责 |
| 暴力血腥 | 暴力内容、血腥画面 | 限流+删除 |
| 恶意营销 | 诱导关注、诱导分享 | 限流 |

#### 3.3.2 敏感行业广告资质

| 行业 | 所需资质 |
|------|----------|
| 医疗 | 《医疗广告审查证明》 |
| 金融 | 金融监管资质 |
| 教育 | 教育许可证 |
| 房地产 | 房地产预售许可证 |
| 酒类 | 酒类销售许可证 |

### 3.4 数据运营规则

#### 3.4.1 核心指标体系

| 指标 | 计算公式 | 优化目标 |
|------|----------|----------|
| 阅读量/播放量 | 实际阅读人数 | 提升曝光 |
| 完播率 | 看完人数/播放人数 | ≥30% |
| 点赞率 | 点赞数/阅读量 | ≥3% |
| 评论率 | 评论数/阅读量 | ≥1% |
| 转发率 | 转发数/阅读量 | ≥0.5% |
| 粉丝增长 | 新增粉丝-流失粉丝 | 正增长 |
| 转化率 | 目标行为数/触达人数 | 行业基准 |

#### 3.4.2 数据分析规则

```yaml
规则1: 内容效果评估
  IF 完播率 < 20% THEN 优化开头吸引力
  IF 点赞率 < 1% THEN 检查内容价值
  IF 转发率 < 0.3% THEN 增强共鸣/实用价值

规则2: 账号健康度评估
  IF 粉丝活跃度 < 5% THEN 需要激活粉丝
  IF 掉粉率 > 10%/月 THEN 检查内容质量
  IF 互动率下降 THEN 调整内容策略

规则3: 竞品分析
  每周分析竞品爆款内容
  提取共性规律应用到自身内容
```

### 3.5 版权保护规则

#### 3.5.1 素材使用规范

| 素材类型 | 规则 |
|----------|------|
| 自创内容 | 完全拥有版权 |
| 授权素材 | 需要书面授权协议 |
| CC0素材 | 可免费使用 |
| 公有领域 | 可免费使用 |
| 用户投稿 | 需要授权协议 |

#### 3.5.2 版权保护措施

```
水印策略:
- 显性水印: 品牌logo、ID
- 隐性水印: 数字水印、指纹

监测策略:
- 全网监测抄袭内容
- 定期排查侵权素材
- 建立素材溯源机制
```

### 3.6 商业化规则

#### 3.6.1 商业模式

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| 广告变现 | 展示/点击/效果广告 | 流量型平台 |
| 付费内容 | 会员/单篇付费 | 知识/专业内容 |
| 电商带货 | 商品销售分成 | 垂直领域KOL |

| IP授权 | 品牌/形象授权 | 头部IP |
| 平台补贴 | 创作激励计划 | 新兴平台 |

#### 3.6.2 广告投放规则

| 广告类型 | 计费方式 | 适用目标 |
|----------|----------|----------|
| CPM | 千次展示 | 品牌曝光 |

## 五、应用场景

- 用户兴趣建模

- 协同过滤推荐
- 实时推荐调整

基于传播规律的舆情监控:
- 热点话题识别
- 传播路径追踪
- 情绪倾向分析
- 影响力评估

### 5.3 内容创作辅助

基于创作规则的智能辅助:
- 标题优化建议

- 违规风险预警

### 5.4 商业价值评估

基于数据运营规则的价值评估:
- IP价值评估
- 广告效果预测
- 账号报价参考
- 投资回报分析

### 5.5 合规审核

基于审核规则的智能审核:
- 内容违规检测
- 版权侵权识别
- 资质验证
- 风险预警
