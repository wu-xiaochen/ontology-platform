# 制造业领域本体设计

## 概述

本文档定义制造业领域的核心本体模型，用于指导生产管理知识库构建和智能化应用。涵盖离散制造、流程制造等多种制造类型，覆盖从原材料到成品的完整生产过程。

---

## 一、概念定义 (Concepts)

### 1.1 核心实体

#### 1.1.1 生产主体

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 工厂 | Factory | 进行生产制造活动的场所 | 工厂编码、名称、地址、规模、所属行业 |
| 生产车间 | Workshop | 工厂内按产品或工艺划分的生产单元 | 车间编码、名称、面积、产能 |
| 生产线 | Production Line | 按工艺顺序排列的设备组合 | 产线编码、名称、产能、节拍 |
| 生产班组 | Shift Team | 负责具体生产操作的作业单元 | 班组编码、班次、人数、技能配置 |

#### 1.1.2 物料与产品

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 原材料 | Raw Material | 用于制造产品的初始物料 | 物料编码、名称、规格、单位、供应商 |
| 在制品 | Work in Progress (WIP) | 正在加工尚未完成的产品 | 物料编码、工序、数量、状态 |
| 成品 | Finished Product | 已完成全部生产工序的产品 | 产品编码、名称、规格、检验状态 |
| 物料清单 | Bill of Materials (BOM) | 产品所需的物料组成清单 | BOM版本、父项物料、子项物料、数量 |
| 工艺路线 | Routing | 产品加工的工序序列 | 路线编码、产品ID、工序顺序、标准工时 |

#### 1.1.3 生产资源

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 生产设备 | Production Equipment | 用于加工制造的机器设备 | 设备编码、名称、型号、产能、维护状态 |
| 工装夹具 | Jig & Fixture | 定位和夹持工件的工艺装备 | 夹具编码、名称、适用工序、寿命 |
| 模具 | Mold | 用于成型加工的工艺装备 | 模具编码、名称、腔数、寿命、适用产品 |
| 计量器具 | Measuring Instrument | 用于质量检测的仪器设备 | 仪器编码、名称、精度、检定周期 |

#### 1.1.4 人员角色

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 生产工人 | Operator | 直接从事生产操作的人员 | 员工编号、姓名、工种、技能等级 |
| 工艺工程师 | Process Engineer | 负责工艺设计和优化的技术人员 | 员工编号、姓名、专业领域、职称 |
| 质量工程师 | Quality Engineer | 负责质量管控的技术人员 | 员工编号、姓名、QC技能证书 |
| 设备工程师 | Equipment Engineer | 负责设备维护保养的技术人员 | 员工编号、姓名、设备类型专长 |
| 生产主管 | Production Supervisor | 负责车间生产管理的人员 | 员工编号、姓名、管理范围、班次 |

#### 1.1.5 生产任务

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 生产订单 | Production Order | 安排生产的正式指令 | 订单编号、产品、数量、交付日期、优先级 |
| 生产工单 | Work Order | 具体到工序的生产任务 | 工单编号、生产订单ID、工序、计划数量 |
| 生产任务 | Production Task | 分配给班组的作业任务 | 任务编号、工单ID、班组、计划开始/结束时间 |
| 生产派工 | Dispatch | 将任务分配给具体操作人员 | 派工编号、任务ID、操作人员、设备 |

#### 1.1.6 质量实体

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 质量标准 | Quality Standard | 产品应满足的质量要求 | 标准编号、名称、适用范围、指标限值 |
| 检验项目 | Inspection Item | 质量检验的具体项目 | 项目编号、名称、检验方法、判定标准 |
| 检验结果 | Inspection Result | 检验的实际测量值 | 结果编号、检验项目ID、实测值、判定结论 |
| 不良品 | Defective Product | 未达到质量要求的产品 | 不良编号、类型、数量、原因、处置方式 |
| 质量异常 | Quality Anomaly | 偏离正常质量状态的情况 | 异常编号、类型、严重程度、处理状态 |

## 二、规律 (Laws)

### 2.1 生产节拍 (Takt Time)

**定义**: 满足客户需求的生产节奏，即单位时间内的产出要求。

**公式**:
```
生产节拍 = 可用工作时间 / 客户需求量
```

**应用**:
- 生产节拍是精益生产的核心指标
- 所有工序的节拍应尽量均衡，避免瓶颈
- 节拍时间决定了产线产能

**示例**:
```
日工作时间为8小时(480分钟)，客户日需求为240件
生产节拍 = 480 / 240 = 2分钟/件
```

### 2.2 产能利用率 (Capacity Utilization)

**定义**: 实际产出与理论产能的比率。

**公式**:
```
产能利用率 = 实际产量 / 设计产能 × 100%
```

**行业基准**:
- 离散制造: 70-85%
- 流程制造: 80-95%
- 最佳实践: >90%

**影响因素**:
- 设备故障停机
- 换型换模时间
- 质量问题返工
- 人员缺勤
- 物料供应不及时

### 2.3 整体设备效率 (OEE)

**定义**: 衡量设备综合效率的指标，由可用率、性能率、质量率三个维度组成。

**公式**:
```
OEE = 可用率 × 性能率 × 质量率

可用率 = 运行时间 / 计划生产时间
性能率 = (理论节拍 × 实际产量) / 运行时间
质量率 = 合格品数量 / 实际产量
```

**OEE等级**:
- 世界级: OEE ≥ 85%
- 优秀: 75% ≤ OEE < 85%
- 良好: 65% ≤ OEE < 75%
- 一般: OEE < 65%

**六大损失**:
1. 设备故障停机
2. 换型换模停机
3. 短暂停机（缺料、堵料等）
4. 速度降低（设备参数未达标准）
5. 工艺缺陷（首件调试、参数调整）
6. 产品缺陷（生产过程中不良）

### 2.4 批次追溯原则

**定义**: 能够追踪物料和产品从源头到终点全过程的能力。

**追溯内容**:
- 原材料来源（供应商、批次）
- 生产时间（日期、班次）
- 生产人员（操作者、检验员）
- 设备信息（生产线、设备编号）
- 工艺参数（温度、压力、速度等）
- 检验结果（来料、过程、成品）

**追溯深度**:
| 级别 | 描述 | 追溯内容 |
|------|------|----------|
| 1级 | 批次级追溯 | 只能追溯到生产批次 |
| 2级 | 工序级追溯 | 可追溯到具体工序 |
| 3级 | 单件级追溯 | 可追溯到单品 |
| 4级 | 完整追溯 | 包含完整工艺参数和检验数据 |

### 2.5 生产准备时间 (Setup Time)

**定义**: 从完成一个产品生产到可以开始下一个产品生产的时间。

**构成**:
```
生产准备时间 = 外部准备时间 + 内部准备时间

外部准备时间：可以在设备运行时完成的准备工作
内部准备时间：必须停机才能完成的准备工作
```

**改善方法**:
- SMED（快速换模）：将内部准备转化为外部准备
- 标准化准备流程
- 预置工装夹具和模具

## 三、业务规则 (Rules)

### 3.1 生产计划规则

#### 3.1.1 计划层次

```
长期计划 (MPS) → 中期计划 (MRP) → 短期计划 (FCS) → 日排程
   │              │              │            │
   ▼              ▼              ▼            ▼
  年度/月度       周计划         日计划       班次任务
```

#### 3.1.2 计划优先级规则

| 优先级 | 订单类型 | 排序规则 |
|--------|----------|----------|
| P1 | 紧急订单 | 交付日期最早优先 |
| P2 | 正常订单 | 按交付日期排序 |
| P3 | 常规订单 | 按订单利润率排序 |
| P4 | 备库订单 | 按库存水平排序 |

#### 3.1.3 排程规则

- 相同产品订单优先连续排产，减少换型
- 瓶颈工序优先排产，确保产能利用率
- 交付紧急度高的订单优先排产
- 考虑设备维护计划，避免冲突

### 3.2 生产执行规则

#### 3.2.1 生产启动规则

| 检查项 | 要求 | 责任方 |
|--------|------|--------|
| BOM确认 | 核对物料清单版本正确 | 生产主管 |
| 工艺文件 | 确认工艺规程已发放 | 工艺工程师 |
| 设备状态 | 设备点检合格 | 设备工程师 |
| 工装夹具 | 确认模具/夹具可用 | 生产工人 |
| 物料准备 | 原材料已配送到位 | 仓库 |
| 首件确认 | 首件检验合格后方可批量生产 | 质量工程师 |

#### 3.2.2 生产过程控制规则

- 关键工序必须100%检验
- 发现异常立即停机，触发异常处理流程
- 每班次进行过程巡检，至少2次/班
- 换班必须进行交接班记录

#### 3.2.3 生产完成规则

- 产量必须达到计划数量的95%以上
- 成品必须经FQC检验合格后方可入库
- 尾数处理：不足整包装部分需单独标识
- 批次完工后进行批次归档

### 3.3 质量管理规则

#### 3.3.1 检验类型

| 检验类型 | 缩写 | 时机 | 检验比例 |
|----------|------|------|----------|
| 来料检验 | IQC | 物料入库前 | 抽检/全检 |
| 过程检验 | IPQC | 生产过程中 | 巡检 |
| 最终检验 | FQC | 生产完成后 | 100% |
| 出货检验 | OQC | 产品出货前 | 抽检 |

#### 3.3.2 质量判定标准

| 不良等级 | 定义 | 处置方式 |
|----------|------|----------|
| 致命缺陷 (A) | 危及使用安全 | 报废，不得使用 |
| 严重缺陷 (B) | 功能丧失或严重性能下降 | 返工/报废 |
| 轻微缺陷 (C) | 不影响功能使用 | 返修/让步接收 |
| 轻微缺陷 (D) | 外观轻微瑕疵 | 特采/降级使用 |

#### 3.3.3 质量异常处理规则

```
发现异常 → 停线标识 → 隔离不良品 → 分析原因 → 制定对策 → 实施改进 → 验证效果 → 标准化
    │          │           │           │           │           │           │          │
   15min      5min        10min       30min       20min       30min       24h        持续
```

**升级机制**:
| 异常等级 | 响应时间 | 升级对象 |
|----------|----------|----------|
| 轻微 | 24小时内 | 班组长 |
| 一般 | 4小时内 | 车间主管 |
| 严重 | 1小时内 | 质量经理 |
| 紧急 | 立即 | 总经理 |

### 3.4 设备管理规则

#### 3.4.1 设备状态定义

| 状态 | 定义 | 标记颜色 |
|------|------|----------|
| 运行 | 正常生产状态 | 绿色 |
| 待机 | 等待生产任务 | 蓝色 |
| 保养 | 进行计划保养 | 黄色 |
| 故障 | 设备故障停机 | 红色 |
| 维修 | 正在进行维修 | 橙色 |

#### 3.4.2 设备维护策略

| 维护类型 | 实施时机 | 内容 | 责任人 |
|----------|----------|------|--------|
| 日常点检 | 每班次/每日 | 设备运行状态检查、清洁 | 操作工 |
| 定期保养 | 按计划周期 | 润滑、紧固、更换易损件 | 设备工程师 |
| 预防性维修 | 故障前 | 基于运行时间的主动维修 | 设备工程师 |
| 预测性维修 | 故障前兆 | 基于状态监测的精准维修 | 设备工程师 |
| 故障维修 | 故障后 | 故障诊断与修复 | 维修人员 |

#### 3.4.3 设备故障响应规则

| 故障等级 | 响应时间 | 修复时间 | 处理流程 |
|----------|----------|----------|----------|
| 轻微 | 2小时内 | 8小时内 | 日常维修 |
| 一般 | 30分钟内 | 4小时内 | 计划维修 |
| 严重 | 15分钟内 | 2小时内 | 紧急抢修 |
| 关键 | 立即 | 1小时内 | 启动应急预案 |

### 3.5 人员配置规则

#### 3.5.1 技能矩阵规则

- 每个岗位至少配置2名以上合格操作人员
- 关键工序操作人员必须持证上岗
- 新员工须通过技能鉴定后方可独立作业
- 每季度进行技能复核

#### 3.5.2 班次配置规则

| 班次 | 时间段 | 人员配置要求 |
|------|--------|--------------|
| 白班 | 08:00-16:00 | 满编配置 |
| 中班 | 16:00-24:00 | 满编配置或轮换 |
| 夜班 | 00:00-08:00 | 减少配置，需注意安全 |

#### 3.5.3 加班规则

- 单日加班不超过4小时
- 每周加班不超过12小时
- 连续加班不超过3天
- 加班须提前申请并批准

### 3.6 物料管理规则

#### 3.6.1 物料配送规则

- 直供物料：JIT配送到工位
- 线边仓物料：定时配送
- 紧急缺料：2小时内配送到位

#### 3.6.2 物料追溯规则

- 所有物料须有批次标识
- 批次追溯记录保存至少3年
- 不良品必须追溯同批次物料

#### 3.6.3 物料损耗规则

| 物料类型 | 允许损耗率 |
|----------|------------|
| 原材料 | 2-5% |
| 辅料 | 3-8% |
| 包材 | 5-10% |
| 特殊物料 | 按技术文件规定 |
