# 医疗健康领域本体设计

## 概述

本文档定义医疗健康领域的核心本体模型，用于指导医疗知识库构建和智能化应用。涵盖临床诊疗、药品管理、健康管理、医疗质量管理等领域。

---

## 一、概念定义 (Concepts)

### 1.1 核心实体

#### 1.1.1 患者主体

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 患者 | Patient | 接受医疗服务的自然人 | 患者编号、姓名、性别、年龄、身份证号、联系方式、医保类型、过敏史 |
| 住院患者 | Inpatient | 办理住院手续接受治疗的患者 | 住院号、入院日期、科室、床位、主治医师、诊断信息 |
| 门诊患者 | Outpatient | 在门诊接受诊疗的患者 | 门诊号、就诊日期、接诊医师、诊断、治疗方案 |
| 急诊患者 | Emergency Patient | 急诊科接收的患者 | 急诊号、到院时间、病情分级、生命体征、抢救记录 |
| 慢病患者 | Chronic Patient | 患有慢性疾病的患者 | 患者编号、慢病类型、病程年限、随访计划、用药方案 |

#### 1.1.2 医护人员

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 医师 | Physician | 具备执业资格的医疗专业人员 | 医师编号、姓名、职称、科室、执业范围、资质证书 |
| 主治医师 | Attending Physician | 负责患者诊疗全过程的责任医师 | 医师编号、患者编号、诊疗方案制定权 |
| 主任医师 | Chief Physician | 科室最高级别医师，负责疑难病例会诊 | 医师编号、科室、专长领域、会诊记录 |
| 护士 | Nurse | 从事护理工作的专业人员 | 护士编号、姓名、职称、护理等级、执业科室 |
| 药师 | Pharmacist | 从事药学服务的专业人员 | 药师编号、姓名、职称、执业范围、处方审核权限 |

#### 1.1.3 医疗机构

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 医院 | Hospital | 提供医疗服务的医疗机构 | 医院编号、医院名称、等级（甲/乙/丙）、科室设置、床位数 |
| 科室 | Department | 医院内按专业划分的诊疗单元 | 科室编号、科室名称、所属医院、学科分类、负责人 |
| 门诊 | Outpatient Department | 提供门诊服务的科室 | 门诊编号、开放时间、接诊能力、预约规则 |
| 住院部 | Inpatient Department | 提供住院服务的科室 | 住院部编号、床位数、护理单元、住院流程 |
| 急诊科 | Emergency Department | 24小时提供急诊服务的科室 | 急诊编号、抢救能力等级、应急响应机制 |

#### 1.1.4 药品与医疗器械

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 药品 | Medicine | 用于预防、治疗、诊断疾病的物质 | 药品编号、通用名称、商品名称、剂型、规格、生产厂家、批准文号 |
| 处方药 | Prescription Drug | 需医师开具处方才能使用的药品 | 药品编号、处方要求、用药指导、禁忌症 |
| 非处方药 | OTC Drug | 无需处方即可购买的药品 | 药品编号、适应症、用法用量、不良反应 |
| 基本药物 | Essential Medicine | 国家基本保障的常用药品 | 药品编号、基药目录编号、价格、报销比例 |
| 医疗器械 | Medical Device | 用于医疗目的的仪器、设备、器具 | 器械编号、名称、分类、适用范围、使用说明 |

#### 1.1.5 诊疗活动

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 诊断 | Diagnosis | 判断患者疾病状态的过程 | 诊断编号、患者编号、诊断类型、主要诊断、辅助诊断、诊断依据 |
| 病历 | Medical Record | 记录患者诊疗过程的文档 | 病历编号、患者编号、就诊类型、主诉、病史、检查、诊断、医嘱 |
| 处方 | Prescription | 医师开具的用药指令 | 处方编号、患者编号、医师编号、药品列表、用法用量、处方日期 |
| 检验 | Laboratory Test | 通过实验室检查获取患者数据 | 检验编号、患者编号、检验项目、检验结果、参考值、报告时间 |
| 影像检查 | Imaging Examination | 通过医学影像设备进行检查 | 检查编号、患者编号、检查类型、影像所见、诊断意见 |

#### 1.1.6 健康管理

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 健康档案 | Health Record | 记录个人健康信息的档案 | 档案编号、个人信息、健康史、体检记录、随访记录 |
| 体检报告 | Physical Exam Report | 健康体检的结果报告 | 报告编号、体检日期、检查项目、异常指标、健康建议 |
| 随访记录 | Follow-up Record | 对患者进行定期追踪的记录 | 记录编号、患者编号、随访日期、随访内容、下次随访时间 |
| 健康评估 | Health Assessment | 对健康状态进行综合评估 | 评估编号、评估日期、评估维度、评分、风险等级 |
| 健康干预 | Health Intervention | 为改善健康状态采取的措施 | 干预编号、干预类型、干预内容、执行情况、效果评价 |

## 三、规则定义 (Rules)

### 3.1 诊断流程规则

#### 3.1.1 诊断流程

```
诊断流程:
├── 步骤1: 收集病史
│   ├── 主诉（患者主要不适）
│   ├── 现病史（本次疾病发展过程）
│   ├── 既往史（既往疾病史）
│   └── 家族史（遗传性疾病史）
├── 步骤2: 体格检查
│   ├── 生命体征（体温/脉搏/呼吸/血压）
│   ├── 一般状态（意识/营养/发育）
│   └── 各系统检查
├── 步骤3: 辅助检查
│   ├── 实验室检验（血/尿/粪常规，生化等）
│   ├── 影像学检查（X线/CT/MRI/超声）
│   └── 特殊检查（心电图/内镜等）
├── 步骤4: 综合分析
│   ├── 汇总检查结果
│   ├── 鉴别诊断分析
│   └── 确定诊断
└── 步骤5: 诊断记录
    ├── 主要诊断
    ├── 辅助诊断
    └── 诊断依据
```

#### 3.1.2 诊断分类规则

```
规则1: 诊断类型判定
  IF 病情为本次就诊的主要原因
  THEN 为"主要诊断"
  
  IF 检查发现的其他疾病
  THEN 为"辅助诊断"

规则2: 诊断级别判定
  IF 根据症状和体征初步判断
  THEN 为"初步诊断"
  
  IF 完成辅助检查后判断
  THEN 为"临床诊断"
  
  IF 病理或病原学确认
  THEN 为"确认诊断"

规则3: 诊断编码规则
  根据ICD-10编码规范
  - 第一字母为疾病大类
  - 后续数字为具体疾病编码
```

### 3.2 处方规则

#### 3.2.1 处方开具规则

```
规则1: 处方权限
  IF 医师职称为主治医师及以上
  THEN 具有处方权
  ELSE 无处方权

规则2: 处方内容完整性
  处方必须包含:
  ├── 患者信息（姓名/年龄/性别）
  ├── 诊断信息
  ├── 药品信息（名称/剂型/规格/数量）
  ├── 用法用量（每次剂量/给药频次/给药途径/疗程）
  └── 医师签名及日期

规则3: 处方限量
  门（急）诊一般不超过7日常用量
  急诊一般不超过3日常用量
  慢性病可适当延长（不超过30日）
```

#### 3.2.2 药品配伍规则

```
规则1: 禁忌药物检查
  IF 患者有某药物过敏史
  AND 处方包含该药物
  THEN 拒绝处方并提示"药物过敏"

规则2: 相互作用检查
  IF 同时使用会产生严重相互作用的药物
  THEN 拒绝处方并提示"药物相互作用警告"
  
  示例:
  ├── 华法林 + 阿司匹林 → 出血风险增加
  ├── 氨基糖苷类 + 呋塞米 → 耳毒性增加
  └── ACEI + 钾盐 → 高钾血症

规则3: 配伍禁忌检查
  IF 处方中药物存在配伍禁忌
  THEN 拒绝处方并提示"配伍禁忌"

规则4: 重复用药检查
  IF 处方包含相同活性成分的多种药物
  THEN 提示"重复用药"
```

#### 3.2.3 特殊人群用药规则

```
规则1: 儿童用药
  根据年龄和体重计算剂量
  禁用或慎用某些成人药物
  注意剂型适宜性

规则2: 老年人用药
  剂量通常为成人剂量的1/2～3/4
  注意肝肾功能
  简化用药方案

规则3: 孕妇用药
  禁用可能致畸的药物
  权衡母体获益与胎儿风险
  参考FDA妊娠分级

规则4: 哺乳期用药
  注意药物乳汁分泌
  选择安全替代药物
  建议哺乳与用药时间间隔
```

### 3.3 医疗质量控制规则

#### 3.3.1 病历质量规则

```
规则1: 病历时限
  门（急）诊病历: 就诊结束后24小时内完成
  入院记录: 入院后24小时内完成
  首次病程记录: 入院后8小时内完成
  手术记录: 术后24小时内完成
  出院记录: 出院后24小时内完成

规则2: 病历内容完整性
  入院记录必须包含:
  ├── 主诉
  ├── 现病史
  ├── 既往史
  ├── 体格检查
  ├── 辅助检查
  └── 诊断/诊疗计划
```

#### 3.3.2 会诊规则

```
规则1: 会诊指征
  符合以下情况之一应进行会诊:
  ├── 病情复杂或诊断不明确
  ├── 治疗效果不佳
  ├── 涉及多学科疾病
  └── 疑难危重病例

规则2: 会诊时限
  普通会诊: 48小时内完成
  急诊会诊: 立即响应
  疑难会诊: 科主任组织
```

## 四、健康管理 (Health Management)

### 4.1 健康档案管理

#### 4.1.1 健康档案内容

```
健康档案包含:
├── 基本信息
│   ├── 个人基本信息（姓名/性别/出生日期/身份证号）
│   ├── 联系方式
│   └── 医保信息
├── 健康史
│   ├── 既往病史（疾病名称/诊断日期/治疗情况）
│   ├── 手术史（手术名称/手术日期/恢复情况）
│   ├── 过敏史（过敏原/过敏表现/严重程度）
│   └── 输血史（输血日期/输血类型/不良反应）
├── 家族史
│   ├── 父母健康状况
│   ├── 兄弟姐妹健康状况
│   └── 遗传性疾病
├── 生活习惯
│   ├── 饮食习惯
│   ├── 运动习惯
│   ├── 吸烟饮酒情况
│   └── 作息规律
├── 体检记录
│   ├── 各次体检日期
│   ├── 体检项目及结果
│   └── 健康评估
└── 随访记录
```

#### 4.2.2 常见慢病管理要点

```
高血压管理:
├── 诊断标准: 收缩压≥140mmHg 或 舒张压≥90mmHg
├── 管理目标: 血压控制在140/90mmHg以下
├── 随访周期: 至少每3个月1次
├── 关键指标: 血压、BMI、血糖、血脂、尿蛋白
└── 干预措施: 降压药物、低盐饮食、适度运动、戒烟限酒

糖尿病管理:
├── 诊断标准: 空腹血糖≥7.0mmol/L 或 餐后2h血糖≥11.1mmol/L
├── 管理目标: HbA1c<7%
├── 随访周期: 每3个月1次
├── 关键指标: 血糖、HbA1c、血压、血脂、尿微量白蛋白
└── 干预措施: 降糖药物、饮食控制、运动疗法、糖尿病教育

冠心病管理:
├── 管理目标: 预防心血管事件
├── 随访周期: 每6个月1次
├── 关键指标: 血脂、血压、血糖、心电图
└── 干预措施: 抗血小板聚集、ACEI/ARB、β受体阻滞剂、他汀类药物
```
