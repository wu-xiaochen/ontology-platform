# 法律领域本体设计

## 概述

本文档定义法律领域的核心本体模型，用于指导法律知识库构建和智能化应用。涵盖民事、刑事、行政、商事、劳动法等多个法律分支的知识体系与业务规则。

---

## 一、概念定义 (Concepts)

### 1.1 法律主体

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 自然人 | Natural Person | 基于出生而取得民事主体资格的人 | 姓名、性别、出生日期、国籍、身份证号码、住所地 |
| 法人 | Legal Person | 依法成立，具有民事权利能力和民事行为能力的组织 | 统一社会信用代码、名称、住所地、法定代表人、成立日期 |
| 非法人组织 | Unincorporated Organization | 不具有法人资格但能依法以自己的名义从事民事活动的组织 | 名称、住所地、负责人、设立依据 |
| 代理人 | Agent | 依法或依授权代表他人进行民事法律行为的人 | 姓名、证件类型、证件号码、代理权限、代理期限 |
| 法定代表人 | Legal Representative | 依法代表法人行使民事权利和履行民事义务的主要负责人 | 姓名、职务、身份证号码、任命依据 |
| 当事人 | Party | 与案件有利害关系，参与诉讼或仲裁的各方 | 当事人类型、名称/姓名、地址、联系方式 |
| 法院 | Court | 国家审判机关，依法行使审判权 | 法院名称、级别、管辖区域、地址 |
| 检察院 | Procuratorate | 国家法律监督机关，依法行使检察权 | 检察院名称、级别、管辖区域、地址 |

### 1.2 法律文件

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 法律法规 | Laws and Regulations | 国家制定或认可，具有普遍约束力的规范性文件 | 名称、制定机关、效力层级、发布日期、实施日期 |
| 司法解释 | Judicial Interpretation | 最高人民法院、最高人民检察院对法律适用问题的解释 | 名称、发布机关、发布日期、解释要点 |
| 合同 | Contract | 当事人之间设立、变更、终止民事法律关系的协议 | 合同编号、合同名称、合同类型、当事人、签订日期、有效期 |
| 判决书 | Judgment | 法院对案件实体问题作出的权威裁判 | 案号、案件类型、法院、当事人、判决日期、判决结果 |
| 裁定书 | Ruling | 法院在案件审理过程中对程序性事项作出的裁判 | 案号、法院、裁定日期、裁定事项、裁定结果 |
| 仲裁裁决 | Arbitration Award | 仲裁庭对争议问题作出的终局性裁决 | 案号、仲裁机构、仲裁员、裁决日期、裁决内容 |
| 公证书 | Notarial Certificate | 公证机构依法出具的具有法律效力的证明文书 | 公证书编号、公证事项、公证员、公证日期、公证内容 |

### 1.3 法律行为与事实

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 民事法律行为 | Civil Legal Act | 民事主体通过意思表示设立、变更、终止民事法律关系的行为 | 行为类型、意思表示内容、效力状态 |
| 行政行为 | Administrative Act | 行政机关依法行使行政职权作出的产生行政法律效果的行为 | 行为类型、作出机关、相对人、效力时间 |
| 犯罪行为 | Criminal Act | 危害社会、违反刑法规定的行为 | 罪名、犯罪构成要件、法定刑 |
| 侵权行为 | Tortious Act | 侵害他人合法权益应当承担民事责任的行为 | 侵权类型、侵权事实、损害结果、因果关系 |
| 违约行为 | Breach of Contract | 当事人不履行合同义务或者履行合同义务不符合约定的行为 | 违约类型、违约事实、责任承担方式 |
| 法律事件 | Legal Event | 不以当事人意志为转移，能够引起法律关系产生、变更、消灭的客观现象 | 事件类型、发生时间、法律后果 |

### 1.4 权利与义务

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 物权 | Property Right | 权利人依法对特定物享有直接支配和排他的权利 | 物权类型、标的、权利内容、公示方式 |
| 债权 | Creditor's Right | 债权人请求债务人为特定行为的权利 | 债权类型、债务人、债权内容、诉讼时效 |
| 知识产权 | Intellectual Property | 智力成果所有人依法享有的专有权利 | 权利类型、权利人、保护期限、权利范围 |
| 人格权 | Personality Right | 民事主体享有的以人格利益为客体的基本民事权利 | 权利类型、权利内容、侵权认定 |
| 股权 | Equity Right | 股东对公司享有的综合性权利 | 股权比例、权利内容、义务内容 |
| 义务 | Obligation | 法律上应当作为或不作为的约束 | 义务类型、义务人、义务内容、履行方式 |
| 责任 | Liability | 违反法定义务或合同义务应当承担的法律后果 | 责任类型、责任主体、责任范围、责任方式 |

### 1.5 争议解决

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 民事诉讼 | Civil Litigation | 人民法院在当事人和诉讼参与人参加下，审理民事案件的活动 | 案件类型、管辖法院、诉讼阶段、审理程序 |
| 刑事诉讼 | Criminal Procedure | 公安机关、检察院、法院依法追究犯罪、惩罚犯罪的活动 | 案件类型、侦查机关、审查起诉、审判程序 |
| 行政诉讼 | Administrative Litigation | 公民、法人或其他组织认为行政行为侵犯其合法权益向法院提起的诉讼 | 案件类型、被告、管辖法院、诉讼请求 |
| 仲裁 | Arbitration | 当事人自愿将争议提交仲裁机构进行裁判的争议解决方式 | 仲裁机构、仲裁规则、仲裁员、仲裁程序 |
| 调解 | Mediation | 中立第三方协助当事人达成和解的争议解决方式 | 调解类型、调解机构、调解结果 |
| 行政复议 | Administrative Reconsideration | 公民、法人或其他组织认为行政行为违法向行政机关申请审查的制度 | 申请期限、受理机关、审查标准、决定类型 |

## 四、规律 (Laws)

### 4.1 诉讼时效规律

**定义**: 权利人在法定期间内不行使权利，法律规定其权利不受保护的制度。

**基本时效**:
```
一般诉讼时效：3年（自权利人知道或应当知道权利受到损害以及义务人之日起计算）
最长诉讼时效：20年（自权利受到损害之日起计算）
```

**特殊时效**:
| 类型 | 时效 | 起算点 |
|------|------|--------|
| 合同履行争议 | 3年 | 合同履行期限届满 |
| 侵权损害赔偿 | 3年 | 知道或应当知道侵权行为和损害结果 |
| 继承纠纷 | 3年 | 知道或应当知道继承权被侵害 |
| 劳动争议 | 1年 | 知道或应当知道劳动权利被侵害 |

**时效中止与中断**:
- 中止：权利人向法院请求、权利人主张权利、义务人同意履行
- 中断：不可抗力、权利人被控制

### 4.2 举证责任分配规律

**定义**: 当事人对自己提出的诉讼请求所依据的事实或者反驳对方诉讼请求所依据的事实，应当提供证据证明。

**基本原则**:
```
"谁主张，谁举证"
```

**举证责任分配**:
| 争议类型 | 主张权利存在 | 主张权利不存在 |
|----------|--------------|----------------|
| 侵权纠纷 | 原告证明侵权行为、损害结果、因果关系、过错 | 被告证明免责事由 |
| 合同纠纷 | 原告证明合同成立生效、履行情况 | 被告证明合同无效、已履行、时效抗辩 |
| 劳动争议 | 劳动者证明劳动关系、损害事实 | 用人单位证明合法解除、已支付报酬 |

**举证责任倒置**:
- 高度危险作业：被告证明受害人故意
- 环境污染：被告证明污染行为与损害无因果关系
- 建筑物倒塌：建设单位证明无过错

### 4.3 管辖权确定规律

**定义**: 确定案件由哪一个法院或仲裁机构受理的规则。

**民事诉讼管辖**:
```
级别管辖：
- 基层法院：第一审民事案件（除法律规定外）
- 中级法院：重大涉外案件、本辖区内有重大影响的案件
- 高级法院：全省（自治区、直辖市）有重大影响的案件
- 最高法院：全国有重大影响的案件

地域管辖：
- 被告住所地法院管辖
- 合同履行地法院管辖（合同约定优先）
- 侵权行为地法院管辖
- 财产所在地法院管辖（财产纠纷）
```

**仲裁管辖**:
```
- 双方达成仲裁协议
- 仲裁协议约定的仲裁机构
- 仲裁协议无效或不明确时由法院管辖
```

### 4.4 法律效力层级

**定义**: 不同法律文件之间的效力等级关系。

```
效力层级（从高到低）：
宪法 > 法律 > 行政法规 > 地方性法规 > 规章
同一层级：新法优于旧法、特别法优于一般法
```

## 五、业务规则 (Rules)

### 5.1 起诉条件

| 条件 | 要求 |
|------|------|
| 原告资格 | 与本案有直接利害关系的公民、法人或其他组织 |
| 被告明确 | 有明确的被告 |
| 诉讼请求具体 | 有具体的诉讼请求和事实、理由 |
| 管辖法院 | 属于法院受理民事诉讼的范围和受诉法院管辖 |
| 形式要件 | 提交起诉状，书写确有困难的可以口头起诉 |

### 5.2 立案标准

| 案件类型 | 立案条件 | 所需材料 |
|----------|----------|----------|
| 民事案件 | 符合起诉条件 | 起诉状、证据清单、身份证明、委托手续 |
| 刑事案件 | 有犯罪事实需要追究刑事责任 | 报案材料、证据、犯罪嫌疑人信息 |
| 行政案件 | 属于受诉法院管辖 | 起诉状、行政行为依据、身份证明 |
| 执行案件 | 生效法律文书确定的义务未履行 | 执行申请书、生效法律文书、证据 |

### 5.3 诉讼费用

| 案件类型 | 费用标准 |
|----------|----------|
| 财产案件 | 2.5%（≤1万元） + 2%（1-10万元） + 1.5%（10-20万元）...分段累加 |
| 非财产案件 | 离婚案件50-300元、侵害姓名权等100-500元 |
| 知识产权案件 | 发明专利1000元、实用新型500元、商标500元 |
| 劳动争议案件 | 10元（简易程序免交） |
| 行政案件 | 50-100元 |

**减免政策**:
- 困难当事人可申请缓交、减交或免交诉讼费
- 赡养费、抚养费等案件免交诉讼费

### 5.4 诉讼时效中断规则

**法定中断事由**:
| 事由 | 法律效果 |
|------|----------|
| 权利人向法院起诉 | 诉讼时效从提交诉状之日起中断 |
| 权利人向义务人提出履行请求 | 诉讼时效从提出请求之日起中断 |
| 义务人同意履行义务 | 诉讼时效从同意之日起中断 |
| 权利人申请仲裁 | 诉讼时效从申请仲裁之日起中断 |

**时效抗辩**:
- 诉讼时效抗辩应在答辩期间提出
- 法院不得主动适用诉讼时效
- 诉讼时效届满，义务人可拒绝履行

### 5.5 证据规则

**证据种类**:
```
书证、物证、视听资料、电子数据、证人证言、鉴定意见、勘验笔录、当事人陈述
```

**证据效力**:
| 证据类型 | 效力等级 |
|----------|----------|
| 公文书证 | 较高（推定真实） |
| 鉴定意见 | 较高（专业判断） |
| 公证文书 | 较高（推定真实） |
| 原始证据 | 优于传来证据 |
| 直接证据 | 优于间接证据 |

**证据审查**:
- 证据必须具有合法性、真实性、关联性
- 非法证据排除：严重侵害他人合法权益、以严重违背公序良俗的方式取得
- 证据不足以证明事实的，承担举证不能的后果

### 5.6 财产保全规则

**保全类型**:
| 类型 | 适用情形 | 担保要求 |
|------|----------|----------|
| 诉前保全 | 紧急情况，不立即保全会使合法权益受到难以弥补的损害 | 必须提供担保 |
| 诉讼保全 | 判决难以执行或造成其他损害 | 可以要求提供担保 |

**保全范围**:
- 请求范围或与本案有关的财物
- 保全限于请求的范围或与案件有关的财物

**保全期限**:
| 保全类型 | 期限 |
|----------|------|
| 银行存款 | 1年 |
| 动产 | 2年 |
| 不动产 | 3年 |
