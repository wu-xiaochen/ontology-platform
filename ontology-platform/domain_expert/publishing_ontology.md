# 出版领域本体 (Publishing Ontology)

## 概述

本文档定义出版领域的核心本体模型，涵盖图书、期刊、报纸、电子出版物等多种出版形态，用于指导出版内容管理、版权运营、智能编目、发行分析等应用场景。

---

## 一、概念定义 (Concepts)

### 1.1 出版主体

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 出版社 | Publisher | 从事出版物编辑、出版、发行的机构 | 出版社编码、名称、级别、资质、地址、联系方式 |
| 作者 | Author | 创作出版内容的个人 | 作者ID、笔名、擅长领域、作品列表、知名度 |
| 编辑 | Editor | 从事内容编辑加工的专业人员 | 编辑ID、姓名、专业领域、职称、审稿风格 |
| 译者 | Translator | 从事内容翻译的专业人员 | 译者ID、擅长语言对、翻译风格、代表译作 |
| 印刷厂 | Printing Factory | 负责出版物印刷制作的机构 | 印刷厂编码、名称、资质、产能、设备 |
| 发行商 | Distributor | 负责出版物发行分销的机构 | 发行商编码、名称、渠道网络、物流能力 |
| 书商 | Bookseller | 零售出版物给终端读者的商家 | 书商店编码、名称、类型、规模、线上/线下 |
| 版权代理 | Copyright Agent | 代理版权交易的中介机构 | 代理ID、名称、代理范围、代表案例 |

### 1.2 出版物

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 图书 | Book | 以纸本或电子形式出版的书籍 | ISBN、书名、作者、定价、页数、开本、字数 |
| 纸质书 | Paperback/Hardcover | 传统纸质印刷书籍 | 纸张规格、装订方式、印刷工艺 |
| 电子书 | E-book | 数字格式的书籍 | 格式(epub/mobi/pdf)、 DRM、 价格 |
| 期刊 | Periodical | 定期出版的刊物 | ISSN、刊名、刊期(周刊/月刊/季刊)、定价 |
| 杂志 | Magazine | 综合性期刊内容 | 类别、发行周期、页数、广告页数 |
| 学术期刊 | Academic Journal | 学术研究论文发表刊物 | 影响因子、收录数据库、审稿制度 |
| 报纸 | Newspaper | 每日或定期发行的新闻刊物 | 刊号、发行周期、版数、发行量 |
| 电子出版物 | Digital Publication | 以数字形式出版的各类内容 | 格式、平台、定价模式、访问方式 |
| 有声书 | Audiobook | 以音频形式出版的内容 | 时长、朗读人、格式、价格 |
| 漫画 | Comic | 以图像为主的内容出版物 | 画风、篇幅(单行本/连载)、目标读者 |

### 1.3 出版流程相关

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 书稿 | Manuscript | 作者提交的原始稿件 | 标题、字数、题材、完稿状态、提交日期 |
| 选题 | Book Proposal | 出版选题策划方案 | 选题名称、作者、市场定位、预估成本、预期销量 |
| 审稿 | Manuscript Review | 对书稿进行内容审核 | 审稿人、审稿意见、审稿周期、通过/修改/否决 |
| 编辑加工 | Editing | 对稿件进行文字、内容加工 | 编辑环节、校对遍数、加工标准 |
| 校对 | Proofreading | 纠正书稿中的文字错误 | 校次、差错率、校对人员 |
| 版式设计 | Layout Design | 出版物版式编排设计 | 开本、版心、字体、行距、图片排版 |
| 封面设计 | Cover Design | 出版物封面创意设计 | 设计风格、元素、色彩、风格定位 |
| 申请书号 | ISBN Application | 申请出版物唯一标识号 | ISBN号、条形码、CIP数据 |
| 印刷 | Printing | 批量复制生产出版物 | 印刷工艺、纸张、用墨量、印数 |
| 装订 | Binding | 将印张装订成册 | 装订方式(胶装/线装/精装)、工艺 |
| 发行 | Distribution | 出版物上市销售流通 | 发行渠道、发行区域、发行折扣 |
| 新书发布会 | Book Launch | 新书出版后的宣传推广活动 | 活动时间、地点、嘉宾、媒体 |

### 1.4 版权与知识产权

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 版权 | Copyright | 作品的法定排他性权利 | 权利类型、期限、登记号 |
| 著作权 | Copyright | 创作者对作品的专有权利 | 作者、权利内容、转让情况 |
| 版权转让 | Copyright Transfer | 版权从一方转移到另一方 | 转让范围、期限、地域、价格 |
| 版权许可 | Copyright License | 授权他人使用作品 | 许可类型(独占/普通)、范围、费用 |
| 版税 | Royalty | 按销量支付给作者的报酬 | 版税率、版税结算周期、最低保证金 |
| 稿酬 | Author Fee | 一次性支付给作者的报酬 | 稿酬标准、计算方式、支付时间 |
| ISBN | ISBN | 国际标准书号 | 13位数字、条形码 |
| ISSN | ISSN | 国际标准连续出版物号 | 8位数字 |
| CIP | CIP | 图书在版编目数据 | 分类编目、主题词 |

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|

| 批发 | Wholesale | 大批量销售给中间商 | 批发折扣、批量门槛、账期 |

## 二、规律 (Laws)

### 2.1 图书市场"二八定律"

**定义**: 出版市场中约20%的品种贡献80%的码洋(销售额)，头部效应显著。

**规律表现**:
- 畅销书占出版社总销售额的比重大
- 长尾图书虽然品种多但销量有限
- 出版社倾向于追逐确定性的畅销书

**应对策略**:
- 多元化产品线平衡风险
- 重视长尾渠道(图书馆、线上)
- 精细化运营小众品类

### 2.2 图书定价规律

**定价公式**:
```
图书定价 = (印刷成本 + 版税) × (1 + 出版社利润率) / (1 - 发行折扣)
```

**定价影响因素**:
| 因素 | 影响 |
|------|------|
| 字数 | 正相关，字数多定价高 |
| 纸张 | 轻型纸略低于纯质纸 |
| 装订 | 精装约平装定价1.5-2倍 |
| 品牌 | 知名作者/出版社可定价更高 |
| 印数 | 量大可降低单本定价 |
| 渠道 | 线上渠道折扣空间更大 |

**行业参考**:
- 大众图书: 字数/10 ~ 字数/8 元
- 教材教辅: 成本加成15-25%
- 学术专著: 字数/5 ~ 字数/3 元

### 2.3 版税计算规律

**版税公式**:
```
版税 = 定价 × 实际销售数量 × 版税率
```

**版税率参考**:
| 类型 | 首版版税率 | 重版版税率 |
|------|------------|------------|
| 知名作家 | 10-15% | 12-18% |
| 普通作者 | 6-10% | 8-12% |
| 翻译作品 | 4-8% | 6-10% |
| 学术专著 | 3-6% | 5-8% |
| 教材 | 5-8% | 7-10% |

**保底与预付**:
- 首印保底: 出版社承诺最低版税支付
- 预付版税: 出版前支付部分版税(版税预支)

### 2.4 图书生命周期

**定义**: 图书从出版到退出市场的完整周期。

**阶段划分**:
```
出版上市期(1-3月) → 销售高峰期(3-12月) → 长尾销售期(1-3年) → 绝版/重版
```

**生命周期特征**:
- 首年销量约占生命周期销量60-70%
- 三年后进入长尾销售
- 学术专著生命周期较长(5-10年)
- 畅销书可多次重版

### 2.5 发行折扣体系

**折扣层级**:
```
出版社 → 一级批发 → 二级批发 → 零售商 → 读者
  55-65折    60-70折    65-75折     定价
```

**常见折扣策略**:
| 渠道 | 折扣范围 | 说明 |
|------|----------|------|
| 实体书店 | 68-75折 | 标准渠道 |
| 网上书店 | 60-70折 | 薄利多销 |
| 图书馆 | 55-65折 | 馆配专供 |
| 团购 | 50-65折 | 大批量采购 |
| 直销 | 60-80折 | 出版社自营 |

### 2.6 畅销书形成规律

**关键要素**:
1. 内容质量: 口碑传播基础
2. 营销投入: 首月决定生死
3. 话题性: 社交媒体传播
4. 时机: 节点营销(节日、考试)
5. 渠道配合: 重点陈列、推荐位
6. 价格策略: 首发优惠

**"上市决定论"**:
- 首月销量决定图书命运
- 三个月后难有大突破
- 再版难度大

## 三、业务规则 (Rules)

### 3.1 选题策划规则

| 条件 | 要求 |
|------|------|
| 基本信息 | 选题名称、作者、字数、内容简介 |
| 市场分析 | 同类书销售情况、目标读者定位 |
| 成本预算 | 预估稿酬、印刷、编辑成本 |
| 预期收益 | 预估销量、定价、利润 |
| 出版计划 | 交稿时间、出版时间、印数 |
| 审批要求 | 重大选题需备案 |

**选题否决条件**:
- 内容违反法律法规
- 侵犯他人版权
- 市场定位不清晰
- 成本收益倒挂

### 3.2 书号申请规则

| 出版社类型 | 年书号数量限制 |
|------------|----------------|
| 中央级出版社 | 根据级别200-2000个 |
| 省级出版社 | 100-500个 |
| 大学出版社 | 50-200个 |
| 电子出版社 | 按业务规模审批 |

**书号使用规则**:
- 一书一号，严禁一号多用
- 套书、丛书需分别申请
- 停版图书号作废后不可复用
- 电子书号与纸质书号分开

### 3.3 审稿流程规则

```
来稿登记 → 初审 → 复审 → 终审 → 发稿
```

**各环节要求**:
| 环节 | 责任人 | 审核重点 |
|------|--------|----------|
| 初审 | 编辑 | 内容完整性、基本质量 |
| 复审 | 室主任 | 政治导向、学术水平 |
| 终审 | 社领导 | 整体把关、重大问题 |

**审稿时限**:
- 初审: 收到稿件30天内
- 复审: 初审通过15天内
- 终审: 复审通过10天内

### 3.4 印刷管理规则

| 印数确定因素 | 说明 |
|--------------|------|
| 首印数 | 通常3000-10000册 |
| 加印条件 | 销量达60%时考虑加印 |
| 最低印数 | 3000册(成本考量) |
| 绝版条件 | 销量持续低迷1年以上 |

**印刷质量标准**:
- 纸张符合合同约定
- 印刷清晰无重影
- 装订平整无脱页
- 差错率<0.01%(万分之一)

### 3.5 发行渠道管理规则

**发货原则**:
- 新书先发主渠道(新华、民营大鳄)
- 重点书重点发货
- 滞销书控制发货
- 断版书及时通知

**退货率控制**:
| 渠道类型 | 退货率控制 |
|----------|------------|
| 新华书店 | ≤15% |
| 民营渠道 | ≤20% |
| 网上书店 | ≤10% |
| 团购 | ≤5% |

**账期管理**:
- 网上书店: 60-90天
- 实体书店: 90-180天
- 团购: 货到付款或30天

### 3.6 版权合同规则

**合同必备条款**:
| 条款 | 内容 |
|------|------|
| 作品信息 | 名称、字数、形式 |
| 权利范围 | 出版权、电子版权、翻译权等 |
| 期限 | 通常5-10年 |
| 版税 | 税率、计算方式、支付时间 |
| 保证金 | 预付金额、抵扣方式 |
| 双方责任 | 交稿时间、出版时间 |
| 违约责任 | 违约金、合同解除 |

**数字版权**:
- 电子书: 需明确授权
- 有声书: 需单独授权
- 翻译权: 按语种分别授权

### 3.7 出版物质量管理规则

**质量标准**:
| 指标 | 合格标准 |
|------|----------|
| 文字差错率 | ≤0.01% |
| 编排规范 | 符合国家标准 |
| 印刷质量 | 合格品以上 |
| 装订质量 | 合格品以上 |
| 环保标准 | 符合绿色印刷要求 |

**质量检查**:
- 三审三校制度
- 付印前样书检查
- 上市前抽检
- 读者反馈跟踪
