# 农业领域本体设计

## 概述

本文档定义农业领域的核心本体模型，用于指导农业知识库构建和智能化应用。涵盖种植业、养殖业、农产品加工等细分领域，覆盖从选种到收获的完整农业生产周期。

---

## 二、规律 (Laws)

### 2.1 作物生长积温法则

**定义**: 作物从播种到成熟需要累积一定的有效积温。

**公式**:
```
有效积温 = Σ(日均温 - 生物学零度)
生育期所需积温 = 播种到成熟的日均温累加值
```

**应用**:
- 积温是划分作物适宜种植区的重要依据
- 不同品种对积温需求不同，决定了适宜种植区域
- 实际积温与所需积温的匹配程度决定产量潜力

**常见作物积温需求**:
| 作物 | 所需积温(°C·d) |
|------|---------------|
| 水稻 | 2400-3200 |
| 小麦 | 1600-2400 |
| 玉米 | 2500-2800 |
| 大豆 | 2400-2800 |
| 棉花 | 3000-3600 |

### 2.2 土壤肥力递减规律

**定义**: 在同一地块上连续种植同一作物，产量会随种植年限增加而递减。

**公式**:
```
产量递减率 = (基准产量 - 实际产量) / 基准产量 × 100%
```

**应对原则**:
- 轮作倒茬：不同作物轮换种植
- 秸秆还田：增加土壤有机质
- 合理施肥：补充土壤养分消耗
- 深耕松土：改善土壤结构

### 2.3 气象灾害影响规律

**定义**: 气象条件对农业生产的影响具有规律性特征。

| 灾害类型 | 影响时期 | 影响机理 | 减产幅度 |
|----------|----------|----------|----------|
| 干旱 | 全生育期 | 水分胁迫影响光合作用 | 20-50% |
| 洪涝 | 生长中后期 | 根系缺氧、倒伏 | 30-60% |
| 低温冷害 | 生长初期 | 影响发芽和幼苗生长 | 15-30% |
| 高温热害 | 开花结实期 | 影响授粉和灌浆 | 20-40% |
| 台风 | 生长中后期 | 植株倒伏、果实脱落 | 30-70% |

### 2.4 轮作增产规律

**定义**: 合理轮作可以改善土壤结构、提高肥力、减少病虫害，从而提高产量。

**轮作原则**:
- 禾本科与豆科作物轮作：固氮养地
- 深根系与浅根系作物轮作：充分利用不同土层养分
- 喜肥作物与养地作物轮作：平衡土壤肥力
- 病虫害寄主作物轮作：减少病虫积累

**典型轮作模式**:
```
水稻-小麦轮作
水稻-油菜轮作
玉米-大豆间作
棉花-小麦套作
```

### 2.5 养殖料肉比规律

**定义**: 养殖动物消耗的饲料与产出的肉、蛋、奶的比例关系。

**公式**:
```
料肉比(FCR) = 消耗饲料量 / 动物增重
料蛋比 = 消耗饲料量 / 产蛋量
料奶比 = 消耗饲料量 / 产奶量
```

**行业基准**:
| 养殖品种 | 料肉比 | 备注 |
|----------|--------|------|
| 肉鸡 | 1.8-2.0:1 | 优质品种可达1.6:1 |
| 肉猪 | 2.5-3.0:1 | 优质品种可达2.4:1 |
| 肉牛 | 6-8:1 | 育肥后期可达5:1 |
| 蛋鸡 | 2.2-2.5:1 | 产蛋高峰期更低 |
| 奶牛 | 0.9-1.2:1 | 优质奶牛可达0.8:1 |

### 2.6 农产品价格波动规律

**定义**: 农产品价格受供求关系、季节性、周期性等因素影响而波动。

**波动类型**:
- **季节性波动**: 收获期价格下降，淡季价格上涨
- **周期性波动**: 生猪价格3-4年一个周期
- **年际波动**: 受气象灾害影响导致的年度间波动

**规律**:
- 鲜活农产品价格弹性小，供求变化导致价格大幅波动
- 粮食价格相对稳定，受政策调控影响大
- 经济作物价格受国际市场影响大

## 三、业务规则 (Rules)

### 3.1 农作物种植管理规则

#### 3.1.1 播期确定规则

| 作物 | 播期确定依据 | 适宜温度 |
|------|--------------|----------|
| 水稻 | 根据前茬收获期和品种生育期倒推 | 日均温稳定通过12°C |
| 小麦 | 根据积温要求和安全越冬确定 | 日均温16-18°C |
| 玉米 | 根据品种熟期和当地积温 | 10cm地温稳定通过10°C |
| 棉花 | 根据霜期倒推 | 5cm地温稳定通过12°C |
| 油菜 | 根据安全越冬和花期避灾 | 日均温18-20°C |

#### 3.1.2 种植密度规则

**确定因素**:
- 品种特性：紧凑型宜密，松散型宜稀
- 土壤肥力：肥地宜密，瘦地宜稀
- 灌溉条件：水浇地宜密，旱地宜稀
- 栽培方式：间作套种宜稀

**常见作物适宜密度**:
| 作物 | 适宜密度(株/亩) | 行株距(m) |
|------|-----------------|-----------|
| 水稻 | 1.8-2.2万 | 0.2×0.15 |
| 小麦 | 15-25万 | 0.2 |
| 玉米 | 3000-4500 | 0.6×0.25-0.33 |
| 大豆 | 1.2-1.8万 | 0.4×0.1-0.15 |
| 棉花 | 3000-5000 | 0.9×0.2-0.3 |

### 3.2 施肥管理规则

#### 3.2.1 施肥原则

| 原则 | 说明 |
|------|------|
| 有机肥为主，化肥为辅 | 改良土壤结构，提高土壤肥力 |
| 测土配方施肥 | 根据土壤养分含量和作物需求确定用量 |
| 底肥为主，追肥为辅 | 保证养分持续供应 |
| 氮磷钾配合 | 平衡营养，避免单一元素过量 |
| 看天看地看苗 | 根据具体情况灵活调整 |

### 3.3 农药使用规则

#### 3.3.1 农药选用原则

| 原则 | 说明 |
|------|------|
| 对症下药 | 根据病虫害种类选择合适药剂 |
| 交替用药 | 避免同一药剂连续使用导致抗性 |
| 精准用药 | 严格按照推荐剂量使用 |
| 安全间隔期 | 收获前停止用药，保证农产品安全 |
| 绿色优先 | 优先使用生物农药和低毒农药 |

#### 3.3.2 禁止使用农药清单

- 国家明令禁止使用的农药：六六六、DDT、甲基对硫磷、甲胺磷等
- 在蔬菜、果树、茶叶上禁止使用的高毒农药：氧化乐果、水胺硫磷等
- 限制使用农药：需按照指定范围使用

#### 3.3.3 农药安全间隔期

| 农药种类 | 作物 | 安全间隔期(天) |
|----------|------|----------------|
