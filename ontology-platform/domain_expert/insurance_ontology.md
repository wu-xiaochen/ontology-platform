# 保险领域本体设计

## 概述

本文档定义保险领域的核心本体模型，用于指导知识库构建和智能化应用。涵盖保险产品、核保理赔、风险管理、客户服务、保险中介、再保险、监管合规等核心业务领域。本体与金融风控领域本体（finance_ontology.md）形成互补，本体侧重保险业务运营层面，风控本体侧重信用风险度量层面。

---

## 一、概念定义 (Concepts)

### 1.1 核心实体

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 保险产品 | Insurance Product | 保险公司提供的风险保障商品 | 产品编码、名称、类型、保障范围、保险期限、保费 |
| 投保人 | Policyholder | 与保险公司签订保险合同的主体 | 客户编码、姓名/名称、证件类型、联系方式、职业 |
| 被保险人 | Insured | 享有保险保障权益的人 | 与投保人关系、健康状况、职业风险等级 |
| 受益人 | Beneficiary | 保险金受领人 | 与被保险人关系、受益顺序、受益份额 |
| 保单 | Policy | 保险合同的书面凭证 | 保单号、承保公司、产品类型、投保日期、生效日期 |
| 保费 | Premium | 投保人支付给保险公司的费用 | 保费金额、缴费方式（年缴/季缴/月缴）、缴费期限 |
| 保险金额 | Sum Insured | 保险公司承担的最高赔偿限额 | 基本保额、保额单位、是否可调整 |
| 理赔 | Claim | 保险事故发生后的赔偿申请 | 赔案号、出险日期、报案日期、赔付金额、案件状态 |
| 核保 | Underwriting | 保险公司评估风险并决定是否承保的过程 | 核保结论（标体/加费/除外/拒保）、核保评分 |
| 保险标的 | Insured Object | 保险合同保障的对象 | 标的类型、标的属性、价值评估 |
| 等待期 | Waiting Period | 保险责任生效前的时间限制 | 等待期天数、责任范围 |
| 免赔额 | Deductible | 保险人不承担赔偿的金额 | 免赔额类型（绝对/相对）、金额 |
| 现金价值 | Cash Value | 保单具有的累计价值 | 当前现金价值、退保金、贷款额度 |

## 二、规律 (Laws)

### 2.1 大数法则

**定义**: 大量随机现象的整体规律性，单个事件的不确定性在大量重复中将趋于稳定。

**保险应用**:
```
预期损失 = Σ(损失金额 × 发生概率) / n
```

**意义**:
- 保费厘定的基础：基于大量同类风险单位的损失数据进行精算
- 风险分散：通过承保大量同质风险降低不确定性
- 稳定性保证：承保数量越大，赔付率波动越小

### 2.2 保险费率厘定原则

**基本公式**:
```
毛保费 = 纯保费 + 附加保费
纯保费 = 预期损失 × 安全系数
附加保费 = 营业费用 + 预期利润 + 风险附加
```

**费率因子**:
| 因子类别 | 具体因素 |
|----------|----------|
| 风险因素 | 年龄、性别、健康状况、职业、使用性质 |
| 车辆因素 | 车龄、车型、排量、零整比 |
| 驾驶因素 | 驾龄、违章记录、出险次数 |
| 地域因素 | 地区、行驶区域、停车环境 |

### 2.3 损失率规律

**定义**: 实际赔付金额与保费收入的比率。

**计算公式**:
```
损失率 = 已决赔款 / 保费收入 × 100%
综合成本率 = 损失率 + 费用率
```

**行业基准**:
| 险种 | 损失率范围 | 综合成本率参考 |
|------|------------|----------------|
| 车险 | 55-70% | 95-105% |
| 医疗险 | 60-80% | 95-110% |
| 财产险 | 40-60% | 85-100% |
| 寿险 | 20-40% | 85-95% |
| 意外险 | 30-50% | 85-100% |

### 2.4 赔付率与保费调整规律

**规律**: 赔付率过高时需调整保费，过低时应回馈客户。

**调整触发线**:
| 综合成本率 | 应对措施 |
|------------|----------|
| < 95% | 盈利状态，可考虑优惠 |
| 95%-100% | 盈亏平衡，监控 |
| 100%-110% | 亏损，评估涨价 |
| > 110% | 严重亏损，必须调整 |

### 2.5 复效与现金价值规律

**复效规律**:
- 宽限期（通常60天）内未缴费，保单效力中止
- 两年内可申请复效，复效后需重新计算等待期
- 复效应补缴保费及利息

**现金价值规律**:
```
现金价值 = 已缴保费 × 现金价值比例 - 已领取金额
```
- 现金价值随缴费年限增长
- 前期现金价值较低，后期逐渐增加
- 退保损失通常在前几年较大

### 2.6 准备金规律

**定义**: 保险公司为履行保险责任而提取的资金储备。

**准备金类型**:
| 准备金类型 | 用途 | 提取依据 |
|------------|------|----------|
| 未到期责任准备金 | 承担未来保险责任 | 保费收入、风险期间 |
| 未决赔款准备金 | 已发生未赔付案件 | 报案数量、赔付预估 |
| 长期责任准备金 | 长期险种责任准备 | 精算模型、死亡率 |
| 保险保障基金 | 应对保险公司风险 | 保费一定比例 |

### 2.7 再保险分散规律

**分出规律**:
```
自留额 = 资本金 × 自留比例
分出额 = 风险敞口 - 自留额
```

**再保险安排原则**:
- 大数法则：通过再保险扩大风险分散面
- 责任分散：单一风险不超过资本金的一定比例
- 成本效益：再保费率应低于自留风险的预期损失

#### 理赔计算规则

**医疗险理赔**:
```
赔付金额 = (合理医疗费用 - 免赔额) × 赔付比例
```
- 社保先赔：社保报销后剩余部分×100%赔付
- 未经社保：医疗费用×60%赔付

**重疾险理赔**:
```
赔付金额 = 确诊即赔（一次性给付）
```
- 确诊合同约定疾病：100%保额
- 达到约定状态：100%保额
- 实施约定手术：100%保额

**车险理赔**:
```
车辆损失 = min(实际修复费用, 保额) - 免赔额
第三者责任 = min(实际损失, 保额) - 免赔额
```

**寿险理赔**:
```
身故金 = 保额（扣除已领取年金）
全残金 = 保额
```

- "购买了医疗险，哪些情况可以理赔？"

- "车险续保需要什么材料？"
- "职业类别5类能投保意外险吗？"
- "什么是保证续保？"

### 5.2 智能核保

根据核保规则自动评估：
- 根据年龄、职业、健康状况给出核保结论
- 识别高风险因素并给出除外/加费建议
- 自动匹配产品与客户风险等级
- 计算自留额和再保险分出额

### 5.3 理赔自动化

根据理赔规则实现：
- 自动识别理赔材料完整性
- 根据条款计算赔付金额
- 识别异常理赔模式，触发调查
- 自动生成理赔调查报告

### 5.4 产品推荐

- 根据投保人/被保险人特征推荐产品

- 根据风险偏好推荐投资型产品
- 根据职业推荐合适的险种

### 5.5 风险管理

- 赔付率监控与预警
