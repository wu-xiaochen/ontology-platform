# 娱乐领域本体设计

## 概述

本文档定义娱乐领域的核心本体模型，用于指导娱乐内容管理、艺人经纪、版权运营等智能化应用。涵盖影视、音乐、游戏、演出、综艺等细分领域。

---

## 一、概念定义 (Concepts)

### 1.1 核心实体

#### 1.1.1 内容产品

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 影视作品 | Film & TV | 电影、电视剧、网络剧等视听内容 | 内容编号、标题、类型、时长、题材、年代、内容分级 |
| 综艺节目 | Variety Show | 电视或网络综艺节目 | 节目编号、节目名称、类型、期数、播出平台、收视率 |
| 音乐作品 | Music | 歌曲、专辑、音乐视频 | 作品编号、标题、类型、时长、语种、发行日期、流派 |
| 游戏产品 | Game | 电子游戏产品 | 产品编号、名称、类型、平台、发行商、评级、上市日期 |

| 概念 | 英文 | 定义 | 关键属性 |

| 版权 | Copyright | 内容作品的知识产权 | 版权编号、作品编号、权利类型、授权范围、有效期限 |

#### 1.1.5 运营指标

| 概念 | 英文 | 定义 | 计算方式/说明 |
|------|------|------|---------------|
| 收视率 | Rating | 节目观看人数占目标观众比例 | 观看人数 / 目标观众总数 × 100% |
| 播放量 | View Count | 内容的观看次数 | 累计播放次数 |
| 票房 | Box Office | 电影票销售总收入 | 累计票房金额 |
| 销量 | Sales | 音乐专辑/衍生品销售数量 | 累计销售数量/金额 |
| 热度指数 | Popularity Index | 内容在社交媒体的讨论热度 | 社交媒体声量加权计算 |
| 艺人商业价值 | Commercial Value | 艺人的商业变现能力 | 代言费 + 带货GMV + 版权收入 |
| 粉丝经济 | Fan Economy | 粉丝群体的消费规模 | 粉丝数 × 人均消费额 |
| 版权收益 | Royalty Income | 版权授权带来的收入 | 授权费用分成 + 版税 |

## 二、规律 (Laws)

### 2.1 内容为王原则

**定义**: 优质内容是娱乐产业的核心竞争力，无论渠道如何变化，好内容始终是吸引用户的根本。

**应用**:
- 平台竞争本质是内容竞争
- 独家优质内容可带来用户增长
- 内容质量决定口碑和长尾收益

### 2.2 马太效应

**定义**: 头部内容/艺人占据大部分资源和收益，强者恒强。

**娱乐应用**:
- 头部艺人获得更多曝光和商业机会
- 顶级IP开发价值远超普通IP
- 平台扶植头部内容获取流量

### 2.3 粉丝经济效应

**定义**: 粉丝群体的忠诚消费是娱乐产业的重要收入来源。

**收入模式**:
- 演唱会门票（粉丝见面会）
- 专辑/周边购买
- 会员订阅（爱奇艺、腾讯视频）
- 直播打赏
- 代言商品带货

**商业逻辑**:
```
粉丝经济价值 = 核心粉丝数量 × 人均年消费 × 粉丝生命周期
```

### 2.4 IP生态开发原则

**定义**: 优质IP应进行多形态、跨平台、持续性开发，最大化商业价值。

**开发路径**:
```
文学作品 → 漫画/动画 → 影视 → 游戏 → 衍生品 → 线下体验
```

**成功要素**:
- 保持IP核心世界观一致性
- 逐步升级的IP培育路径
- 多平台协同放大IP价值

### 2.5 热点周期规律

**定义**: 娱乐热点具有时效性，话题热度呈爆发、持续、衰退曲线。

**周期阶段**:

| 阶段 | 持续时间 | 特征 | 运营策略 |
|------|----------|------|----------|
| 爆发期 | 0-48小时 | 热度急剧上升、全网讨论 | 快速响应、蹭热点 |
| 高峰期 | 3-7天 | 热度达到峰值、深度讨论 | 深度内容、参与话题 |
| 衰退期 | 7-14天 | 热度逐渐下降、转移关注 | 保持存在感 |
| 长尾期 | 14天后 | 仅核心粉丝关注 | 沉淀为经典 |

## 三、业务规则 (Rules)

### 3.1 内容引进规则

#### 3.1.1 版权采购规则

| 维度 | 规则描述 | 阈值 |
|------|----------|------|
| 内容合规 | 不得包含违规内容 | 通过广电审核/平台自查 |
| 版权清晰 | 版权链完整 | 拥有完整授权链 |
| 独家性 | 是否独家授权 | 独家价格通常高30-50% |
| 时效性 | 内容新鲜度 | 优先采购近期热门内容 |
| 性价比 | ROI评估 | 预期收益 > 采购成本 |

#### 3.1.2 内容分级规则

| 级别 | 名称 | 观众群体 | 典型内容 |
|------|------|----------|----------|
| G | 普遍级 | 全年龄 | 儿童节目、合家欢 |
| PG | 辅导级 | 需家长指导 | 青春剧、喜剧 |
| PG-13 | 家长级 | 13岁以上 | 动作片、爱情片 |
| R | 限制级 | 17岁以上 | 成人内容 |

### 3.2 艺人经纪规则

#### 3.2.1 艺人签约规则

| 维度 | 规则描述 | 标准 |
|------|----------|------|
| 年龄 | 练习生/新人年龄要求 | 16-25岁 |
| 颜值 | 形象气质要求 | 符合大众审美 |
| 才艺 | 专业能力 | 歌舞/表演至少一项突出 |
| 品行 | 道德合规 | 无违法犯罪记录 |
| 潜力 | 商业价值评估 | 经过市场验证 |

#### 3.2.2 艺人分成规则

**典型经纪约分成模式**:
```
模式一：保底+分成
- 保底收入：签约时约定的最低收入
- 超额部分：经纪公司50-70%，艺人30-50%

模式二：纯分成
- 无保底，按实际收入分成
- 艺人级别越高，分成比例越高

模式三：阶梯分成
- 收入0-100万：艺人40%/公司60%
- 收入100-500万：艺人50%/公司50%
- 收入500万+：艺人60%/公司40%
```

#### 3.2.3 解约赔偿规则

| 违约情况 | 赔偿标准 |
|----------|----------|
| 提前解约（艺人原因） | 支付剩余合同期收入的30-50% |
| 违规解约 | 支付全部投入+违约金 |
| 合同期满正常解约 | 按协议约定办理 |

### 3.3 票房/播放分账规则

#### 3.3.1 电影票房分账

**中国电影票房分账比例**:
```
总票房
├── 国家电影事业发展专项资金（5%）
└── 增值税
    └── 影院分账（50-52%）
        ├── 影院（50%）
        └── 院线（5-7%）
    └── 片方分账（38-43%）
        └── 发行方（通常分5%）
```

**网络电影分账**:
```
会员付费分账 = 有效会员观看时长 × 分账单价
平台采购：一次性买断或保底+分成
```

#### 3.3.2 音乐版权收益

| 收益类型 | 分成比例 |
|----------|----------|
| 数字播放（流媒体） | 词曲50%+演唱50%，平台通常抽50% |
| 版权授权 | 双方协商，通常词曲40%/演唱60% |
| 线下演出 | 主办方扣除成本后与艺人分成 |
| 衍生品 | 版权方与被授权方协商 |

### 3.4 品牌代言规则

#### 3.4.1 代言费用评估

**核心公式**:
```
代言费 = 基础费 × 品类系数 × 艺人系数 × 独家系数

品类系数：
- 快消品：1.0
- 3C数码：1.2
- 奢侈品：1.5
- 汽车：2.0

艺人系数：
- S+级：3.0
- S级：2.0
- A级：1.5
- B级：1.0
- C级：0.8

独家系数：
- 全品类独家：2.0
- 单品类独家：1.5
- 非独家：1.0
```

#### 3.4.2 品牌匹配规则

| 艺人等级 | 可代言品牌类型 |
|----------|----------------|
| S+ | 一线奢侈品牌、国际知名品牌 |
| S | 国内一线品牌、新锐品牌 |
| A | 知名消费品牌 |
| B | 成长型品牌、细分品牌 |
| C-D | 试用/体验型合作 |

## 四、应用场景 (Applications)

### 4.1 智能推荐系统

**应用**: 基于用户画像和内容标签的个性化推荐

**核心逻辑**:
```
推荐得分 = 内容匹配度 × 艺人热度 × 时效性 × 个性化权重

内容匹配度 = 用户兴趣标签 ∩ 内容标签
艺人热度 = 最近30天社交媒体声量
时效性 = 发布时间衰减因子
个性化权重 = 用户历史行为学习
```

### 4.2 商业价值评估

**应用**: 艺人商业价值评估、品牌代言匹配

**评估维度**:
- 流量指标（粉丝数、互动率）
- 作品表现（收视/票房/播放量）
- 商业转化（代言费、带货能力）
- 口碑声誉（正面/负面舆情）
- 发展趋势（成长性分析）

### 4.3 IP开发决策

**应用**: IP是否值得开发、如何开发

**决策模型**:
```
IP价值 = 核心粉丝基数 × IP延展性 × 开发成本可行性

延展性评估：
- 跨媒介改编难度
- 衍生品开发空间
- 国际化潜力
```

### 4.4 舆情监控

**应用**: 实时监测艺人/作品口碑

**监控指标**:
- 正面声量占比
- 情感倾向分析
- 热点事件响应
- 竞品对比分析

## 五、数据模型 (Data Model)

### 5.1 核心数据表

#### 5.1.1 艺人表 (Artist)

```sql
CREATE TABLE artist (
    artist_id VARCHAR(32) PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    stage_name VARCHAR(100),
    gender VARCHAR(10),
    birth_date DATE,
    agency_id VARCHAR(32),
    tier VARCHAR(10),  -- S+/S/A/B/C/D
    followers_count BIGINT,
    created_at TIMESTAMP,
    updated_at TIMESTAMP
);
```

#### 5.1.2 作品表 (Content)

```sql
CREATE TABLE content (
    content_id VARCHAR(32) PRIMARY KEY,
    title VARCHAR(200) NOT NULL,
    content_type VARCHAR(20),  -- movie/tv/music/game
    genre VARCHAR(50),
    release_date DATE,
    duration INT,  -- 分钟
    rating VARCHAR(10),  -- G/PG/PG-13/R
    box_office DECIMAL(15,2),  -- 票房
    view_count BIGINT,  -- 播放量
    status VARCHAR(20),  -- released/ongoing/cancelled
    created_at TIMESTAMP
);
```

#### 5.1.3 版权表 (Copyright)

```sql
CREATE TABLE copyright (
    copyright_id VARCHAR(32) PRIMARY KEY,
    content_id VARCHAR(32),
    right_type VARCHAR(50),  -- 发行权/改编权/复制权等
    owner_id VARCHAR(32),
    licensee_id VARCHAR(32),
    start_date DATE,
    end_date DATE,
    fee DECIMAL(15,2),
    territory VARCHAR(100),  -- 授权地域
    exclusive BOOLEAN,
    created_at TIMESTAMP
);
```

#### 5.1.4 代言表 (Endorsement)

```sql
CREATE TABLE endorsement (
    endorsement_id VARCHAR(32) PRIMARY KEY,
    artist_id VARCHAR(32),
    brand_name VARCHAR(100),
    brand_category VARCHAR(50),
    fee DECIMAL(15,2),
    start_date DATE,
    end_date DATE,
    exclusive BOOLEAN,
    territory VARCHAR(100),
    status VARCHAR(20),  -- active/expired/cancelled
    created_at TIMESTAMP
);
```
