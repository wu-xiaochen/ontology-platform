# 影视领域本体设计 (Cinema Ontology)

## 概述

本文档定义影视领域的核心本体模型，用于指导电影、电视剧、网络剧等视听内容的管理、推荐、创作分析等智能化应用。涵盖影视制作、发行放映、评奖参展、观众反馈等全生命周期。

## 版本信息

|------|------|----------|

## 一、概念定义 (Concepts)

### 1.1 核心实体

#### 1.1.1 影视作品

| 实体ID | 概念 | 英文 | 定义 | 关键属性 |
|--------|------|------|------|----------|
| cinema_movie | 电影 | Movie | 在院线或流媒体上映的视听作品 | 影片编号、片名、时长、类型、题材、上映日期、制作成本、票房 |

#### 1.1.5 运营与数据

| 实体ID | 概念 | 英文 | 定义 | 计算方式/说明 |
|--------|------|------|------|---------------|
| cinema_box_office | 票房 | Box Office | 电影票销售总收入 | 累计票房金额（含线上线下） |

| cinema_popularity_index | 热度指数 | Popularity Index | 内容社交媒体讨论热度 | 微博/抖音/豆瓣等平台声量加权 |
| cinema_douban_rating | 豆瓣评分 | Douban Rating | 豆瓣平台用户评分 | 1-10分制，加权平均 |
| cinema_imdb_rating | IMDb评分 | IMDb Rating | IMDB平台用户评分 | 1-10分制 |
| cinema_maoyan_rating | 猫眼评分 | Maoyan Rating | 猫眼平台用户评分 | 1-10分制 |
| cinema_taopiaopiao_rating | 淘票票评分 | Taopiaopiao Rating | 淘票票平台用户评分 | 1-10分制 |
| cinema_market_share | 市场份额 | Market Share | 票房/播放量占比 | 该作品票房 / 同期总票房 × 100% |
| cinema_occupancy_rate | 上座率 | Occupancy Rate | 影院座位占用比例 | 观影人次 / 可放映场次座位数 × 100% |
| cinema_per_screening | 场均人次 | Per-Screening Attendance | 每场平均观影人数 | 总观影人次 / 总场次 |
| cinema_completion_rate | 完播率 | Completion Rate | 用户完整观看比例 | 完整观看人数 / 总播放人数 × 100% |
| cinema_engagement_rate | 互动率 | Engagement Rate | 用户互动行为比例 | 互动数（点赞+评论+转发）/ 播放量 × 100% |
| cinema_ancillary_revenue | 衍生收入 | Ancillary Revenue | 影视衍生收入 | 版权销售 + 衍生品 + 广告分成 |
| cinema_roi | 投资回报率 | ROI | 影视投资收益 | （总收入 - 投资成本）/ 投资成本 × 100% |
| cinema_cpm | 千次播放成本 | CPM | 广告投放成本 | 广告费用 / 播放量(千次) × 1000 |
| cinema_retention_rate | 留存率 | Retention Rate | 用户续费/续看比例 | 续费用户数 / 总用户数 × 100% |

## 三、规则与约束 (Rules)

### 3.1 业务规则

1. **票房统计规则**
   - 票房统计截止时间为影片上映后30天（特殊影片可申请延期）
   - 服务费计入票房统计
   - 网络代售占比不低于50%

2. **评分计算规则**
   - 豆瓣评分=所有用户评分加权平均（去除异常数据）
   - 评分人数低于1000人不显示评分
   - 评分每24小时更新一次

3. **内容分级规则**
   - 国产电影分级：全年龄、PG-12、PG-18
   - 引进片分级：G、PG、PG-13、R、NC-17
   - 电视剧分级：TV-G、TV-PG、TV-14、TV-MA

4. **奖项评选规则**
   - 评选周期内上映的影视作品方可参评
   - 每个奖项每个类别最多提名5部/人
   - 评委回避原则：参评作品相关人员不得担任评委

5. **版权授权规则**
   - 版权授权需明确授权范围、期限、地域
   - 独家授权期限一般不超过5年
   - 全媒体版权优先于单平台版权

### 3.2 数据完整性约束

1. 影视作品必须有：中文名、制作年份、类型
2. 电影必须有：上映日期、时长
3. 电视剧必须有：集数、单集时长
4. 人物必须有：姓名（或艺名）
5. 公司必须有：公司名称、成立日期
6. 票房数据必须关联具体影院

### 3.3 关系约束

1. 一部影视作品可以有多个导演、编剧、制片人
2. 一个导演可以执导多部影视作品
3. 同一演员在同一部影视作品中只能饰演一个角色
4. 翻拍作品必须标注原作品信息
5. 续集必须标注前作信息

## 五、规律 (Laws)

### 5.1 影视作品票房分布规律

**定义**: 影视作品票房呈现明显的"二八定律"和头部效应。

**核心规律**:
- 头部20%的电影贡献80%的票房
- 爆款电影的票房可达平均水平的10-50倍
- 长尾影片占比超过60%，但票房贡献不足20%

**公式**:
```
票房集中度 = Top10影片票房 / 同期总票房 × 100%
```
**行业基准**:
- 旺季(春节/国庆): 票房集中度 40-60%
- 淡季: 票房集中度 30-40%

### 5.2 剧集收视率衰减规律

**定义**: 电视剧收视率随播出集数推进呈递减趋势。

**核心规律**:
- 首播收视率最高，平均占全剧收视的35-45%
- 中段(第15-25集)衰减最快，平均下降20-30%
- 结局周略有回升，俗称"大结局效应"

**计算公式**:
```
收视衰减率 = (首播收视 - 结局收视) / 首播收视 × 100%
```
**类型差异**:
- 精品剧衰减率: <15%
- 普通剧衰减率: 20-35%
- 烂尾剧衰减率: >40%

### 5.3 内容生命周期规律

**定义**: 不同类型影视作品的市场生命周期遵循特定规律。

**电影生命周期**:
```
上映期(首周) → 发酵期(2-3周) → 长尾期(4周后) → 复映期(特殊节点)
```
- 院线生命周期: 通常30-90天
- 流媒体生命周期: 上线首周决定80%长期表现

**剧集生命周期**:
```
预热期 → 播出期(热度峰值) → 完结期 → 衍生期 → 长尾期
```
- 热播期: 首月占生命周期流量的60%
- 长尾期: 可持续1-3年

### 5.4 IP价值链延伸规律

**定义**: 优质IP具备多业态衍生能力，价值逐级放大。

**价值链模型**:
```
原始IP → 影视改编 → 游戏/动漫 → 衍生品 → 线下实景 → 主题乐园
```
**倍率规律**:
- 影视化放大: 3-10倍
- 游戏化放大: 5-20倍
- 衍生品放大: 2-5倍
- 实景化放大: 10-50倍

**成功案例参考**:
- 漫威宇宙: IP价值超3000亿美元
- 哈利波特: 衍生品收入超300亿美元
- 甄嬛传: 播出10年后仍具商业价值

### 5.5 演员票房/收视号召力规律

**定义**: 演员的商业价值可通过历史作品数据量化评估。

**计算公式**:
```
票房号召力指数 = Σ(主演票房) / 主演作品数 × 类型权重
收视号召力指数 = Σ(主演收视) / 主演作品数 × 平台权重
```

**等级划分**:
- S级: 指数 ≥ 15亿(票房) / 2.0%(收视)
- A级: 指数 5-15亿 / 1.5-2.0%
- B级: 指数 1-5亿 / 1.0-1.5%
- C级: 指数 < 1亿 / <1.0%

### 5.6 档期效应规律

**定义**: 影视票房受档期影响呈现显著波动。

**中国电影档期**:
| 档期 | 时间 | 票房占比 | 特点 |
|------|------|----------|------|
| 春节档 | 除夕-初七 | 15-18% | 刚需最强 |
| 五一档 | 5.1-5.3 | 5-7% | 短途旅游分流 |
| 暑期档 | 6-8月 | 25-30% | 周期最长 |
| 国庆档 | 10.1-10.7 | 8-12% | 主旋律集中 |
| 贺岁档 | 12月 | 10-15% | 喜剧扎堆 |

### 5.7 内容审核合规规律

**定义**: 影视内容需符合监管要求，不同题材有不同的合规边界。

**红线规则**:
- 禁止内容: 危害国家安全、宣扬色情暴力、违背历史事实
- 限制内容: 需要修改或标注提示
- 倡导内容: 主流价值观优先推荐

**审查周期**:
- 电视剧: 40-60个工作日
- 网络剧: 20-40个工作日
- 电影: 60-90个工作日
