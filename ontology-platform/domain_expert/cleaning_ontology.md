## 4. 清洁剂本体

### 4.1 按酸碱性质分类

| 类型 | pH范围 | 代表产品 | 用途 |
|-----|--------|---------|------|
| 酸性清洁剂 | pH < 7 | 洁厕剂、盐酸 | 去除水垢、尿垢 |
| 中性清洁剂 | pH ≈ 7 | 洗洁精、洗手液 | 日常清洁 |
| 碱性清洁剂 | pH > 7 | 洗衣粉、烧碱 | 去除油污 |
| 两性清洁剂 | pH 5-8 | 温和洗涤剂 | 通用 |

### 4.2 按功能分类

| 清洁剂类型 | 主要成分 | 功能 | 注意事项 |
|-----------|---------|------|---------|
| 多功能清洁剂 | 表面活性剂、助剂 | 日常清洁 | 需稀释使用 |
| 洁厕剂 | 盐酸、磷酸 | 去除污垢、杀菌 | 不可混用漂白水 |
| 厨房清洁剂 | 碱性、表面活性剂 | 去除油污 | 避免接触皮肤 |
| 玻璃清洁剂 | 酒精、 ammonia | 玻璃表面清洁 | 通风使用 |
| 地板清洁剂 | 表面活性剂、聚合物 | 木地板、瓷砖 | 按材质选择 |
| 衣物清洁剂 | 表面活性剂、酶 | 衣物洗涤 | 按颜色、材质选择 |
| 消毒剂 | 酒精、含氯化合物 | 杀菌消毒 | 浓度适宜 |
| 漂白剂 | 次氯酸钠、过氧化氢 | 漂白、杀菌 | 需稀释、避免混用 |
| 除垢剂 | 酸类 | 去除水垢 | 定期使用 |
| 家具清洁剂 | 硅油、蜡 | 保养、上光 | 适量使用 |

### 4.3 天然清洁剂

| 清洁剂 | 成分 | 用途 |
|-------|------|------|
| 白醋 | 醋酸 | 除臭、软化水、清洁玻璃 |
| 柠檬酸 | 柠檬酸 | 除水垢、清洁饮水机 |

## 8. 清洁标准与流程本体

### 8.1 日常清洁标准

| 区域 | 标准 | 频次 |
|-----|------|------|
| 客厅 | 无灰尘、无杂物、地面干净 | 每日 |
| 卧室 | 床铺整洁、地面干净 | 每日 |
| 厨房 | 台面无油渍、餐具干净 | 每日 |
| 卫生间 | 无异味、无污渍、干燥 | 每日 |
| 地面 | 无明显灰尘、污渍 | 每日 |

### 8.2 深度清洁标准

| 项目 | 标准 |
|-----|------|
| 表面除尘 | 100%清除可见灰尘 |
| 缝隙清洁 | 100%清除可见杂物 |
| 厨房 | 油烟机内外部、橱柜内部、冰箱内部 |
| 卫生间 | 瓷砖缝、排水口、角落 |
| 玻璃 | 无水渍、无手印 |
| 床品 | 100%更换、晾晒 |
| 窗帘 | 100%清洗 |

### 8.3 清洁流程规范

```
清洁前准备
    ↓
分区清洁（从上到下）
    ↓
先难后易原则
    ↓
最后处理地面
    ↓
检查与补漏
    ↓
工具归位
```

**核心原则**:
- 从上到下：先清洁高处，再清洁低处
- 从里到外：先清洁内部，再清洁外部
- 干湿分离：先干除灰尘，后湿擦清洁
- 分类处理：不同区域使用不同工具和清洁剂

## 10. 清洁问题与解决方案本体

### 10.1 常见清洁问题

| 问题 | 原因 | 解决方案 |
|-----|------|---------|
| 地面有水渍 | 拖把太湿、未及时干燥 | 拧干拖把、选择晴天 |
| 玻璃有水印 | 水质问题、清洁剂残留 | 使用玻璃刮、蒸馏水 |
| 油烟机滴油 | 油杯满、过滤网堵塞 | 清洗油杯、过滤网 |
| 马桶有黄渍 | 水质硬、尿垢沉积 | 使用洁厕剂、定期清洁 |
| 地毯有异味 | 潮湿、污渍残留 | 晾晒、吸尘、清洗 |
| 沙发有污渍 | 液体渗透、未及时处理 | 立即吸干、专业清洗 |
| 衣柜有异味 | 潮湿、衣物未晾干 | 通风、干燥剂 |
| 空调有异味 | 滤网脏、霉菌 | 清洗滤网、消毒 |

### 10.2 顽固污渍处理

| 污渍类型 | 处理方法 |
|---------|---------|
| 油污 | 洗洁精、厨房清洁剂、热水 |
| 血渍 | 冷水+双氧水（不可用热水） |
| 红酒 | 盐吸附+洗涤剂 |
| 咖啡 | 热水+洗涤剂 |
| 墨水 | 酒精+洗涤剂 |
| 口香糖 | 冰冻后刮除 |
| 胶痕 | 酒精、风油精 |
| 锈迹 | 酸性清洁剂、草酸 |
| 水垢 | 醋酸、柠檬酸 |
| 霉菌 | 漂白剂、84消毒液 |

| 避免混用 | 漂白水+氨水=有毒气体 |
| 稀释使用 | 按说明比例调配 |
| 通风使用 | 保持空气流通 |
| 防护装备 | 手套、口罩、护目镜 |
| 儿童远离 | 放在儿童接触不到的地方 |
| 标签清晰 | 保留原标签、不换容器 |
