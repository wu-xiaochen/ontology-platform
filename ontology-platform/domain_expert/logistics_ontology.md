# 物流领域本体设计

## 概述

本文档定义物流领域的核心本体模型，用于指导知识库构建和智能化应用。涵盖运输、仓储、配送、供应链物流等核心业务领域。

---

## 一、概念定义 (Concepts)

### 1.1 核心实体

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 货物 | Cargo/Goods | 待运输或仓储的物品 | 货物编码、名称、规格、重量、体积、价值、属性 |
| 运输工具 | Vehicle | 货物运输的载体 | 车型、车牌、载重、体积、GPS设备 |
| 仓库 | Warehouse | 存储货物的设施 | 仓库编码、位置、面积、容量、类型、温控 |

| 客户 | Customer | 物流服务的使用者 | 客户编码、名称、联系方式、地址 |
| 集装箱 | Container | 标准化的货物装载单元 | 箱号、类型(20'/40'/45')、尺寸、状态 |

## 二、规律 (Laws)

### 2.1 物流成本构成规律

**定义**: 物流总成本由运输、仓储、库存、管理等部分构成，各部分存在配比关系。

**成本结构公式**:
```
物流总成本 = 运输成本 + 仓储成本 + 库存持有成本 + 包装成本 + 装卸成本 + 物流管理成本
```

**典型比例** (制造业):
- 运输成本: 50-60%
- 仓储成本: 20-25%
- 库存持有成本: 15-20%
- 管理及其他: 5-10%

**规模效应**:
- 运输: 批量越大，单位成本越低
- 仓储: 面积越大，单位租金越低
- 配送: 密度越高，单位成本越低

### 2.2 配送时效规律

**定义**: 配送时效受距离、路况、配送量、天气等因素影响。

**时效计算公式**:
```
配送时效 = 装车时间 + 运输时间 + 卸货时间 + 等待时间
```

**城市配送时效基准**:
| 配送距离 | 同城配送 | 区域配送 |
|----------|----------|----------|
| 50km内 | 2-4小时 | - |
| 50-100km | 4-6小时 | 6-8小时 |
| 100-200km | - | 8-12小时 |
| 200km以上 | - | 12-24小时 |

**影响因素**:
- 交通管制: 城区限行增加20-30%时间
- 天气因素: 恶劣天气增加30-50%时间
- 配送密度: 高密度区域效率提升30%

### 2.3 车辆装载率规律

**定义**: 车辆实际装载量与额定装载能力的比值。

**计算公式**:
```
装载率 = 实际装载量 / 额定装载量 × 100%
```

**优化目标**:
| 运输类型 | 目标装载率 |
|----------|------------|
| 整车运输 | ≥95% |
| 零担运输 | ≥75% |
| 城市配送 | ≥80% |
| 快递配送 | ≥70% |

**提升策略**:
- 拼单优化: 同一方向订单合并
- 智能配载: 算法优化装载方案
- 回程带货: 利用返程空间

### 2.4 库存周转与物流效率

**定义**: 库存周转速度影响物流仓储和配送效率。

**关系公式**:
```
库存周转天数 = 365 / 库存周转率
物流响应速度 ∝ 1 / 库存周转天数
```

**行业基准**:
| 行业 | 周转天数 | 对应物流模式 |
|------|----------|--------------|
| 快消品 | 15-30天 | 敏捷物流 |
| 制造业 | 30-60天 | 精益物流 |
| 家电 | 45-60天 | 供应链物流 |
| 建材 | 60-90天 | 计划物流 |

### 2.5 运输时效与成本权衡

**定义**: 运输时效要求越高，成本越高，存在边际效益递减。

**时效-成本曲线**:
```
成本
  │
  │    ╲
  │     ╲___
  │         ╲___
  │             ╲___
  │                 ╲___
  └─────────────────────→ 时效要求
      普通   加急   极速
```

**加急系数参考**:
| 时效要求 | 成本系数 | 适用场景 |
|----------|----------|----------|
| 普通 (3-5天) | 1.0x | 大宗货物 |
| 加快 (1-2天) | 1.3-1.5x | 常规加急 |
| 加急 (当日) | 2.0-3.0x | 紧急订单 |
| 极速 (小时) | 5.0x+ | 医疗/特种 |

### 2.6 物流网络布局规律

**定义**: 物流节点数量与覆盖范围、配送时效、成本的关系。

**网络层级模型**:
```
全国分拨中心 → 区域配送中心 → 城市配送站 → 末端网点
    (3-5个)        (10-20个)      (100+个)    (1000+个)
```

**布局原则**:
- 层级越多，配送越快，但成本越高
- 节点密度与配送密度正相关
- 核心城市优先覆盖，再向外延伸

## 三、业务规则 (Rules)

### 3.1 货物收运规则

| 货物类型 | 收运要求 |
|----------|----------|
| 普通货物 | 品名、数量、体积、重量准确申报 |
| 鲜活货物 | 提供保鲜措施，限定运输时间 |
| 危险货物 | 提供MSDS、危包证、正确UN编号 |
| 超限货物 | 提前办理超限运输许可 |
| 贵重货物 | 声明价值，办理保价运输 |
| 禁运货物 | 不得收运: 毒品、武器、违禁品等 |

### 3.2 运输方式选择规则

| 货物重量/体积 | 优先运输方式 | 适用场景 |
|--------------|--------------|----------|
| <30kg | 快递/空运 | 高价值、时间敏感 |
| 30-500kg | 公路零担 | 中小批量、中短途 |
| 500kg-5吨 | 公路整车 | 大批量、同城/区域 |
| 5-20吨 | 铁路/公路 | 大宗货物、中长距离 |
| >20吨 | 海运/铁路 | 超大批量、长距离 |
| 跨国 | 海运/空运/多式联运 | 国际贸易 |

### 3.3 仓库存储规则

| 仓库类型 | 存储要求 |
|----------|----------|
| 普通仓 | 常温干燥、通风、防尘 |
| 恒温仓 | 温度15-25℃，恒湿 |
| 冷藏仓 | 0-5℃保鲜 |
| 冷冻仓 | -18℃以下 |
| 危险品仓 | 隔离存储、防爆通风 |
| 保税仓 | 海关监管、账册管理 |

**存储限制**:
- 堆高不超过3米
- 通道宽度≥1.5米
- 消防通道保持畅通
- 危化品按GB分类存储

### 3.4 配送优先级规则

| 优先级 | 订单类型 | 处理时限 |
|--------|----------|----------|
| P0-紧急 | 医疗急救、应急物资 | 2小时内 |
| P1-加急 | 重要客户、样品 | 当日达 |
| P2-标准 | 常规订单 | 1-2日达 |
| P3-经济 | 大宗货物、计划配送 | 3-5日达 |

### 3.5 物流服务等级协议 (SLA)

| 服务类型 | 时效要求 | 签收率要求 | 异常处理 |
|----------|----------|------------|----------|
| 当日达 | ≤24小时 | ≥98% | 免运费+赔偿 |
| 次日达 | ≤36小时 | ≥97% | 免运费+补偿 |
| 标准达 | ≤72小时 | ≥95% | 运费退还 |
| 经济达 | ≤5天 | ≥90% | 协商处理 |

**赔偿标准**:
- 延迟送达: 运费退还10-50%
- 货物损毁: 按实际损失赔偿
- 货物丢失: 按声明价值赔偿(最高不超运费的3倍)
- 保价货物: 按保价金额全额赔偿

### 3.6 车辆调度规则

**调度原则**:
1. 装载率优先: 优先安排装载率高的路线
2. 时效保障: 紧急订单优先调度
3. 成本最优: 综合考虑距离、油耗、路况
4. 车辆匹配: 货物特性匹配车辆类型

**车辆类型匹配**:
| 货物特性 | 适用车型 |
|----------|----------|
| 普通货物 | 厢式货车 |
| 鲜活货物 | 冷藏车 |
| 危化品 | 专用运输车 |
| 大件货物 | 平板车/挂车 |
| 快递包裹 | 小型厢货/面包车 |

### 3.7 物流跟踪规则

**节点跟踪要求**:
| 节点 | 必须记录 | 时效要求 |
|------|----------|----------|
| 揽收 | 时间、地点、操作人 | 即时 |
| 运输中 | 位置、状态 | 2-4小时 |
| 中转 | 时间、场地 | 1小时内 |
| 签收 | 时间、签收人 | 即时 |

**异常预警规则**:
- 超时预警: 节点停留超过正常时效2倍
- 滞留预警: 货物在节点超过24小时
- 轨迹异常: 偏离规划路线超过10%

### 3.8 退货物流规则

**退货处理流程**:
```
退货受理 → 退货验收 → 分类处理 → 退换货/报废 → 数据反馈
```

**退货时限**:
- 7天无理由退货: 电商件
- 15天质量问题退货: 大件商品
- 30天性能故障退货: 电子产品

**退货费用**:
- 物流责任: 免费退货
- 客户责任: 客户承担运费
- 商品问题: 商家承担运费

## 五、应用场景

1. **智能调度**: 基于车辆位置、货物信息、实时路况进行智能派单和路径优化
2. **运输追踪**: 全程可视化追踪货物状态，异常自动预警
3. **成本优化**: 分析运输、仓储成本结构，提出优化建议
4. **网络规划**: 辅助物流网络布局决策，优化仓配节点选址
5. **时效预测**: 基于历史数据和实时因素预测配送时效
6. **风险预警**: 识别运输风险、库存风险、供应商风险
