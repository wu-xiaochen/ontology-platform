## 测试用例 1：燃气调压箱选型

### 1.1 测试目的
验证本体论推理引擎能够根据用户提供的供气需求，准确推荐合适的燃气调压箱型号。

### 1.2 测试输入

| 字段 | 值 |
|------|-----|
| 用户类型 | 居民 |
| 户数 | 100户 |
| 建筑类型 | 高层建筑 |
| 供暖需求 | 有供暖 |
| 供气压力 | 中压B级 (0.1-0.3MPa) |
| 用气区域 | 住宅区 |

### 1.3 期望输出

```json
{
  "recommended_model": "RTZ-150/25",
  "capacity": "150m³/h",
  "inlet_pressure": "中压B级",
  "outlet_pressure": "2.8-5.0kPa",
  "confidence": "A",
  "reasoning_chain": [
    "100户高层住宅属于中型供气规模",
    "根据用气需求计算，流量≥100m³/h",
    "RTZ-150/25额定流量150m³/h，满足需求",
    "该型号支持楼宇供气场景"
  ]
}
```

### 1.4 验证标准

| 验证项 | 标准 | 权重 |
|--------|------|------|
| 型号正确性 | 输出型号为RTZ-150/25 | 40% |
| 置信度 | 置信度≥B级 | 30% |
| 推理链完整性 | 包含至少3步推理步骤 | 20% |
| 响应时间 | <2秒 | 10% |

## 测试用例 2：采购供应商选择

| 紧急程度 | 一般 |

