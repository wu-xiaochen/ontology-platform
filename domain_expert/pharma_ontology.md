# 制药领域本体设计

## 概述

本文档定义制药领域的核心本体模型，用于指导药品研发、生产、流通和监管的知识库构建和智能化应用。

---

## 一、概念定义 (Concepts)

### 1.1 核心实体

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 药品 | Pharmaceutical | 用于预防、治疗、诊断疾病的活性物质或其组合 | 药品名称、批准文号、规格、剂型、适应症 |
| 原料药 | Active Pharmaceutical Ingredient (API) | 药品中具有药理活性的成分 | CAS号、化学名、分子式、纯度 |
| 制剂 | Dosage Form | 药品的具体剂型形态 | 剂型名称、规格、包装 |
| 药品生产企业 | Pharmaceutical Manufacturer | 从事药品生产活动的企业 | 生产许可证、GMP证书、产能 |
| 药品经营企业 | Pharmaceutical Distributor | 从事药品批发零售的企业 | 经营许可证、经营范围、GSP认证 |
| 药品注册 | Drug Registration | 药品上市前的行政审批程序 | 注册申请号、审评结论、批准日期 |
| 临床试验 | Clinical Trial | 药品人体安全性有效性评价研究 | 试验方案、阶段、结果 |
| 药品质量标准 | Quality Standard | 药品应符合的技术规范 | 标准编号、检验项目、限度要求 |
| 监管部门 | Regulatory Authority | 药品监督管理机构 | 机构名称、职责范围 |
| 不良反应 | Adverse Drug Reaction (ADR) | 药品在正常用法用量下产生的有害反应 | 报告编号、严重程度、因果关系 |

### 1.2 关系定义

```
药品 ⊚ 包含 [原料药] (一对多)
药品 ⊚ 采用 [制剂] (一对多)
药品 ⊚ 需要 [药品注册] (一对一)
药品 ⊚ 遵循 [药品质量标准] (多对一)
原料药 ⊚ 生产 [药品生产企业] (多对多)
制剂 ⊚ 生产 [药品生产企业] (一对多)
药品 ⊚ 流通 [药品经营企业] (多对多)
药品 ⊚ 引发 [不良反应] (一对多)
临床试验 ⊚ 评价 [药品] (多对一)
药品 ⊚ 监管 [监管部门] (多对一)
```

```
药品
```

## 二、规律 (Laws)

### 2.1 药品研发周期规律

**定义**: 创新药从研发立项到上市所需的平均时间周期。

**规律描述**:
```
总研发周期 = 临床前研究(3-5年) + 临床试验(5-7年) + 注册审批(1-2年)
总周期: 10-15年
```

**阶段分布**:
- 药物发现: 1-2年
- 临床前研究: 2-3年
- 临床试验I期: 1年
- 临床试验II期: 1-2年
- 临床试验III期: 2-3年
- 注册审批: 1-2年

**成功概率**:
- 药物发现 → 临床前: 10%
- 临床前 → I期: 30%
- I期 → II期: 33%
- II期 → III期: 25-30%
- III期 → 上市: 60-70%

### 2.2 药品质量源于设计 (QbD)

**定义**: 药品质量通过合理设计获得，而非通过检验保证。

**核心要素**:
```
QTPP (目标产品质量概况) → CQA (关键质量属性) → CPP (关键工艺参数)
```

**关系公式**:
```
CQA = f(CPP, 原辅料属性, 设备参数)
```

### 2.3 仿制药一致性评价规律

**定义**: 仿制药与原研药在质量和疗效上一致性评价。

**评价指标**:
```
生物等效性 (BE): AUC和Cmax的90%置信区间在80%-125%范围内
溶出曲线相似性: 相似因子f2 ≥ 50
杂质谱一致性: 杂质种类和含量不高于原研药
```

### 2.4 药品价格形成规律

**创新药定价模型**:
```
价格 = 研发成本 / (预期销量 × 生命周期) + 合理利润
```

**影响因素**:
- 研发投入和创新程度
- 市场需求和竞争格局
- 医保支付能力和政策调控
- 患者可及性

### 2.5 GMP合规成本规律

**定义**: 药品生产企业为满足GMP要求所需的成本投入。

**成本构成**:
```
GMP合规成本 = 硬件投资 + 软件建设 + 人员培训 + 持续验证 + 官方检查
```

**行业基准** (占产值比例):
- 无菌制剂: 8-15%
- 普通制剂: 3-5%
- 原料药: 2-4%

## 三、业务规则 (Rules)

### 3.1 药品注册规则

| 注册分类 | 申报要求 | 审评时间 |
|----------|----------|----------|
| 1类创新药 | 完整的安全性有效性数据 | 60个工作日（优先审评） |
| 2类改良型新药 | 证明优于已上市药品 | 60个工作日 |
| 3类仿制药 | 生物等效性研究 + 药学研究 | 120个工作日 |
| 4类仿制药 | 生物等效性研究 | 120个工作日 |
| 5类境外药品 | 境外上市证明 + 完整研究资料 | 60个工作日 |

**申报资料要求** (CTD格式):
1. 模块一: 行政文件和药品信息
2. 模块二: 药学研究资料
3. 模块三: 药学研究资料(续)
4. 模块四: 非临床研究资料
5. 模块五: 临床研究资料

### 3.2 GMP生产规范规则

**无菌药品生产环境要求**:

| 洁净度级别 | 悬浮粒子(≥0.5μm) | 悬浮粒子(≥5μm) | 微生物 |
|------------|------------------|----------------|--------|
| A级 | ≤3520/m³ | ≤20/m³ | 0 |
| B级 | ≤3520/m³ | ≤20/m³ | ≤10 |
| C级 | ≤352000/m³ | ≤2900/m³ | ≤100 |
| D级 | ≤3520000/m³ | ≤29000/m³ | ≤200 |

**关键操作要求**:
- 无菌操作: A级背景下的A
- 人员培训: 每年≥40学时
- 设备确认: IQ/OQ/PQ三阶段验证
- 工艺验证: 连续3批商业化生产
- 偏差处理: 24小时内启动调查
- 变更控制: 重大变更需备案或审批

### 3.3 药品经营质量管理规范 (GSP)

**仓库储存条件**:

| 药品类别 | 温度要求 | 湿度要求 |
|----------|----------|----------|
| 常温储存 | 10-30°C | 35-75% |
| 阴凉储存 | ≤20°C | 35-75% |
| 冷链储存 | 2-8°C | 35-75% |
| 冷冻储存 | ≤-15°C | — |

**经营范围许可**:
- 麻醉药品和第一类精神药品: 需特殊许可
- 血液制品、疫苗: 需冷链资质
- 中药饮片: 需中药材相关专业人员

### 3.4 药品不良反应监测规则

**报告要求**:

| 报告类型 | 报告时限 | 报告范围 |
|----------|----------|----------|
| 严重不良反应 | 15日内 | 导致死亡、危及生命、永久性残疾等 |
| 新的不良反应 | 30日内 | 药品说明书中未载明的不良反应 |
| 一般不良反应 | 季度汇总 | 其他已知不良反应 |

**因果关系评价**:
- 肯定相关: 再次用药反应重复出现
- 很可能相关: 停药后改善/无可疑替代
- 可能相关: 可能有其他原因
- 可能无关: 很可能有其他原因
- 无法评价: 信息不足

### 3.5 药品追溯码规则

**追溯码构成**:
```
追溯码 = 药品本位码(20位) + 包装码(9位) + 序列号(13位) = 42位
```

**追溯信息要求**:
- 生产环节: 原料来源、生产日期、有效期、批号
- 流通环节: 批发商、零售商、物流信息
- 使用环节: 医疗机构、用药患者(脱敏)

**核注核销规则**:
- 入库核注: 追溯码与库存关联
- 出库核销: 追溯码与流向关联
- 异常追溯: 问题药品快速定位流向

### 3.6 医保准入规则

**医保药品目录调整**:
- 每年调整一次
- 优先纳入: 独家品种、创新药、临床必需药品
- 谈判准入: 价格较高药品通过谈判降价
- 调出: 安全性问题、滥用严重的药品

**药品定价机制**:
- 国家定价: 麻醉药品、第一类精神药品
- 省级招标采购: 医保目录内药品
- 自主定价: 自主定价药品

## 五、核心关系详解

### 5.1 药品-原料药关系

```
药品(成品药)
├── 单一成分药品: 1个原料药
├── 复方制剂: 2个或以上原料药
└── 案例: 阿莫西林胶囊 = 阿莫西林(原料药) + 胶囊(制剂)

关系属性:
- 原料药含量: g/片、mg/粒
- 原料药来源: 自主生产/外购
- 原料药供应商审计: 必须
```

### 5.2 药品-注册关系

```
注册阶段:
1. 临床试验申请 (CTA) → 获批临床
2. 上市注册申请 (NDA) → 获批上市
3. 补充申请 → 变更批准
4. 再注册 → 有效期延续

注册状态:
- 申请中
- 审评中
- 批准上市
- 不批准
- 注销
```

### 5.3 追溯关系

```
全生命周期追溯:
[原料药采购] → [生产投料] → [制剂加工] → [包装检验] → [出库销售] → [终端使用]

追溯节点:
- 原料来源可追溯到具体供应商和批次
- 生产过程可追溯到具体设备和操作人员
- 流通环节可追溯到每个经手企业
- 终端使用可追溯到具体医疗机构或药店
```

## 六、应用场景

### 6.1 智能问答
- 药品注册流程咨询
- GMP合规问题解答
- 药品追溯政策查询
- 不良反应报告指导

### 6.2 流程自动化
- 药品注册申报辅助
- GMP检查准备清单生成
- 不良反应报告自动上报
- 追溯码核注核销

### 6.3 数据分析
- 药品研发效率分析
- 质量偏差趋势分析
- 市场价格趋势预测
- 不良反应信号检测

### 6.4 知识推理
- 药品关联性发现(相同原料药的不同药品)
- 供应链风险识别
- 仿制药一致性评价预测
- 药品选择建议(疗效、安全性、经济性)
