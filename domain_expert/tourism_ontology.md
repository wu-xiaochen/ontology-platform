# 旅游领域本体设计

## 概述

本文档定义旅游领域的核心本体模型，涵盖旅游活动中的核心实体、关系、业务规律和规则，用于指导旅游知识库构建和智能化应用。

---

## 一、概念定义 (Concepts)

### 1.1 核心实体

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 旅游者 | Traveler | 进行旅游活动的个人或群体 | 用户ID、姓名、国籍、偏好、预算 |
| 旅游目的地 | Destination | 旅游者前往的地区或景点 | 目的地ID、名称、类型、位置、评级 |
| 旅游产品 | Tourism Product | 旅行社或平台提供的旅游服务组合 | 产品ID、名称、行程天数、价格、类型 |
| 旅游资源 | Tourism Resource | 目的地的自然、人文景观和设施 | 资源ID、名称、类型、位置、开放时间 |
| 旅行社 | Travel Agency | 提供旅游服务的组织 | 旅行社ID、名称、资质、等级、联系方式 |
| 导游 | Tour Guide | 陪同旅游者进行游览的专业人员 | 导游ID、姓名、语言、资质、评分 |
| 酒店 | Hotel | 为旅游者提供住宿的设施 | 酒店ID、名称、星级、位置、房型、价格 |
| 交通 | Transportation | 往返目的地的出行方式 | 交通ID、类型(飞机/火车/汽车/船)、路线、时间 |
| 旅游订单 | Travel Order | 旅游者与旅行社或平台达成的服务协议 | 订单ID、日期、状态、金额、行程 |
| 景区 | Scenic Spot | 收票或免费的游览区域 | 景区ID、名称、级别(5A/4A等)、门票价格 |
| 签证 | Visa | 入境所需的官方许可文件 | 签证类型、办理时长、有效期、费用 |
| 保险 | Travel Insurance | 旅游期间的意外和医疗保障 | 保险类型、保额、费用、保障范围 |

## 二、规律 (Laws)

### 2.1 旅游需求季节性规律

**定义**: 旅游需求随季节变化呈现周期性波动的规律。

**规律表现**:
- **旺季** (7-8月、春节、国庆): 需求达峰值，价格上涨30%-100%
- **平季** (4-6月、9-10月): 需求平稳，价格适中
- **淡季** (11-3月，除春节): 需求低谷，促销力度大

**公式**:
```
季节性指数 = 当月游客量 / 年度月均游客量
```

**影响因子**:
- 气候条件 (气温、降水)
- 法定节假日分布
- 学校假期 (寒暑假)
- 传统节日 (春节、中秋)

### 2.2 旅游花费结构规律

**定义**: 旅游总花费在不同消费项目间的分配比例。

**参考结构** (国内游):
| 消费项目 | 占比 |
|----------|------|
| 交通 | 25%-30% |
| 住宿 | 20%-25% |
| 餐饮 | 15%-20% |
| 门票/景区 | 10%-15% |
| 购物 | 10%-15% |
| 娱乐/其他 | 5%-10% |

**规律**:
- 出行天数越长，交通占比越低，住宿餐饮占比越高
- 高端游住宿占比可达35%以上
- 入境游购物占比可达25%

### 2.3 旅游决策周期规律

**定义**: 旅游者从产生出游想法到完成预订的时间周期。

**规律**:
| 产品类型 | 平均决策周期 |
|----------|--------------|
| 周边游 | 1-3天 |
| 国内长线游 | 7-14天 |
| 出境游 | 20-45天 |
| 邮轮/定制游 | 30-60天 |

**影响因素**:
- 产品复杂度 (行程越长决策越早)
- 价格敏感度 (高价产品决策周期长)
- 假期确定度 (需提前请假)
- 签证办理时间

### 2.4 旅游价格弹性

**定义**: 旅游需求随价格变化的敏感程度。

**规律**:
```
需求价格弹性 = 需求量变化率 / 价格变化率
```

- **高弹性** (|E| > 1): 观光旅游、低端产品、对价格敏感客群
- **低弹性** (|E| < 1): 度假旅游、高端产品、商务旅游
- **负弹性**: 特价产品可能出现"吉芬效应"，价格越低需求越低

**应用**:
- 淡季降价可显著提升入住率
- 高端产品降价空间有限
- 套餐产品弹性高于单品

### 2.5 目的地生命周期规律 (Butler Model)

**定义**: 旅游目的地从发展到衰退的演进过程。

**阶段**:
```
探索期 → 参与期 → 发展期 → 巩固期 → 停滞期 → 衰落期/复兴期
```

**特征**:
- 探索期: 少量游客，自然环境为主
- 发展期: 游客快速增长，设施完善
- 巩固期: 游客量达峰，开始出现环境压力
- 停滞期: 容量饱和，满意度下降
- 衰落期: 游客减少，需要转型
- 复兴期: 通过创新实现再发展

### 2.6 旅游重游率规律

**定义**: 旅游者再次访问同一目的地的比例。

**规律**:
- 国内热门目的地: 重游率20%-30%
- 出境热门目的地: 重游率15%-25%
- 度假型目的地: 重游率可达40%以上
- 观光型目的地: 重游率通常低于15%

**影响因素**:
- 目的地的多样性和可重复体验性
- 交通便利性和成本
- 旅游者的探索偏好

## 三、业务规则 (Rules)

### 3.1 旅游产品定价规则

| 定价因素 | 规则说明 |
|----------|----------|
| 基础价格 | 交通成本 + 住宿成本 + 景区门票 + 导游服务 + 利润 |
| 季节调价 | 旺季加价30%-100%，淡季降价20%-40% |
| 提前预订优惠 | 提前30天以上优惠3%-5%，提前60天优惠5%-10% |
| 人数优惠 | 10人以上团队优惠5%-15% |
| 儿童优惠 | 儿童不占床优惠30%-50%，门票根据身高减免 |
| 特价规则 | 特价产品不退不改，或收取80%以上违约金 |

### 3.2 签证办理规则

| 签证类型 | 办理时长 | 有效停留期 | 适用人群 |
|----------|----------|------------|----------|
| 免签 | 0天 | 7-30天 | 特定国家护照 |
| 落地签 | 1-3天 | 15-30天 | 指定国家 |
| 旅游签 (单次) | 5-15天 | 15-90天 | 个人/团队 |
| 旅游签 (多次) | 10-20天 | 每次30-90天 | 商务/探亲 |
| 团队签证 | 5-10天 | 随团进出 | 团队游客 |

**规则**:
- 护照有效期需大于回程日期6个月以上
- 部分国家需提供在职证明/资产证明
- 旺季签证办理时间延长30%-50%

### 3.3 退改规则

| 产品类型 | 距出发天数 | 退团费 | 改期费 |
|----------|------------|--------|--------|
| 常规团队游 | >30天 | 5% | 免费 |
| | 15-30天 | 10% | 5% |
| | 7-14天 | 20% | 10% |
| | 3-6天 | 40% | 20% |
| | 1-2天 | 60% | 30% |
| | 当天 | 80% | 50% |
| 自由行 | >7天 | 0-10% | 根据航司/酒店规则 |
| | ≤7天 | 20%-100% | 按实际损失 |

**特殊规则**:
- 机票已出票按航司规则执行
- 酒店已确认按酒店规则执行
- 不可抗力因素可协商全额退款

### 3.4 旅游保险配置规则

| 保障类型 | 建议保额 | 适用场景 |
|----------|----------|----------|
| 意外身故/伤残 | 30-50万 | 所有旅游 |
| 医疗费用 | 10-30万 | 国内游 |
| | 30-100万 | 出境游 |
| 紧急救援 | 含 | 出境游/高原游 |
| 航班延误 | 200-500元/次 | 航空出行 |
| 行李丢失 | 1000-2000元 | 航空出行 |
| 个人责任 | 10-30万 | 境外游 |

**强制保险**:
- 出境游: 必须购买旅游意外险
- 高原游: 建议购买含高原反应的保险
- 探险游: 必须购买高风险意外险

### 3.5 导游服务规则

| 导游等级 | 资质要求 | 服务范围 |
|----------|----------|----------|
| 初级导游 | 取得导游资格证 | 散客接待、简单讲解 |
| 中级导游 | 初级满2年+考核 | 团队游、常规线路 |
| 高级导游 | 中级满4年+考核 | 品质团、深度游 |
| 特级导游 | 高级满5年+考核 | VIP团、政务接待 |

**服务标准**:
- 团队游: 导游全程陪同
- 自由行: 司导分开，导游仅负责讲解
- 接机/送机: 需提前确认时间地点

### 3.6 酒店入住规则

| 星级 | 入住时间 | 退房时间 | 加床规则 |
|------|----------|----------|----------|
| 五星 | 14:00后 | 12:00前 | 可加床，加床费300-500元 |
| 四星 | 14:00后 | 12:00前 | 部分可加床 |
| 三星 | 12:00后 | 12:00前 | 不可加床 |
| 经济型 | 14:00后 | 12:00前 | 不可加床 |

**规则**:
- 预订确认后不可取消
- 入住需出示身份证件
- 儿童加床政策因酒店而异
- 涉外酒店需确认接待外宾资质

### 3.7 旅游投诉处理规则

| 投诉类型 | 处理时限 | 处理方式 |
|----------|----------|----------|
| 立即处理 | 2小时内 | 导游现场协调 |
| 一般投诉 | 24小时内 | 旅行社回复 |
| 重大投诉 | 72小时内 | 负责人出面处理 |
| 争议投诉 | 7天内 | 旅游局调解/司法途径 |

**赔偿标准**:
- 行程减少: 退还未游景点门票 + 相应餐费
- 擅自降低标准: 退差额 + 赔偿差额20%
- 延误行程: 赔偿直接损失
- 人身伤害: 按保险条款理赔

## 五、应用场景

1. **智能问答**: 基于本体知识回答签证政策、退改规则、旅游产品推荐等常见问题
2. **行程规划**: 根据旅游者偏好和预算自动生成个性化行程
3. **价格预测**: 基于季节性规律和历史数据预测最佳出行时间
4. **产品推荐**: 根据用户画像和目的地特征进行精准推荐
5. **风险预警**: 基于目的地生命周期和安全信息提供出行建议
6. **投诉处理**: 基于业务规则自动判定责任和赔偿方案
