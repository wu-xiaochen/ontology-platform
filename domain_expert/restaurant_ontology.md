# 餐饮领域本体设计

## 概述

本文档定义餐饮领域的核心本体模型，涵盖餐饮运营中的核心实体、关系、业务规律和规则，用于指导餐饮知识库构建和智能化应用。

---

## 一、概念定义 (Concepts)

### 1.1 核心实体

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 餐厅 | Restaurant | 提供餐饮服务的场所 | 餐厅ID、名称、地址、业态、面积、座位数、营业时间 |
| 菜品 | Dish | 餐厅制作和销售的食物产品 | 菜品ID、名称、菜系、分类、价格、成本、毛利率 |
| 食材 | Ingredient | 制作菜品所用的原材料 | 食材ID、名称、分类、单位、采购价、库存量、保质期 |
| 顾客 | Customer | 来餐厅用餐的消费者 | 顾客ID、姓名、联系方式、偏好、消费记录 |
| 订单 | Order | 顾客点餐和结算的记录 | 订单ID、下单时间、顾客ID、桌号、金额、状态 |
| 厨师 | Chef | 负责菜品制作的员工 | 厨师ID、姓名、岗位、擅长菜系、级别、工龄 |
| 服务员 | Waiter | 负责接待和服务顾客的员工 | 服务员ID、姓名、工龄、服务评分 |
| 桌台 | Table | 餐厅内顾客用餐的设施 | 桌台ID、编号、容量、状态(空闲/使用中/预定) |
| 供应商 | Supplier | 提供食材的企业或个人 | 供应商ID、名称、联系方式、食材类型、合作状态 |
| 采购单 | Purchase Order | 餐厅采购食材的订单 | 采购单ID、日期、供应商ID、金额、状态 |
| 库存 | Inventory | 食材的库存数量 | 食材ID、库存量、警戒线、补货点 |
| 菜系 | Cuisine | 菜品的地域风味分类 | 菜系ID、名称、特点、代表菜品 |
| 会员 | Member | 餐厅会员计划的消费者 | 会员ID、姓名、积分、等级、储值余额 |
| 评价 | Review | 顾客对餐厅或菜品的评价 | 评价ID、顾客ID、评分(1-5)、评价内容、日期 |

## 二、规律 (Laws)

### 2.1 餐饮客流规律

**定义**: 餐厅客流量随时间变化呈现的周期性波动规律。

**规律表现**:
- **时段规律**:
  - 午餐高峰: 11:30-13:30
  - 晚餐高峰: 17:30-21:00
  - 非饭点: 客流较低
- **周规律**:
  - 工作日: 午餐客流大于晚餐
  - 周末/节假日: 晚餐客流大于午餐
- **月规律**: 月初消费力强于月末
- **季节规律**: 夏季夜宵需求增加，冬季火锅类需求上升

**公式**:
```
时段系数 = 时段客流量 / 日均客流量
翻台率 = 就餐人次 / (桌台数 × 营业小时数)
```

### 2.2 菜品销售分布规律

**定义**: 餐厅菜品销售数量呈现"二八定律"，少数菜品贡献大部分营收。

**规律表现**:
- 20%的菜品贡献80%的营收
- 爆款菜品销售占比可达30%-50%
- 新品上架后呈"S型"增长曲线

**公式**:
```
单品销售占比 = 单品销售量 / 总销售量
菜品贡献度 = 单品营收 / 总营收
```

**应用**:
- 重点维护爆款菜品品质
- 优化菜单结构，淘汰低销量菜品
- 合理搭配流量款、利润款、形象款

### 2.3 毛利率控制规律

**定义**: 餐饮毛利率受菜品定价、成本控制、损耗管理影响的规律。

**行业基准**:
| 业态 | 毛利率参考区间 |
|------|----------------|
| 正餐 | 55%-70% |
| 快餐 | 45%-60% |
| 火锅 | 60%-70% |
| 茶饮 | 65%-75% |
| 烘焙 | 50%-65% |

**公式**:
```
毛利率 = (销售价 - 成本) / 销售价 × 100%
成本率 = 成本 / 销售价 × 100%
```

**影响因素**:
- 食材采购成本 (占比60%-70%)
- 人力成本 (占比15%-25%)
- 租金成本 (占比5%-15%)
- 能源成本 (占比3%-5%)

### 2.4 盈亏平衡规律

**定义**: 餐厅营业收入与总成本相等时的经营状态。

**公式**:
```
盈亏平衡点 = 固定成本 / (1 - 变动成本率)
日均盈亏平衡营业额 = 盈亏平衡点 / 30
所需桌台数 = 固定成本 / (客单价 × 日均翻台 × 30)
```

**固定成本构成**:
- 租金
- 人员工资(固定部分)
- 折旧摊销

**变动成本构成**:
- 食材成本
- 人员加班费
- 水电能耗

### 2.5 食材损耗规律

**定义**: 食材从采购到使用过程中的损耗控制规律。

**行业损耗率参考**:
| 食材类别 | 损耗率参考 |
|----------|------------|
| 叶菜类 | 10%-15% |
| 根茎类 | 5%-8% |
| 肉类 | 3%-8% |
| 海鲜 | 5%-10% |
| 禽类 | 5%-8% |

**公式**:
```
损耗率 = 损耗量 / 采购量 × 100%
出成率 = 可用量 / 采购量 × 100%
```

## 三、业务规则 (Rules)

### 3.1 菜品定价规则

| 定价方法 | 适用场景 | 计算公式 |
|----------|----------|----------|
| 成本加成法 | 通用 | 售价 = 成本 × (1 + 目标毛利率) |
| 价值定价法 | 特色菜 | 售价 = 顾客感知价值 |
| 竞争导向法 | 市场竞品 | 参考竞品价格 ± 差异化调整 |
| 组合定价法 | 套餐 | 套餐价 < 单品之和(引导消费) |

**定价原则**:
- 招牌菜价格高于平均菜价20%-50%
- 流量菜价格具有竞争力
- 菜单要有价格层次(低-中-高)

### 3.2 采购管理规则

| 食材类型 | 采购周期 | 库存策略 |
|----------|----------|----------|
| 鲜货(肉类、蔬菜) | 每日/隔日 | 零库存或1天库存 |
| 干货调料 | 每周/每月 | 安全库存制 |
| 冷冻食品 | 每周 | 适量库存 |
| 酒水饮料 | 每周/每月 | 适量库存 |

**采购审批权限**:
| 采购金额 | 审批人 |
|----------|--------|
| < ¥1,000 | 厨师长 |
| ¥1,000 - ¥5,000 | 店长 |
| ¥5,000 - ¥20,000 | 区域经理 |
| > ¥20,000 | 总部采购 |

### 3.3 库存管理规则

**库存警戒线设置**:
```
安全库存 = 日均消耗量 × 采购周期 + 缓冲量
补货点 = 安全库存
```

**预警机制**:
| 库存状态 | 触发动作 |
|----------|----------|
| 库存 > 安全库存 | 正常 |
| 库存 ≤ 补货点 | 预警，提交采购申请 |
| 库存 ≤ 补货点 × 50% | 紧急采购 |
| 临期食材(前3天) | 优先使用或促销 |
| 过期食材 | 立即报废 |

### 3.4 人员配置规则

**正餐餐厅人员配置参考**:
| 岗位 | 100㎡以下 | 100-300㎡ | 300-500㎡ | 500㎡以上 |
|------|-----------|-----------|-----------|-----------|
| 店长 | 1 | 1 | 1 | 1 |
| 厨师长 | 1 | 1 | 1-2 | 2 |
| 厨师 | 2-3 | 4-6 | 6-10 | 10+ |
| 服务员 | 2-3 | 4-6 | 6-10 | 10+ |
| 收银 | 1 | 1-2 | 2 | 2-3 |
| 洗碗 | 1 | 1-2 | 2-3 | 3+ |

**人效指标**:
```
人均产值 = 月营业额 / 员工人数
人效基准: 正餐 > ¥15,000/月, 快餐 > ¥10,000/月
```

### 3.5 会员运营规则

**会员等级体系**:
| 等级 | 累计消费 | 权益 |
|------|----------|------|
| 普通会员 | 0-999 | 积分倍率1x |
| 银卡会员 | 1,000-4,999 | 积分倍率1.5x, 9.5折 |
| 金卡会员 | 5,000-19,999 | 积分倍率2x, 9折, 优先预订 |
| 黑卡会员 | ≥20,000 | 积分倍率3x, 8.8折, 专属服务 |

**积分规则**:
```
消费积分 = 消费金额 × 积分倍率
积分抵扣 = 100积分 = ¥1
```

### 3.6 卫生安全规则

**食品安全等级**:
| 检查项目 | 要求 |
|----------|------|
| 人员健康 | 持健康证上岗，每日晨检 |
| 食材来源 | 索证索票，可追溯 |
| 加工环境 | 专间专用，生熟分开 |
| 餐具消毒 | 热力/化学消毒，保洁存放 |
| 食品留样 | 125g以上，48小时留存 |

**日常检查频次**:
- 食材验收: 每日
- 加工检查: 每餐
- 餐具抽检: 每日
- 环境卫生: 每日
- 安全培训: 每月

### 3.7 营业时间规则

| 业态 | 常规营业时间 | 高峰时段 |
|------|--------------|----------|
| 早餐店 | 06:00-10:00 | 07:00-09:00 |

## 五、应用场景

1. **智能问答**: 基于本体知识回答菜品推荐、餐厅选择、会员优惠等常见问题
2. **点餐推荐**: 根据顾客偏好和历史消费，推荐合适菜品组合
3. **库存预警**: 基于销售预测和库存规则，自动触发补货提醒
4. **成本管控**: 监控菜品毛利率，预警成本异常
5. **会员营销**: 基于消费数据，进行精准营销和个性化优惠
6. **人力资源优化**: 基于客流规律，优化排班和人效
7. **选址评估**: 基于周边业态和客流，分析新店选址可行性
8. **竞品分析**: 基于行业数据，分析竞争对手优劣势
