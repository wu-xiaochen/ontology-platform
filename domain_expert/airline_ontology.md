# 航空领域本体设计

## 概述

本文档定义航空领域的核心本体模型，用于指导知识库构建和智能化应用。涵盖航空公司、机场、航线、航班、旅客服务、货运、空管等核心业务领域。

---

## 二、规律 (Laws)

### 2.1 航班正常率规律

**定义**: 航班按照计划时间起飞的比率，反映航空公司和机场的运行保障能力。

**计算公式**:
```
航班正常率 = 正常航班数 / 计划航班数 × 100%

其中:
- 离港正常: 航班实际起飞时间在计划时间±15分钟以内
- 到港正常: 航班实际到达时间在计划时间±15分钟以内
```

**影响因素**:
| 因素 | 影响程度 | 说明 |
|------|----------|------|
| 天气 | 40-50% | 出发地、目的地、航路天气 |
| 航空公司 | 20-30% | 机械故障、机组超时、运力调配 |
| 流量控制 | 15-25% | 空域繁忙、军事活动 |
| 机场保障 | 10-15% | 登机口、行李、地面设备 |
| 其他 | 5-10% | 安检、旅客原因等 |

**行业基准**:
- 优秀: 正常率 ≥ 85%
- 良好: 80% ≤ 正常率 < 85%
- 一般: 75% ≤ 正常率 < 80%
- 较差: 正常率 < 75%

### 2.2 客座率规律

**定义**: 飞机实际载客量与可用座位的比值，反映航班收益水平。

**计算公式**:
```
客座率 = 实际旅客数 / 可用座位数 × 100%

或者:
客座率 = 旅客公里数 / 可用座位公里数 × 100%
```

**运营规律**:
- 淡季客座率: 65-75%
- 旺季客座率: 85-95%
- 临界客座率: 约70%（盈亏平衡点附近）
- 高客座率风险: 服务质量下降、收益管理难度增加

**影响因素**:
- 航班时刻: 黄金时刻高客座率
- 竞争程度: 独飞航线高于竞争航线
- 提前预订: 提前越早客座率越低但票价也低
- 舱位配置: 超售比例与客座率负相关

### 2.3 航空煤油消耗规律

**定义**: 航空器飞行过程中的燃油消耗规律。

**计算公式**:
```
小时耗油量 = 起飞重量 × 燃油消耗率

其中:
- 燃油消耗率: 约0.03-0.05升/吨公里
- 远程航班还需考虑备份燃油
```

**油耗影响因素**:
| 因素 | 影响机制 |
|------|----------|
| 飞行距离 | 距离越长，单位公里油耗越高（爬升段占比大） |
| 载重量 | 重量每增加1吨，油耗增加约3% |
| 风速 | 逆风增加油耗，顺风减少油耗 |
| 飞行高度 | 最佳高度层可节省3-5%燃油 |
| 机型 | 新一代机型节油10-20% |
| 发动机 | 新一代发动机节油15-25% |

**典型航油耗参考** (A320/B737):
- 短程 (1-2小时): 2-4吨
- 中程 (2-4小时): 5-10吨
- 远程 (4-8小时): 15-30吨

### 2.4 航班收益管理规律

**定义**: 通过价格和库存优化实现航班收益最大化。

**核心原则**:
```
收益 = 票价 × 座位数

优化策略:
1. 舱位控制: 不同票价对应不同座位库存
2. 超售: 适当超售应对No-show (通常3-5%)
3. 收益管理系统 (RMS): 动态调整票价
4. 差别定价: 淡旺季、提前购票、团体等差异化
```

**收益管理关键指标**:
- 旅客收益 (RASM) = 营业收入 / 可用座位公里
- 单位成本 (CASM) = 运营成本 / 可用座位公里
- 边际贡献 = 票价 - 边际成本

### 2.5 飞行安全规律

**定义**: 航空运输安全运行的规律和原则。

**安全指标**:
```
事故率 = 事故次数 / 飞行小时 × 10^6
致命事故率 = 致命事故次数 / 飞行小时 × 10^6

行业标准 (ICAO):
- 可接受事故率: < 1.0 / 10^6 飞行小时
- 目标事故率: < 0.1 / 10^6 飞行小时
```

**安全影响因素**:
- 人员: 飞行员资质、身心状态、训练水平
- 飞机: 适航性、维护状态、设计可靠性
- 环境: 天气、空域、机场条件
- 管理: 安全管理体系 (SMS)、安全文化

**安全规律**:
- 海恩法则: 每起严重事故背后有29次轻微事故和300次未遂先兆
- 墨菲定律: 可能出错的事情终将出错
- 瑞士奶酪模型: 多重屏障同时失效才会导致事故

### 2.6 航班延误成本规律

**定义**: 航班延误带来的直接或间接成本。

**延误成本构成**:
```
延误成本 = 旅客成本 + 航空公司成本 + 社会成本

其中:
- 旅客成本: 改签、食宿、时间损失
- 航空公司成本: 机组超时、后续航班连锁延误、赔偿
- 社会成本: 声誉损失、时刻资源浪费
```

**延误时间与成本关系**:
| 延误时间 | 成本级别 | 主要成本项 |
|----------|----------|------------|
| <1小时 | 低 | 旅客不满、少量改签 |
| 1-2小时 | 中 | 餐食、住宿、机组调配 |
| 2-4小时 | 高 | 大规模改签、赔偿、机组超时 |
| >4小时 | 极高 | 航班取消、诉讼风险 |

## 三、业务流程 (Processes)

### 3.1 航班运行流程

```
计划制定 → 航线申请 → 航班编排 → 飞机排班 → 机组排班
    ↓
航前准备 → 签派放行 → 旅客登机 → 推出开车 → 滑行起飞
    ↓
飞行阶段 → 巡航 → 下降进近 → 落地滑行 → 停靠登机口
    ↓
航后保障 → 下客 → 卸货 → 清洁 → 航前检查 → 下一航班
```

### 3.2 旅客服务流程

```
出行计划 → 机票预订 → 付款出票 → 值机选座 → 行李托运
    ↓
安检出境 → 候机登机 → 机上服务 → 落地入境 → 行李提取 → 离开机场
```

### 3.3 货运保障流程

```
货物收运 → 危险品检查 → 称重计费 → 货物组装 → 装机运输
    ↓
到达卸货 → 分拣delivery → 通知提货 → 货物交付
```

## 四、关键指标 (KPIs)

### 4.1 运营指标

| 指标 | 英文 | 计算方法 | 优秀标准 |
|------|------|----------|----------|
| 航班正常率 | On-time Performance | 正常航班/计划航班 | ≥85% |
| 客座率 | Load Factor | 旅客数/可用座位 | ≥80% |
| 飞机日利用率 | Aircraft Utilization | 飞行小时/飞机天 | ≥10小时 |
| 签派放行率 | Dispatch Reliability | 成功放行/计划放行 | ≥99% |
| 行李差错率 | Baggage Mishandling Rate | 差错件数/旅客数 | <0.5‰ |

### 4.2 财务指标

| 指标 | 英文 | 计算方法 | 优秀标准 |
|------|------|----------|----------|
| 客公里收入 | RASK | 营业收入/可用座位公里 | 行业领先 |
| 人公里成本 | CASK | 运营成本/可用座位公里 | 行业最低 |
| 边际贡献率 | Contribution Margin | (票价-变动成本)/票价 | >30% |
| 资产回报率 | ROA | 净利润/总资产 | >5% |

### 4.3 安全指标

| 指标 | 英文 | 计算方法 | 优秀标准 |
|------|------|----------|----------|
| 事故率 | Accident Rate | 事故数/飞行小时 | 0 |
| 事件率 | Incident Rate | 事件数/飞行小时 | <1/10^6 |
| 维修事故症候 | Maintenance Incident | 维修原因症候数 | 行业最低 |

## 六、法规与标准

### 6.1 国际规范

- ICAO (国际民航组织): 国际标准和建议措施
- IATA (国际航协): 行业最佳实践和标准
- EASA (欧洲): 欧盟航空安全法规
- FAA (美国): 美国联邦航空法规

### 6.2 国内规范

- CCAR (中国民航规章): 中国民航法规体系
- 适航指令 (AD): 飞机型号设计更改要求
- 运行规范: 航空公司具体运营许可

### 6.3 行业标准

- SITA: 航空电信和IT标准
- OAG: 航班时刻和数据标准
- ONE Record: 航空货运数据交换标准
