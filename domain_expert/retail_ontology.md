# 零售领域本体设计

## 概述

本文档定义零售领域的核心本体模型，涵盖零售核心概念、供应链管理、库存控制、促销规则等关键领域，用于指导零售知识库构建和智能化应用。

---

## 一、概念定义 (Concepts)

### 1.1 核心实体

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 零售商 | Retailer | 直接向消费者销售商品的企业主体 | 零售商编码、名称、业态、门店数量、区域 |
| 消费者 | Consumer | 购买商品或服务的个人或家庭 | 消费者ID、类型(会员/非会员)、偏好 |
| 商品 | Product | 零售商销售的物品 | 商品编码、名称、规格、品牌、类别 |
| 门店 | Store | 零售商实际经营场所 | 门店编码、地址、面积、业态类型 |
| 供应商 | Supplier | 向零售商提供商品的企业 | 供应商编码、名称、等级、合作模式 |
| 采购订单 | Purchase Order | 向供应商下达的订货单 | 订单编号、商品、数量、单价、交期 |
| 库存 | Inventory | 门店或仓库中商品的实际数量 | 商品ID、位置、数量、批次、保质期 |
| 促销活动 | Promotion | 刺激销售的营销活动 | 活动编号、名称、类型、力度、时间 |
| 促销规则 | Promotion Rule | 促销活动的具体参与条件和优惠方式 | 规则编号、条件、优惠、适用范围 |
| 会员 | Member | 零售商的注册会员 | 会员编号、等级、积分、消费记录 |
| 销售交易 | Transaction | 消费者购买商品的交易记录 | 交易编号、门店、时间、金额、商品明细 |
| 供应链 | Supply Chain | 商品从供应商到消费者的完整链路 | 链路ID、环节、参与方、时效要求 |
| 配送 | Distribution | 商品从配送中心到门店或消费者的运输 | 配送单号、方式、状态、时效 |
| 价格 | Price | 商品的售价 | 商品ID、价格类型、金额、生效日期 |
| 货架 | Shelf | 门店中展示商品的设施 | 货架编码、位置、陈列规则 |

## 二、规律 (Laws)

### 2.1 零售商品周转规律

**定义**: 商品从入库到售出的循环效率，衡量零售运营效率的核心指标。

**核心指标**:
```
商品周转率 = 期间销售数量 / 平均库存数量
周转天数 = 365 / 周转率
```

**行业基准**:
| 业态 | 周转天数 | 周转次数 |
|------|----------|----------|
| 便利店 | 15-21天 | 17-24次 |
| 超市 | 25-35天 | 10-15次 |
| 百货 | 45-60天 | 6-8次 |
| 电商 | 30-45天 | 8-12次 |

### 2.2 供应链时效规律

**定义**: 商品从供应商到门店的各环节时间消耗规律。

**环节时效标准**:
| 环节 | 时效要求 | 说明 |
|------|----------|------|
| 供应商响应 | ≤ 24小时 | 收到订单后确认 |
| 配送中心入库 | 1-3天 | 供应商到DC |
| 配送中心加工 | 1-2天 | 拣货、复核、包装 |
| 门店配送 | 1-2天 | DC到门店 |
| **总时效** | **3-8天** | 订单到上架 |

### 2.3 需求预测规律

**定义**: 基于历史数据和市场因素预测未来销售需求的规律。

**影响因素**:
- 季节性因素：春夏秋冬、节假日
- 促销因素：活动前后销量波动
- 天气因素：气温、降雨影响
- 新品因素：新品上市爬坡期
- 竞品因素：竞争对手价格调整

**预测模型**:
```
预测销量 = 基础销量 × 季节因子 × 促销因子 × 趋势因子
```

### 2.4 定价规律

**定义**: 商品价格制定和调整的规律。

**定价原则**:
```
毛利率 = (售价 - 成本) / 售价 × 100%
```

**行业加价率参考**:
| 品类 | 加价率 | 毛利率 |
|------|--------|--------|
| 生鲜 | 15-25% | 13-20% |
| 食品 | 20-30% | 17-25% |
| 日用百货 | 30-50% | 23-33% |
| 服装 | 50-100% | 33-50% |
| 化妆品 | 40-60% | 29-38% |

### 2.5 会员消费规律

**定义**: 会员消费行为的特点和规律。

**会员价值指标**:
```
会员贡献度 = 会员消费额 / 总销售额 × 100%
会员复购率 = 复购会员数 / 总会员数 × 100%
会员生命周期价值 = 平均客单价 × 年均消费次数 × 平均生命周期
```

**会员消费特征**:
- 会员客单价通常高于非会员 20-40%
- 会员消费频次与会员等级正相关
- 积分兑换可提升会员粘性 15-25%

### 2.6 促销效果规律

**定义**: 促销活动对销售的影响规律。

**促销效果衡量**:
```
促销提升率 = (促销期销量 - 非促销期销量) / 非促销期销量 × 100%
促销ROI = 促销增量毛利 / 促销成本
```

**促销效果特征**:
- 价格敏感度高的品类促销效果显著（提升50-200%）
- 品牌形象品类促销可能损害品牌价值
- 促销后通常存在销量回落期（后抑效应）
- 组合促销效果优于单一促销

## 三、业务规则 (Rules)

### 3.1 供应商准入规则

| 条件 | 要求 |
|------|------|
| 资质要求 | 具备营业执照、生产许可证(自有品牌)、相关资质 |
| 品牌授权 | 需提供品牌方授权书或自有品牌证明 |
| 质量体系 | 具备完善的质量控制体系 |
| 财务要求 | 资产负债率 < 75%，经营稳定 |
| 配送能力 | 可支持区域或全国配送 |
| 合规要求 | 商品符合国家法规和行业标准 |

### 3.2 商品上架规则

**新品引入流程**:
```
供应商推荐 → 商品评审 → 样品检验 → 谈判定价 → 合同签订 → 订单下达 → 商品上架
```

**上架资质要求**:
| 商品类型 | 必需资质 |
|----------|----------|
| 食品 | 生产许可证、检测报告、标签备案 |
| 化妆品 | 备案凭证、检测报告 |
| 药品/医疗器械 | 注册证/经营许可证 |
| 3C产品 | 3C认证、能效标识 |
| 儿童用品 | 3C认证、年龄标识 |

**商品准入标准**:
- 加价率满足品类要求
- 供应商配合度达标
- 商品有竞争优势（价格/品质/差异化）
- 符合零售商定位

### 3.3 库存管理规则

**库存分类**:
| 库存类型 | 定义 | 管理方式 |
|----------|------|----------|
| 正常库存 | 可售库存 | 按最高/最低库存控制 |
| 残次库存 | 滞销/损坏 | 折价清仓或报废 |
| 季节库存 | 季节性商品 | 季前备货、季中调整 |
| 促销库存 | 大促期间备货 | 提前锁货、监控动销 |

**库存预警规则**:
| 预警级别 | 触发条件 | 处理时限 |
|----------|----------|----------|
| 黄色预警 | 库存 ≤ 最低库存 × 150% | 3天内补货 |
| 橙色预警 | 库存 ≤ 最低库存 | 24小时内补货 |
| 红色预警 | 库存 ≤ 最低库存 × 50% | 紧急补货 |
| 缺货 | 库存 = 0 | 立即处理 |

**库存盘点规则**:
- 全面盘点：每年至少1次（通常年初/年末）
- 部分盘点：每月/每周重点品类
- 每日动碰：每日销售即时扣减
- 差异率标准：生鲜 ≤ 2%，食品 ≤ 1%，非食 ≤ 0.5%

### 3.4 采购补货规则

**补货模式**:
| 模式 | 适用场景 | 补货依据 |
|------|----------|----------|
| 自动补货 | 常规商品 | 系统自动计算建议订单 |
| 定量补货 | 稳定销量商品 | 库存降至再订货点时 |
| 定期补货 | 季节性商品 | 按固定周期审查补货 |
| 紧急补货 | 缺货/促销 | 特殊审批快速补货 |

**再订货点计算**:
```
再订货点 = 平均日销量 × 采购周期 + 安全库存
```

**安全库存计算**:
```
安全库存 = 销量波动系数 × √(采购周期) × 平均日销量
```

### 3.5 促销活动规则

**促销类型**:

| 促销类型 | 定义 | 示例 |
|----------|------|------|
| 价格促销 | 直接降价 | 立减、折扣、买赠 |
| 满减促销 | 达到金额优惠 | 满100减20 |
| 满件促销 | 达到数量优惠 | 第2件半价 |
| 组合促销 | 组合销售 | 套餐优惠 |
| 会员促销 | 会员专享 | 会员价、双倍积分 |
| 节日促销 | 节日主题 | 春节、中秋、618、双11 |
| 清仓促销 | 清库存 | 季末清仓、特卖 |

**促销审批规则**:
| 促销力度 | 审批人 | 备注 |
|----------|--------|------|
| ≤ 9折 | 品类经理 | - |
| 8-7折 | 品类总监 | 需提前7天申请 |
| ≤ 7折 | 营销总监 | 需提前14天申请 |
| 清仓/特卖 | 总经理 | 需专项审批 |

**促销时间规则**:
- 常规促销：提前7天申请
- 大促（618/双11等）：提前30天准备
- 临时促销：需加急审批
- 单次促销时长：最长30天
- 促销间隔：同类促销间隔至少7天

**促销价格规则**:
- 促销价不得低于成本价
- 促销力度需与供应商协商
- 促销结束后恢复原价需确认

### 3.6 促销规则引擎

**规则结构**:
```
促销规则 = 触发条件 + 优惠内容 + 适用范围 + 限制条件
```

**常见促销规则**:

| 规则类型 | 触发条件 | 优惠内容 | 示例 |
|----------|----------|----------|------|
| 满减规则 | 消费金额达到阈值 | 减免金额 | 满200减30 |
| 满件规则 | 购买数量达到阈值 | 价格优惠 | 买2送1 |
| 折扣规则 | 指定商品/品类 | 固定折扣 | 会员8.5折 |
| 组合规则 | 组合购买 | 套餐价格 | 99元任选3件 |
| 积分规则 | 消费行为 | 积分奖励 | 双倍积分 |
| 换购规则 | 消费+额外支付 | 换购商品 | 加1元换购 |

**规则优先级**:
1. 会员价 > 促销活动价 > 原价
2. 叠加规则：满减可与积分叠加，不可与折扣叠加
3. 互斥规则：同类促销互斥，选择最优

### 3.7 会员积分规则

| 积分类型 | 获得方式 | 积分倍率 |
|----------|----------|----------|
| 消费积分 | 消费1元 | 1倍 |
| 会员日积分 | 会员日消费 | 2倍 |
| 活动积分 | 参与活动 | 活动指定 |
| 生日积分 | 生日月消费 | 3倍 |

**积分有效期**:
- 获得后12个月过期
- 过期前30天系统提醒

**积分兑换规则**:
| 兑换类型 | 积分要求 | 备注 |
|----------|----------|------|
| 商品兑换 | 按商品价值 | 100积分=1元 |
| 优惠券 | 按券面值 | 满减券/折扣券 |
| 服务兑换 | 按服务类型 | 停车/快递 |

### 3.8 价格管理规则

**价格体系**:
```
售价 = 成本价 × (1 + 加价率)
```

**定价权限**:
| 价格类型 | 定价人 | 调整权限 |
|----------|--------|----------|
| 出厂价 | 供应商 | 合同约定 |
| 采购价 | 采购经理 | 谈判确定 |
| 零售价 | 品类经理 | 按加价率 |
| 促销价 | 营销经理 | 审批确认 |
| 清仓价 | 总监/总经理 | 专项审批 |

**价格调整规则**:
- 常规调价：每月审视，季度调整
- 竞品调价：3天内响应
- 成本调价：按合同约定触发
- 促销调价：按促销计划执行
- 调价记录：保留至少2年

### 3.9 供应链配送规则

**配送模式**:
| 模式 | 定义 | 适用场景 | 时效 |
|------|------|----------|------|
| 供应商直送 | 供应商→门店 | 大件/冷链 | 1-2天 |
| 配送中心配送 | 供应商→DC→门店 | 常规商品 | 2-4天 |
| 跨境配送 | 海外→国内→消费者 | 跨境商品 | 7-15天 |
| 即时配送 | 门店/前置仓→消费者 | 本地O2O | 30-60分钟 |

**配送时效标准**:
- 常规配送：48小时内发货
- 次日达：当日24点前下单，次日送达
- 准时达：预约时间段送达
- 急送：3小时内送达（限特定区域）

**配送成本规则**:
- 免费配送门槛：订单满99元
- 配送费标准：5-10元/单
- 偏远地区加收配送费

## 五、应用场景

### 5.1 智能问答
基于本体知识回答零售常见问题：
- 商品价格和促销信息查询
- 会员积分和权益说明
- 配送时效和费用咨询
