## 一、扩展内容

### 1. 采购供应链本体 v2.0

**文件位置**: `ontology-platform/domain_expert/supply_chain.yaml`

**新增内容**:

#### 1.1 供应商评估维度 (第二章)
- **质量维度**: 来料不良率、批次合格率、返工率、客诉次数
- **交付维度**: 准时交付率(OTD)、订单变更响应速度、交期准确率
- **成本维度**: 价格竞争力、年降幅度、总拥有成本(TCO)
- **服务维度**: 响应速度、技术支持、配合度
- **综合评分公式**: `综合评分 = 质量分×0.30 + 交付分×0.30 + 成本分×0.20 + 服务分×0.10 + 环保分×0.10`

#### 1.2 采购流程 (第三章)
- **寻源阶段**: 需求识别→市场调研→潜在供应商筛选→资质预审→供应商名单确认
- **议价阶段**: 询价发起→报价收集→议价谈判→价格确认→供应商选择
- **下单阶段**: 订单生成→订单审核→订单下达→订单确认→订单跟踪
- **验收阶段**: 到货通知→外观检验→数量核对→质量检验→入库办理
- **付款阶段**: 发票提交→三单匹配→付款申请→付款审批→资金支付

#### 1.3 风险指标 (第四章)
- **供应风险**: 单一来源依赖、产能不足、地理风险、交付中断、质量波动
- **价格波动风险**: 价格涨幅、市场供需、原材料成本、汇率波动、价格欺诈
- **质量事故风险**: 重大质量问题、安全事故、环保违规、资质失效、产品召回
- **供应商财务风险**: 资产负债率、利润状况、涉诉情况、经营异常

#### 1.4 选型决策树 (第五章)
- **燃气设备选型决策树**: 根据用户类型(居民/商业/工业)和流量选择合适的燃气表类型
- **供应商选择决策树**: 资质审查→技术评审→商务评审→综合评分排序
- **采购方式决策树**: 常规采购/战略采购/紧急采购的选择条件

#### 1.5 燃气设备专项 (第六章)
- **燃气设备分类**: 燃气表、调压器、报警器、燃气具、燃气管道
- **燃气设备选型规则**: 按流量、压力、温度选择设备型号
- **燃气供应商评估标准**: 资质合规、产品质量、产能供应、技术支持、售后服务

## 二、推理场景脚本

### 2.1 供应商风险评估 (`supplier_risk_evaluation.py`)

**位置**: `scripts/ontology-clawra/supplier_risk_evaluation.py`

**功能**:
- 输入供应商基础信息、绩效数据、财务数据
- 输出风险等级评分、风险因子分析、预警建议

**推理规则**:
| 规则ID | 规则内容 | 风险分值 |
|--------|----------|----------|
| DR-001 | OTD < 90% | +25 |
| DR-002 | 90% ≤ OTD < 95% | +10 |
| QR-001 | 不良率 > 2% | +25 |
| QR-002 | 1% < 不良率 ≤ 2% | +10 |
| QR-003 | 有质量事故记录 | +20 |
| PR-001 | 价格波动 > 10% | +15 |
| FR-001 | 财务状况差 | +20 |
| FR-002 | 财务状况一般 | +10 |
| SR-001 | 单一供应商占比 > 60% | +15 |

**风险等级判定**:
- 高风险: ≥60分
- 中风险: 30-59分
- 低风险: <30分

### 2.2 采购价格分析 (`purchase_price_analysis.py`)

**位置**: `scripts/ontology-clawra/purchase_price_analysis.py`

**功能**:
- 输入物料信息、历史价格、市场行情、采购量
- 输出价格分析、议价建议、采购时机建议

**推理规则**:
- 价格竞争力计算: 与市场均价比对
- 价格趋势分析: 基于3个月以上历史价格
- 议价空间: 基于采购量和供应商数量
- 采购时机: 基于价格趋势

### 2.3 合同条款审查 (`contract_review.py`)

**位置**: `scripts/ontology-clawra/contract_review.py`

**功能**:
- 输入合同条款数据
- 输出条款合规性评估、风险点识别、修改建议

**审查维度**:
| 条款类型 | 风险点 | 阈值 |
|----------|--------|------|
| 付款条款 | 预付款比例 | >30%高风险 |
| 交付条款 | 交付周期 | >45天(大额) |
| 质量条款 | 质保期 | <12个月 |
| 违约责任 | 违约金比例 | <0.5%/天 |
| 价格保护 | 是否包含 | 无则风险 |
| 续约条款 | 提前通知期 | <30天 |

## 三、燃气本体扩展

### 3.1 新增燃气设备分类

```
燃气设备
├── 燃气表 (Gas Meter)
│   ├── 膜式表 - 家用 (G1.6-G4)
│   ├── 罗茨表 - 商业 (G10-G100)
│   ├── 涡轮表 - 工业 (G100-G1000)
│   ├── 超声表 - 大流量
│   └── 腰轮表 - 精确计量
├── 燃气调压器
│   ├── 区域调压器
│   ├── 楼栋调压器
│   └── 终端调压器
├── 燃气报警器
│   ├── 家用报警器
│   ├── 商业报警器
│   └── 工业报警器
├── 燃气具
└── 燃气管道
```

### 3.2 燃气供应商评估标准

| 评估项 | 要求 |
|--------|------|
| 资质合规 | 计量器具许可证、防爆认证 |
| 产品质量 | 型式批准、出厂检验、不良率<1% |
| 产能供应 | 交货期≤15天 |
| 技术支持 | 24小时响应 |
| 售后服务 | 质保期≥2年 |
