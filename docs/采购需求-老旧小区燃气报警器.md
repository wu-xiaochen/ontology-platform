## 二、技术要求

### 2.1 报警器联动逻辑

```
燃气浓度 ≥10%LEL  →  发出预警声光
燃气浓度 ≥20%LEL  →  发出报警声光 + 远程推送 + 发送脉冲信号给切断阀
切断阀收到信号  →  立即关闭燃气管道
```

### 2.2 远程监控要求

- 支持手机APP接收报警通知
- 支持短信通知（至少3个联系人）
- 支持对接社区/物业监控平台（提供API或协议文档）

### 2.3 安装要求

- 报警器安装在厨房天花板下方30cm内
- 切断阀安装在燃气表前或燃气入户主阀后
- 提供专业安装指导
