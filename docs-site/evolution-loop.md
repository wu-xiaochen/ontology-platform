# 进化闭环

## 8 阶段循环

```
感知 → 学习 → 推理 → 执行 → 评估 → 漂移检测 → 规则修订 → 知识更新
  ↑_________________________________________________________________|
                          (从错误中学习)
```

### Stage 1: 感知 (Perceive)

接收外部输入：
- 用户提供的文本
- 执行结果反馈
- 环境变化信号

### Stage 2: 学习 (Learn)

从感知中提取新知识：
- 实体、关系、约束
- 模式识别
- 关联发现

### Stage 3: 推理 (Reason)

严格推理验证：
- 前向链推导
- 后向链验证
- 一致性检查

### Stage 4: 执行 (Execute)

应用知识到任务：
- 决策生成
- 方案规划
- 行动执行

### Stage 5: 评估 (Evaluate)

评估执行质量：
- 结果正确性
- 推理质量
- 知识覆盖率

### Stage 6: 漂移检测 (Drift Detection)

检测环境变化：
- 知识陈旧度
- 新模式出现
- 规则冲突

### Stage 7: 规则修订 (Rule Revision)

更新规则集：
- 冲突解决
- 规则合并
- 优先级调整

### Stage 8: 知识更新 (Knowledge Update)

持久化更新：
- 存入知识图谱
- 更新向量索引
- 清理冗余

## 触发方式

=== "手动触发"
    ```python
    await clawra.evolve(trigger="manual")
    ```

=== "自动触发"
    ```python
    # 配置后自动运行
    clawra.configure(evolution_interval=3600)  # 每小时
    ```

## 进化示例

```python
# 初始状态
clawra.learn("猫是哺乳动物")
clawra.learn("狗是哺乳动物")

# 执行推理
clawra.reason("猫有什么特征？")

# 评估发现知识不足
# → 自动触发进化
# → 从错误中学习新规则
```
