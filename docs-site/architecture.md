## 核心组件

### Perception Layer（感知层）

基于 LLM 的知识提取。从非结构化文本中提取：
- **实体** (Entities)
- **关系** (Relations)
- **约束** (Constraints)
- **模式** (Patterns)

### Unified Logic Layer（统一逻辑层）

结构化知识存储，支持多种知识类型：
- **规则** (Rules)
- **行为** (Behaviors)
- **策略** (Policies)
- **约束** (Constraints)

### Reasoner（推理机）

- **前向链**：从已有事实推导出新结论
- **后向链**：验证某个结论是否成立
- **混合推理**：结合神经和符号方法

### Memory（记忆层）

- **Neo4j**：知识图谱存储
- **ChromaDB**：向量存储
- **GraphRAG**：混合检索引擎

### Meta Learner（元学习层）

系统的"大脑"：
- 学习如何更高效地学习
- 从错误中提取经验
- 策略自适应调整

## 数据流

1. **输入** → 文本、反馈、结果
2. **感知** → LLM 提取结构化知识
3. **学习** → 存入知识图谱
4. **推理** → 符号逻辑验证
5. **执行** → 输出决策
6. **评估** → 质量评估
7. **进化** → 规则更新
