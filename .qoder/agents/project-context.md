# Clawra 项目上下文

## 项目概述

**名称**: Clawra - 自主进化本体认知引擎  
**版本**: 2.0.0  
**状态**: 核心功能已完成，测试验证阶段

## 核心目标

1. **零硬编码** - 所有规则通过自主学习获得
2. **领域自适应** - 自动识别领域，动态加载逻辑
3. **自我进化** - 从数据、文本、交互中持续学习

## 技术架构

```
┌─────────────────────────────────────────┐
│           Meta Learner (元学习)          │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│  UnifiedLogicLayer │ RuleDiscovery      │
│  (统一逻辑表达)      │ (规则发现)         │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│  Reasoner (推理) │ Memory (记忆)         │
└─────────────────────────────────────────┘
```

## 关键文件

| 文件 | 职责 |
|-----|------|
| `src/evolution/unified_logic.py` | 统一逻辑表达层 |
| `src/evolution/meta_learner.py` | 元学习器 |
| `src/evolution/rule_discovery.py` | 规则发现引擎 |
| `src/core/reasoner.py` | 推理引擎 |
| `tests/test_*.py` | 单元测试 |

## 开发命令

```bash
# 运行测试
python -m pytest tests/ -v

# 运行 Demo
streamlit run examples/autonomous_evolution_demo.py

# 类型检查
mypy src/
```
